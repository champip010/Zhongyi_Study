# 气滞证诊断标准编制说明

# 一、工作简况

# 1.任务背景

气滞证是中医临床常见证候之一，多见于许多重大疾病和慢性疾病，并为其始发证和基本证，常伴随疾病始终，但诊断标准缺如，严重影响了证候诊断的规范性、一致性及可重复性。主要有以下几点问题：未建立科学的量化标准，临床诊断效能及实用性有待考证；不同疾病气滞证主次症状、条目权重、诊断阈值尚未确定。为避免个人因素对辨证的干扰，提高临床诊疗和临床科研辨证的准确性和一致性，结合国内外最新证候及相关术语研究进展，特制定本标准，指导临床与科研气滞证辨证的准确性和规范性。

为落实《中医药发展战略规划纲要（2016—2030年）》提出的“健全中医药法律体系和标准体系”的重要任务，本标准相关法律法规和技术文件指导的框架下，综合运用文献系统评价、德尔菲法、层次分析法、专家访谈、诊断性试验等方法，确立了气滞证诊断条目及其权重，修订了气滞证相关诊断标准，为气滞证的临床诊疗和科学研究提供指导。

# 2.任务来源

本任务来源于广东省重点领域研发计划项目“气滞证辨证标准的系统研究”（项目编号：2020B1111100001）的工作基础上开展相关研究工作，由暨南大学、湖南中医药大学、广州中医药大学负责组织完成。立项时间是 2020年 3月。课题经费资助来自广东省重点领域研发计划项目。

# 3. 标准起草单位

参与起草单位包括：暨南大学、湖南中医药大学、广州中医药大学、北京中医药大学、广东省中医院、湖南中医药大学第一附属医院、湖南省脑科医院、暨南大学附属第一医院、广州中医药大学第二附属医院、广州市惠爱医院、佛山市第三人民医院。

# 二、标准的编制原则和关键技术内容

# 1.标准的编制依据和原则

本标准按照《中华中医药学会中医标准制定的技术方案》、《标准化工作导则

标准的结构和编写GB/T1.1-2009》规定的规则撰写。

本标准编制遵循“开放性、协商一致性、透明性、可追溯性”原则，按照“能够为中医行业内实际应用，能被行业外广泛接受和认可，并与国际诊疗标准接轨”的要求，开展了气滞证诊断标准的研究。

# 2.标准的关键技术内容

《气滞证诊断标准》共有8 部分组成，分别是：前言、引言、范围、规范性引用文件、术语及定义、气滞证的诊断标准、附录 A（资料性附录）历年气滞证类中医辨证标准的相关部分、参考文献。

# 三、主要工作过程

# （一）提案、申请、立项

2020 年 04 月 17 日，陈家旭教授在腾讯会议召开“气滞证诊断标准”项目讨论会，会议确定向中华中医药学会标准化办公室申请《气滞证诊断标准》的立项，并确立了由总指导、临床医师、方法学专家、科研协作组共同构成起草组。2020年05月底向中华中医药学会标准化办公室提交《气滞证诊断标准》立项申请材料。2020 年 08 月，中华中医药学会组织召开团体标准立项函评，《气滞证诊断标准》通过立项论证。会后按照专家意见对立项申请书进行完善，经中华中医药学会秘书长办公会审议批准，于2020年11月16日发文正式立项（附件1：中华中医药学会团体标准立项公告）。

# （三）确定标准的题目和范围

方法：根据广东省重点领域研发计划项目“气滞证辨证标准的系统研究”（项目编号：2020B1111100001）明确题目气滞证诊断标准。

过程：中医气滞证涉及健康/亚健康人群及疾病人群，其中疾病人群涉及疾病谱广（如抑郁症、慢性非萎缩性胃炎、围绝经期综合征、乳腺纤维腺瘤、冠心病、慢性阻塞性肺疾病等），临床患者人群量大。为建立简单易行、实用性强的中医气滞证共性诊断标准；通过现代文献、临床实践中，探索病位肝、胃肠、心、肺四个亚型证个性症状、体征，从而提高临床辨证的准确性，从而提高临床疗效，且受益群体众多，具有重要价值和意义。

# （四）构建标准问题

建立科学的气滞证量化标准，临床诊断效能；明确不同疾病气滞证主次症状、条目权重、诊断阈值。为避免个人因素对辨证的干扰，提高临床诊疗和临床科研辨证的准确性和一致性，结合国内外最新证候及相关术语研究进展，特制定本标准，指导临床与科研关于气滞证辨证的准确性和规范性。

需要统一亚型证与基础证的主症。临床辨证时较少观察理化指标的变化，相关的结论是通过查阅资料等理论方法推导出的，可能与临床实际有较大差距。

（4）董昌武教授

气滞证需要有共性，其他不同脏腑气滞都应含有气滞证共性的内容；理化指标需慎重纳入。

# （5）何建成教授

失眠是否应细化。本团队在做前瞻性研究时，发现冠心病心力衰竭心脉气滞证患者表现出的症状为冠脉痉挛，冠状动脉并没有实质性的损害。病理因素是证型不统一的原因，不仅仅强调辨证论治，还强调辨病论治。

# （6）郑景辉教授

文献检索后是否进行了文献质量评价，制定标准不是所有文献都要进行文献质量评价，尤其是诊断性研究（相关网站 https://www.equator-network.org/）。指标是观察性研究，就用 STROBE 评价工具，诊断性研究用STARD 评价工具。

# （7）张诗军教授

脾虚证将唾液淀粉酶作为金指标，因此，理化指标不一定要很多。气滞证研

究中，胃肠气滞与胃动力有关；肝郁气滞证与情绪指标相关，如5-羟色胺5-HT。

# （8）刘赟教授

临床常见气滞证患者，初期为单纯气滞证，后期多出现化火、生痰、生瘀等变化为气滞的变证，建议应将变证类条目予以排除。肝郁气滞证或气滞证的病人的理化指标在临床上可能观察不到太大的变化。

# （9）钟舒明教授

气滞证问卷分型像临床中所见的焦虑症患者，这些患者的病变存在于人体不同的系统中，如胃肠道系统、神经系统、心血管系统等方面。理化指标变化会存在于气滞证患者身上，但这些指标可能没有特异性。临床推广时，需要考虑临床的可操作性，操作不容易，普及性会低。

# （10）喻斌教授

标准关键在于指导临床还是指导科研，许多理化指标特异性、敏感性的验证，会使临床的可操作性时间很长。气滞证理化指标多出现在功能性胃肠疾病中，如5-羟色胺（5-HT）、胃泌素（GAS）等，但这些指标的变化没有特异性；而指标升高降低到哪一个值也很难确定，所以在制定标准过程中需慎重选择理化指标。

# （11）李新纯教授

气滞证理化指标不够明显，可操作性不强，范围广，如脑区兴奋等。中医最大的特点不是靠实验，而是观察舌脉开方治疗，常规的理化指标用于观察患者服用中药后是否会出现不良作用，因此，理化指标不一定有意义，可能对科研方面有用，但对诊断气滞证必要性不强。

# （12）张涛教授

中医的气滞证与脏腑证应有共性，其次突出脏腑个性。气滞证、肝郁气滞证都是与情志密切相关，中医可以结合西医量化积分表，达到阈值后就可以评定为此证型，此方法更容易被临床接受。

# （13）刘旺华教授

问卷中的理化指标与中医关联度不高，因此，文献质量要进行评估。注意总证与亚型的关系，总证是一个汇总还是共性？考虑不同病位的气滞证共性在哪里？作为科学研究，按照取象比类的思想，气滞心脉证是否可以考虑心室舒张期末直径、容积、压力？胃肠气滞通降能力下降，是否可以考虑胃动力检测指标？肺气郁滞证是否考虑红细胞计数、红细胞压积、血氧饱和度、氧合血红蛋白与还原血红蛋白比？今后可通过大样本研究验证是否可以纳入诊断标准？

(14)简维雄教授

理化指标选择一定要慎重，尤其气滞证和亚型证更倾向于自身选择，慎重选择后对于标准更有意义。

(15)丁长松教授

诊断系统做成后，能不能做个计算机临床辅助系统，具有临床意义。量化指标分值为 0、1、2、3、4，打分的间距定量范围的设置如何统一，需更深入的探索与实施。

(16)李晓娟教授

临床标准，在文献检索后，需要进行文献质量评价和信度效度判定，否则难以推广于临床。

(17)陈家旭教授

宏观指标、试验指标、文献选择专家都提了很多宝贵的意见，后续需认真总结。

# (18)胡志希教授

①证型比较：专家意见集中在基础证和具体证之间的共性与个性。气滞证作为一个基础证，我们研究的气滞证诊断标准，需在基础证上根据不同部位的脏腑，分为心脉气滞证，肝郁气滞证，胃肠气滞证，肺气郁滞证四个亚型证，共五个证。据李先涛教授所说只申报了一个气滞证，团队也在考虑能不能做五个标准，气滞证及其亚型证的标准诊断，也将是本团队下一步要做的一个工作。目前证型集中在共性与个性方面，就是说气滞证的主次症应该在亚型中有体现，如同专家提出证型之间不要相互矛盾，相互冲突，所以我们将根据专家意见予以修正条目。

②理化指标：原本未涉及理化指标的内容，现代中医应该利用现代科技的东西，包括形态学的、功能性的理化指标，但没有特异性指标，实施可能比较困难。理化指标作为一个标准来颁布，需达成专家共识和标准实施是比较困难的，所以我们将会慎重考虑理化指标。

③根据郑教授所提及的意见，我们将会完成文献质量控制。关于专家涉及的量表诊断效能方面，团队将问卷修改后的内容转换成量表的形式发放至临床中，进行后续的临床验证等研究。

通过线上会议谈论，本团队将根据专家意见，完成问卷的修改，开展问卷调查。
