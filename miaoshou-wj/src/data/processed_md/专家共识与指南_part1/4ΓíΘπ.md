# 一 宣传、贯彻共识的措施

# 1. 共识的实施单位

建议《肉芽肿性小叶性乳腺炎中医诊疗专家共识》作为推荐性标准，向全国乃至世界推广，提高临床疗效，突出中医治疗优势，改善患者治疗结局，减少社会医疗资源浪费。

# 2. 其他宣传、贯彻本共识的措施

根据实际情况，采用线上及线下相结合的方式，以专题讲座、典型病例讨论、专家联合会诊等多种形式开展推广活动。预计共 15场次；共1500人；

拟进行新闻发布、媒体报道，推广本共识。

本共识成果拟在专业学术杂志上发表。

# 二 共识的用户评价

拟于 2024 年开展共识的用户评价，用户评价方案采用线上问卷及线下研讨会相结合的方式。

# 三 共识的修订

拟于2024 年对本共识进行更新或修订。
