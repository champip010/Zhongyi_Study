# 一、2020 版共识的主要变化

自1998年以来,多家规模较大的功能神经外科晨和设备国产化进程的推中的应用进入快速发展时，中心的理念要求多学科密切合作,形成整体思维,对临床医生的要求也进一步提高。因此,神经内科、神经外科、康复科、心理科及护理学等多学科领域专家组通力合作，从各自的专业角度出发,制定出 2020 版共识。与2012 版共识相比,2020 版共识的内容更具实用性、科学性和前沿性,对DBS 疗法术前、术中、术后各个环节均提出了具体的指导建议。2020 版共识对患者的选择、手术步骤、术后程控方面的内容进行了补充和更新,新增了靶点选择,术后药物、康复治疗、术后护理等多领域协同管理,并发症的预防和处理,并发表附录1、2和3。

# (二)术前评估

PD 患者手术前需要进行系统许怕,以明佣定否符合DBS 的适应证,是否存在禁忌证,为选择手术靶点、术后程控及药物调整、术后客观评价疗效提供依据。评估内容应包括运动症状、运动并发症、非运动症状、生活能力等要素。

2020 版共识的附录1中对术前评估重点进行了介绍。其强调左旋多巴反应性评测的重要性，更新了评测方法,并指出该方法是诊断PD 的客观指标和重要的疗效预测指标,改善率≥30%的患者可能具有较好的疗效。同时,2020 版共识也指出了该评估方法的局限性，如对难治性震颤疗效的预测价值可能不大。除了传统的头部影像学检查、Hoehn-Yahr分期、基于统一帕金森病评定量表（unifiedParkinson's disease rating scale,UPDRS）或国际运动障碍学会统一帕金森病评定量表（MDS-UPDRS）评分的神经功能整体评估外,2020 版共识还强调对运动并发症、认知功能、神经心理功能、自主神经功能以及其他非运动症状的评估,并推荐了剂末现象、异动症、抑郁、焦虑、淡漠、精神症状、冲动控制障碍、直立性低血压、胃肠道自主神经功能、排尿功能、睡眠障碍、疼痛、疲劳、生活质量及日常生活能力等方面的具体评估量表。
