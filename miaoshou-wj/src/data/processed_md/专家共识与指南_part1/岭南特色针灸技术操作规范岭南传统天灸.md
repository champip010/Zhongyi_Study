# 引言

天灸疗法是一种将对皮肤有刺激性的药物敷贴于穴位或患处，通过局部皮肤自然充血、潮红或起泡来治疗疾病的方法。晋代葛洪、鲍姑夫妇是传统天灸疗法在岭南传播的开创者。从葛洪夫妇在岭南以天灸行医以来，天灸疗法在岭南至少有1700多年的应用历史。近现代岭南医家曾天治、周仲房通过创建中医学校、编写教材等方式把岭南传统天灸疗法纳入现代中医院治疗体系，岭南传统天灸疗法得到了整理和发展，已成为具有浓郁地方特色的中医外治疗法，并成功入选广东省第四批省级非物质文化遗产名录。

据统计，自1984年大规模开展天灸疗法以来，现每年在广东省中医院接受天灸约30万人次，且呈逐年上升趋势。该疗法凭其简便的操作、可靠的疗效，取得了广泛的社会认同，已推广应用至四川、海南、新疆、宁夏、山东、河北、广西、湖南、青海、广东、港澳地区等全国近20个省市50多家医院。现制定本技术操作的标准性文件，以利于岭南特色针灸技术的规范，以及针灸同仁的学习与运用。

# 3 术语和定义

# 3.1 岭南传统天灸

岭南传统天灸是根据经络腧穴理论及中医时间学理论，在特定时间将某些对皮肤有刺激作用的药物涂敷于腧穴或患处，使局部充血潮红、皮肤起泡，以治疗疾病的灸法。又称为药物发泡灸、发泡疗法。

# 3.2三伏

三伏分为初伏、中伏、末伏，夏至后第三个庚日为初伏，第四个庚日为中伏，立秋后第一个庚日为末伏，三伏日均为庚日。

# 3.3三九

从冬至后第一个壬日算起，每九天为一“九”，第一个九天叫“一九”，第二个九天叫“二九”，第三个九天叫“三九”。

# 4操作步骤与要求

# 4.1 施术前准备

# 4.1.1 药物

根据中医辨病、辨证原则选择贴敷药物，其中应含有刺激皮肤发泡类的药物。

药物的使用应遵守《中华人民共和国药典》规定；药物的使用、制备、贮藏原则还应符合GB/T33414-2016 的规定。

将药物制成1cm大小的药饼或软膏备用。

# 4.1.2医用胶布与敷料

准备好医用胶布，胶布中心应有直径约5cm的圆形或方型防渗液敷料。

# 4.1.3穴位选择及定位

根据病症选取适当的穴位或治疗部位。穴位的定位应符合GB/T12346 的规定。

# 4.1.4 体位选择

选择患者舒适、医者便于治疗的体位。

4.1.5 环境要求

应注意环境清洁卫生，避免污染。

# 4.1.6 消毒

医者双手应用肥皂水清洗干净。用75%乙醇消毒液或0.5%～1%碘伏棉球或棉签在施术部位消毒。

# 4.2 施术方法

将药饼或适量药膏直接置于穴位上，然后外覆医用胶布固定；或先将药物置于医用胶布粘面正中，再对准穴位粘贴。

# 4.3施术后处理

# 4.3.1 贴药时间

贴药处皮肤潮红或自觉贴药处瘙痒、灼热、刺痛时，即可移去贴敷药物。成人一般贴敷30～60分钟为宜，老人及部分人皮肤耐受可适当延长时间，但贴敷时间不宜超过2小时；青少年一般贴敷15～45分钟为宜；儿童一般贴敷5～15分钟为宜；幼儿贴敷时间一般在5分钟之内。

# 4.3.2 贴药反应与处理

贴药治疗后局部皮肤出现轻微灼痛、红晕属正常现象。必要时可外涂皮肤软膏以缓解不适。

贴药时间过长可有红肿、瘙痒、水泡出现，避免搔抓破损。水泡处理应参照GB/T 21709.9-2008。

贴敷后若出现范围较大、程度较重的皮肤红斑、水泡、瘙痒等现象，应立即停药，并对症处理，必要时应至专科就诊。

# 4.3.3换药

局部皮肤炎症反应消退后，可根据治疗需要再次行天灸治疗。

# 5注意事项

5.1药物选择与加工、助透剂的选择、贴敷穴位、贴敷时间等因素均可影响疗效。5.2 岭南传统天灸常于三伏、三九当日应用。三伏天灸期间可根据病情在三伏天前后或中间增加治疗次数。  
5.3 贴药时皮肤应保持干燥，贴药后不宜剧烈活动，以免出汗致药膏脱落。5.4 贴药当日戒酒、辛辣、海鲜、蘑菇、牛肉、芋头等易致化脓食物，并避免进食生冷食品及进行冷水浴。  
5.5 年龄越小则贴药时间相应缩短。  
5.6 颜面部、四肢远端、肚脐等部位应控制贴药时间，不宜发泡。

# 6禁忌

6.1对敷料或药物过敏者、局部皮肤溃疡者不可应用天灸。  
6.2 发热病人、合并严重心脑血管疾病、肝肾功能不全、糖尿病血糖控制不佳或有严重并发症者不宜使用天灸。  
6.3一岁以下的婴儿、孕妇慎用天灸。

# 岭南传统天灸的常用药物

# A.1岭南传统天灸药物的一般选用原则

（1）药物的选择以岭南道地药材为佳。（2）药物常为辛散通经活络、辛香化浊开窍之品，多具有刺激皮肤发泡的作用。（3）部分药物需生用或使用鲜品，以保留其辛散、辛香之气。

# A.2岭南传统天灸常用药物的功效与主治

岭南传统天灸使用的药物具有地域特色。常用的无毒药物包括大蒜、生姜、葱白、胡椒、醋、地龙、黄芥子、延胡索、鹅不食草、墨旱莲、大黄、威灵仙、五倍子、何首乌、丁香、肉桂、爵床、小茴香、干姜、明矾；常用的有毒药物包括土荆芥、细辛、吴茱萸、天南星、巴豆、甘遂、斑螯。本标准介绍部分药物的功效与主治，其余药物的功效与主治可参考GB/T33414-2016。葱白：性微温，味辛，归肺、胃经。具有发汗解表，散寒通阳之功。主治感冒风寒，阴寒腹痛，二便不通，痢疾，疮痈肿痛，虫积腹痛等病症。每日内服用量6～30g，亦可取汁服，外用适量。鹅不食草：性温，味辛，归肺经。具有发散风寒，通鼻窍，止咳之功。主治风寒头痛，咳嗽痰多，鼻塞不通，鼻渊流涕等病症。每日内服用量6\~9g。外用适量。墨旱莲：性寒，味甘、酸，归肾、肝经。具有滋补肝肾，凉血止血之功。主治肝肾阴虚，牙齿松动，须发早白，眩晕耳鸣，腰膝酸软，阴虚血热吐血、血、尿血，血痢，崩漏下血，外伤出血等病症。每日内服用量6～12g，外用适量。何首乌：性微温，味苦、甘、涩，微温，归肝、心、肾经。具有补肝肾，益精血，乌须发，强筋骨，化浊降脂之功。主治血虚萎黄，眩晕耳鸣，须发早白，腰膝酸软，肢体麻木，崩漏带下，高脂血症等病症。每日内服用量6～12g，外用适量。小茴香：性温，味辛，归肝、肾、脾、胃经。具有散寒止痛，理气和胃之功。主治寒疝腹痛，睾丸偏坠，痛经，少腹冷痛，皖腹胀痛，食少吐泻等病症。每日内

服用量3～6g，外用适量。

明矾：性寒，味酸、涩，归肺、脾、肝、大肠经。外用具有解毒杀虫，燥湿止痒；内服具有止血止泻，祛除风痰。外治用于湿疹，疥癣，脱肛，痔疮，婷耳流脓等病症；内服用于久泻不止，便血，崩漏，癫痫发狂等病症。每日内服用量0.6～1.5g。外用适量，研末敷或化水洗患处。巴豆：性热，味辛，有大毒，归胃、大肠经。外用具有蚀疮之功。主治恶疮疥癣，疣痣等病症。外用适量，研末涂患处，或捣烂以纱布包擦患处。黄芥子：性温，味辛，归肺经。具有温肺豁痰利气，散结通络止痛之功。主治寒痰咳嗽，胸胁胀痛，痰滞经络，关节麻木、疼痛，痰湿流注，阴疽肿毒等病症。

每日内服用量3～9g，外用适量。

# 岭南传统天灸的药物制备方法

岭南传统天灸药物常用剂型为饼剂或软膏剂，药物的加工流程为：配药→清洗→粉碎→过筛→混合。依照处方配药，清水清洗药物后晾干，续将药物粉碎成细末，然后以60～80目的细筛筛过，混合拌匀而成。使用时取药物适量，以姜汁调和成药饼或药膏后置于胶布上贴敷于穴位或患处。

# 岭南传统天灸的临床适应症与选穴原则

# C.1岭南传统天灸的临床适应症

根据文献与临床调研，列举如下适应症：

（1）肺系相关病症：如慢性支气管炎、支气管哮喘、过敏性鼻炎、虚人感  
冒、慢性肺气肿、慢性咳嗽等;（2）胃肠病症：慢性结肠炎、功能性腹胀、慢性胃炎、食道反流性胃炎、  
胃动力性疾病、便秘等；（3）痛症：颈椎病、腰椎间盘突出症、腰肌劳损、膝骨关节炎、网球肘、  
肩周炎等；（4）抑郁相关病症：抑郁障碍、焦虑障碍、睡眠障碍、阈下抑郁、慢性疲  
劳综合征、产后抑郁等；（5）其它：肥胖症、遗尿、慢性盆腔炎、乳腺增生等。

# C.2 岭南传统天灸的选穴原则

按照辨证一般选取背部膀胱经、督脉，腹部的任脉、胃经、脾经穴位为主，必要时配合四肢穴位，一般每次选8～12个穴位。
