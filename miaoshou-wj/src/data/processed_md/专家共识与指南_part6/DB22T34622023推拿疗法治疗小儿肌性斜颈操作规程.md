# 推拿疗法治疗小儿肌性斜颈操作规程

# 1 范围

本文件规定了推拿疗法治疗小儿肌性斜颈的治疗、注意事项和护理。  
本文件适用于0 月～6 月小儿肌性斜颈的推拿治疗。

# 2 规范性引用文件

本文件没有规范性引用文件。

# 3 术语与定义

下列术语和定义适用于本文件。

3.1 小儿肌性斜颈 muscular torticollis in children 因一侧胸锁乳突肌挛缩，引起颜面部向健侧旋转，下颌转向患侧的病症。

# 4 治疗

# 4.1 准备

患侧胸锁乳突肌热敷，温度40 ℃，10 min～15 min。准备滑石粉、玉米淀粉润滑剂。

# 4.2 体位

仰卧位。

# 4.3 操作流程

# 4.3.1 按揉

患儿头略偏向患侧，术者以单手食、中指桡侧或食指指腹与拇指指腹相对捏住患处，由上至下按揉患侧胸锁乳突肌起点至止点，按揉5 min～10 min，频率100 次/min～120 次/min。先轻渐重。

# 4.3.2 拿揉

术者沉肩、坠肘，以拇指、食指在肿物周围反复进行拿揉。然后用食、中指与拇指固定肿块一侧，拇指对肿块进行提捏往返弹拨，操作时应由轻到重，垂直于肿块，力度适中。拿揉5 min，弹拨3 min。

# 4.3.3 旋转

术者一手托患儿下颌，一手托后枕部，双手协调使患儿头偏向健侧，并使下颌旋向患侧至极限位，停留3 s～5 s后恢复回原位，患三健一，反复操作3 遍～5 遍。

# 4.3.4 扳法

术者一手下压患侧肩部，另一手置于患儿头侧，双手同时向反方向用力，力度适中，使患者颈部向健侧牵拉，反复操作3 遍～5 遍。

# 4.3.5 拔伸

家长固定患儿双手、双脚及肩部，术者一手置于患儿下颌部，一手置于后枕部，双手用力并向上拔伸牵引，反复操作3 遍～5 遍。

# 4.3.6 点按

用拇指轻柔点按患侧的耳后高骨、风池、肩井及翳风等穴位3 min～5 min，点按五经穴各100 次，点按五指节50 次，点按一窝风100 次，点按中脘300 次，点按天枢穴300 次，点按足三里穴300 次，点按脾俞、肾俞各300 次，捏脊3 遍～5 遍，每周2 次。

# 5 注意事项

5.1 保护好局部皮肤，破损不宜推拿。  
5.2 小儿过饱、饥饿、生病、困倦时不宜姿势引导及推拿治疗。  
5.3 不宜疲劳训练。  
5.4 推拿每次 20 min 左右。  
5.5 手法应熟练。  
5.6 注意环境温度及推拿医生手的温度。

# 6 护理

6.1 患儿俯卧，用视听刺激诱导患儿抬头转向患侧 45°。  
6.2 当患儿头部有一定控制能力后，扶抱患儿取坐位，缓慢把患儿身体向患侧倾斜，诱导患儿颈部向健侧侧屈。

6.3 翻身训练促进患儿对称性运动。
