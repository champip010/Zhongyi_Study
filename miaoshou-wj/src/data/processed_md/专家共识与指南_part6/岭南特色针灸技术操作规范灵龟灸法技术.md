# 引言

灵龟灸是林国华教授、王澍欣教授师承全国老中医专家张家维教授对后天八卦在腹部临床应用的基础上，通过对选穴依据的简化、腹部穴位区域的明确、药材的配比、艾条的规格等深入研究，并对灵龟灸的理论和操作规范不断完善。因其起效快、透热均匀、热力强，适应范围广，有明确的疗效，取穴依据腹部八卦的分布而得名。

目前灵龟灸已在临床上应用于虚证、寒证、痹症及痛症等相关疾病，已推广至广东、广西、浙江等全国近十省及港澳地区。故制定本技术操作的标准性文件，以利于岭南特色针灸技术操作的规范，以及针灸同仁的学习与运用。

# 3 术语和定义

# 3.1 灵龟灸

属灸法的一种，根据患者症状，以脏腑辨证选取相对应的病变脏腑，按腹部后天八卦对应的脏腑部位选取腹部一定的区域，以特制的灵龟神灸条进行艾灸，使患者有温热而无灼痛的一种方法。

3.2 腹部后天八卦及其对应脏腑部位

腹部后天八卦是以脐为中心，上至巨阙、不容，下至曲骨，左右至带脉，斜经肋弓、腹股沟，联结章门、五枢围成的一个八卦区域，所对应脏腑部位如附录A所示。

# 3.3灵龟神灸条

灵龟神灸条是林国华教授在张家维教授根据腹部灵龟灸适应症选取数味中药材的基础上进行加减后，然后按照一定的比例配比同艾绒混合制成的艾条（见附录B）。中药材：广藿香、皂角刺、川芎、醋没药、丁香、桂枝、盐杜仲、麸炒枳壳、细辛、独活、木香、续断片、盐牛膝、防己、肉桂、当归，各以等量混匀打粉，药粉和黄金艾绒以1：9配比制成规格为直径5cm、长

20cm、重 600g 的实性圆柱状艾条。

# 4操作步骤与要求

# 4.1 施术前准备

# 4.1.1灸材选择

（1）选择特制的灵龟神灸艾条，检查艾条有无霉变、潮湿。（2）准备好具有消肿止痛作用的液性药物，如万花油。（3）准备好线香、打火机等点火工具以及治疗盘、弯盘、棉签、镊子等辅助工具。

# 4.1.2穴位的选择与定位

根据病症选取相应脏腑的治疗部位。穴位定位应符合GB/T12346 的规定。

# 4.1.3 体位的选择

因操作部位均为腹部，故选择仰卧位。

# 4.1.4环境的要求

应注意环境清洁卫生，避免污染；注意环境保暖通风，避免疗效减低。

# 4.2 施术方法

# 4.2.1取穴

以脏腑辨证论治辨病变脏腑，以此选取腹部八卦相应的脏腑区域。

# 4.2.2选取艾条

选规格为：直径 5cm、长20cm、重 600g 的实性圆柱状艾条。

# 4.2.3点燃艾条

因艾条直径大，需辅予酒精灯点燃，此操作在远离病人身体的操作台上完成。

# 4.2.4施灸手法

分为温和灸、回旋灸、雀啄灸。

温和灸：将艾条燃着端悬于施灸部位上距皮肤 20～30cm处，根据病人体表感受的温热调节，热度以温和为主，灸至病人有温热舒适而无灼痛；

回旋灸：将艾条燃着端悬于施灸部位上距皮肤20～30cm 处，根据病人体表感受的温热调节，热度以温和为主，平行往复回旋熏灸，使皮肤有温热感而不至于灼痛；雀啄灸：将艾条燃着端悬于施灸部位上距皮肤20～30cm处，对准穴位，上下移动，使之像鸟雀啄食样，一起一落，忽近忽远地施灸，根据病人体表感受的温热调节，热度以温和为主。

# 4.2.5控制灸度和灸量

根据患者病情轻重及耐受力，选用合适的艾灸量。病情重者可每日一灸，病情稳定者可隔日或3-5日一灸，灸5～7次为一个疗程。

# 4.2.6 艾灰处理

燃烧过程中会产生艾灰，及时清理艾灰至带水的弯盘，勿掉落病人身上。

# 4.3施术后处理

灸后注意保持局部皮肤清洁干燥，无需特殊护理。  
施灸后处理还应符合GB/T21709.1-2008 的相关规定。

# 5注意事项

5.1 施灸前应告知患者该技术的治疗目的及简单操作过程，并征得患者同意。5.2 艾灸前后嘱患者饮服少量温开水。  
5.3 发泡后局部灸疮当日勿沾水，预防感染。  
5.4 幼儿患者皮肤稚嫩，应注意控制艾灸量。  
5.5 燃烧过程中注意防止线香灰烬掉落而烫伤皮肤或烧坏衣被，治疗结束应及时熄灭线香。  
5.6 患者在接受艾灸治疗时一般不会发生晕灸现象；如发生晕灸现象，应按照GB/T21709.1-2008的相关规定进行处理。

# 6禁忌

6.1 中暑、高血压危象、肺结核晚期大量咯血、高热炎性疾病、局部疮疡等情况

不宜施灸；脑出血急性期者慎用精灸。

6.3 孕妇不宜施灸。

临床适应症

根据前期文献与临床调研，列举如下灵龟灸适应症：

1．痛症：头痛、颈痛、面痛、肩痛、腰椎间盘突出症、膝骨性关节炎、痛风性关节炎、痛经、产后身痛等；2．脑病：中风、面瘫、眩晕、失眠、抑郁、焦虑、慢性疲劳综合征等；3.肝胆脾胃病症：腹胀、胃胀、胃下垂、脏器下垂、呕吐、呃逆、消化不良、肠易激综合征、慢性胃炎、黄疸等；4.肺系及过敏病症：哮喘、支气管炎、咳嗽、过敏性鼻炎、鼻軌、鼻渊、荨麻疹等；5.代谢内分泌疾病：肥胖症、高血脂症、高尿酸血症、糖耐量异常等;6.妇儿病症：月经病、围绝经期综合征、子宫肌瘤、多囊卵巢综合征、妇科恶性肿瘤术后或放化疗后调理、小儿抽动障碍、小儿遗尿、小儿发育不良、小儿腹泻等；

7.养生保健：预防中风、改善体质等。
