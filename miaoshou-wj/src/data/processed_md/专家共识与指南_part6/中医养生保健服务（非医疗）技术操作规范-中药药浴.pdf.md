# 引言

《中医养生保健服务（非医疗）技术操作规范中药药浴》（以下简称《规范》）可用于指导和规范中药药浴操作在养生保健服务中的应用，为中医养生保健服务(非医疗)机构提供中药药浴操作的技术操作规范。

中药药浴是运用中医理论，将中药与洗浴相结合的方法。其历史悠久，源远流长，是中医药文化之瑰宝。药浴是通过全身肌表或局部，使中药成分渗透入肌体，由表及里，内达脏腑，具有改善血液循环，调节全身或局部的功能。

2018年颁布的《中医治未病技术操作规范中药药浴》（T/CACM1105—2018）主要适用于医疗机构。随着中药药浴疗法在中医养生保健服务(非医疗)机构的需求与发展，已形成一定规模的市场。修订和颁布适用于中医养生保健服务(非医疗)机构的标准，能更好地指导中医养生保健服务(非医疗)机构使用中药药浴操作，有利于该行业健康发展。

# 中医养生保健服务 （非医疗）技术操作规范中药药浴

# 1范围

本文件规定了中医养生保健服务(非医疗)机构成人中药药浴的术语和定义、操作步骤与要求、注意事项及禁忌。

本文件适用于在市场监管部门登记的经营范围为中医养生保健服务(非医疗)、提供中医养生保健服务的非医疗机构。

本文件适用于具有中医药类相关专业背景，或者接受过中医养生保健专业培训并具备相关知识和技能的中医养生保健服务人员。

本文件适用于单人单浴。

# 3术语和定义

下列术语和定义适用于本文件。

3.1

中药药浴 herbal bath of TCM

以中医理论为指导，将中药煎汤兑入适量温水中浸泡全身，发挥药物与水的双重作用，达到养生保健目的的一种泡浴方法。

3.2

全身浴 balneum to the skin应用中药浴液对全身进行浸泡的一种药浴方法。

3.3中药浴液 TCM bath liquid中药煎出液与水以一定比例进行混合后所形成的溶液。

3.4浴器 implement中药药浴的器具。包括浴桶、浴池等。

3.5 晕浴 fainting

在药浴过程中的异常表现，轻者表现为头晕、恶心呕吐、胸闷心慌、气短乏力；重者面色苍白、大汗淋漓，四肢厥冷，甚至晕蕨等。

# 4规范性技术要素

# 4.1施术前准备

# 4.1.1 中药饮片

4.1.1.1中药饮片的使用应符合《中华人民共和国药典》的规定。4.1.1.2中药药液的煎煮方法及场地应符合《医疗机构中药煎药室管理规范》的规定。4.1.1.3预先制备中药煎液备用。亚健康人群单人单次中药饮片最小用量 210g，健康人群单人单次中药饮片最小用量190g，或每百公斤水用药不低于60g中药饮片。

# 4.1.2 器材准备

选用规格：推荐使用长\*宽\*高为1.5m\*0.6m\*0.8m木质或竹质浴桶，并少量配备较大及较小规格的浴桶，可根据浴器大小适当调整中药饮片最小用量，每百公斤水用药不低于 60g。浴区应配有水温调节装置及温度测量仪器。

# 4.1.3 从业人员管理

可参考《GB37487-2019公共场所卫生管理规范》中“6从业人员卫生”相关要求。

# 4.1.4 体位选择

取坐位或半卧位，液面应在颈部以下。受术者感觉舒适，施术者便于操作、观察。

# 4.1.5 环境要求

4.1.5.1 设有相对独立的接待区、更衣区、休息区、浴区、中药煎液制备区。  
4.1.5.2 配备防滑设施与防跌倒标识。  
4.1.5.3环境宜安静，避免噪音。室内干净、舒适、无异味，保持房间空气流通，光线柔和，光照充足。  
4.1.5.4室内温度（30±3）℃，湿度（60±20）%。  
4.1.5.5 用水应符合GB 5749-2022生活饮用水卫生标准。  
4.1.5.6配备应急专员、应急场所、应急设备。应急专员提前接受应急培训，具备一定应急能力；应急场所，保持房间空气流通；宜配备4L以上氧气瓶、担架、电子血压计、体温计等便于操作的急救用品。  
4.1.5.7配备一次性浴具套。  
4.1.5.8在开展药浴场所显著位置安放药浴宣传板，宣传板内介绍药浴文化、本文件涉及的注意事项、禁忌症等项并注明来源，同时提醒受术者认真阅读。

# 4.1.6 消毒

可参考《GB 37489.4-2019公共场所设计卫生规范第4部分：沐浴场所》中5.3清洗消毒间及5.4消毒剂专用库房的相关要求。

# 4.2施术方法

4.2.1浴桶内注入400mm 深的清水，根据受术者需求初始水温设定在（40±2）℃。将预制好的中药煎液缓缓倒入浴桶内调和，制备中药浴液。并配备水温调节装置，使水温保持在一定范围内。

4.2.2受术者应先淋浴洁身后，进入浴桶。取坐位或半卧位，液面可至颈部，以舒适为宜。浴中受术者可根据需要自行调节水温，以舒适为宜。单次泡浴时间（20±10）min，受术者也可根据需要离开浴桶休息，也可根据需要适量饮水。药浴结束后，宜缓慢起身，离开浴桶。4.2.3在尊重受术者隐私和保证受术者安全的前提下适度巡视。

# 4.3施术后处理

4.3.1 药浴后，可温水冲洗。  
4.3.2 药浴后，应到更衣区适度休息，无不适表现，方可离开。  
4.3.3 药浴后，皮肤多会发红，属正常反应。  
4.3.4 药浴后，应注意休息、保暖，避免感受风寒。  
4.3.5 药浴后，宜适量补充水分和少量易消化食物。

# 5注意事项

5.1 应在餐后1～2小时内进行。  
5.2 体质虚弱及初次接受药浴者，药浴时间宜短。  
5.3 饮酒、大量出汗、过度劳累后不宜进行药浴。  
5.4 皮肤过敏、破损者慎用。  
5.5 严格做到一人一浴一消毒。  
5.6 泡浴过程中不宜使用沐浴液等日化产品。  
5.7 如出现晕浴、烫伤、汗出过多及皮肤红疹、瘙痒等表现，及时处理（可参考附录A）。  
5.8 泡浴过程中应注意防滑跌倒。

# 6禁忌症

6.1女性月经期禁用。  
6.2 不明原因疼痛、高热者禁用。  
6.3 患有严重高血压（血压≥150／90mmHg）或低血压（血压≤90／60mmHg）者禁用。  
6.4 重度精神性疾病、癫痫及抽搐等处于发作期，症及不合作者禁用。  
6.5 有不明原因肿块及局部水泡者禁用。  
6.6 大范围感染性病灶并已化脓破溃、感染及皮肤有开放性创口者禁用。  
6.7 软组织损伤者48小时内禁用。  
6.8 严重心肺功能不全或低下者禁用。  
6.9 高龄、孕妇、温度感应迟缓、局部麻木、语言障碍、认知障碍者禁用。  
6.10 有出血倾向疾病及凝血功能异常者禁用。  
6.11有传染性疾病者禁用。

# 7异常情况及处理方法

# 7.1 晕浴

# 7.1.1异常表现

在药浴过程中，轻者表现为头晕、恶心呕吐、胸闷心悸、气短乏力；重者面色苍白、大

汗淋漓，四肢厥冷，甚至晕厥等。

# 7.1.2 处理方法

立即停止药浴，将受术者移至阴凉通风处安静平卧，注意保暖，饮温开水或糖水；重者应尽快送医。

# 7.1.3 预防措施

7.1.3.1对于初次泡浴和精神紧张者，应先做好解释工作。

7.1.3.2对于有初次泡浴或晕厥史受术者，应选取适宜体位，水温不宜过高，药浴时间不宜过长，药浴过程中可离开水面休息。可在受术者能够耐受的情况下，逐步提高水温，延长药浴时间。

7.1.3.3饥饿、过饱、劳累、醉酒、极度紧张时不应进行药浴。

7.1.3.4在药浴过程中应有专人巡视。随时注意观察受术者的神色、询问受术者感觉，一旦出现头晕、恶心等表现，应及时采取处理措施，必要时送医救治。

# 7.2 汗出过多

# 7.2.1异常表现

受术者汗出过多，甚至大汗淋漓，不能自止，严重者发生休克、意识不清。此种异常表现可出现于药浴中或单次药浴后，亦可出现在疗程中多次药浴之后。

# 7.2.2 处理方法

立即停止药浴，将受术者放置凉爽处，大量饮温水，必要时送医救治。

# 7.2.3 预防措施

药浴时水温不宜过高，并缩短泡浴时间；不宜频率过高，时间过长。

# 7.3 烫伤

# 7.3.1 异常表现

进行药浴的过程中，受术者出现皮肤红肿、水疱、脱皮等现象，严重者会造成深部组织坏死。

# 7.3.2 处理方法

立即停止药浴，受术者患处涂抹烫伤膏，严重者应立即送医。

# 7.3.3 预防措施

7.3.3.1初次泡浴时，需询问受术者有无感觉神经障碍或皮肤敏感等表现，做好预防烫伤的措施。  
7.3.3.2药浴液的初始温度应控制在（40±2）℃范围内。
