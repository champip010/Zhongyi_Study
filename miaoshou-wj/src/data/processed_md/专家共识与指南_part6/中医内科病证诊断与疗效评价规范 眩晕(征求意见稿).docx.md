# 引盲

2Y/T001.1-94 <中医内科病正珍断寂数析准) 中的弦暈 自度用以来対中医帖床医寂贋量坪定科学研究及教学管理疫揮了釈桜作用。 中医帖床研究、新茹帖床式弘、珍寂指南制汀、 臨床路徭等工作宍施均需要共沢度高、 可操作性弼的中医病正珍断与寂数坪併却范作参考依据。
