# 腧 穴 主 治

# 1 范围

本标准确定了 / — 《腧穴名称与定位》 个经穴中 穴以及 个经外奇穴中穴的常用适应症。本标准所确定的腧穴主治为基础主治,不包括基于特殊针灸法、特殊操作的腧穴治疗病症。

本标准适用于针灸腧穴的医疗、教学、科研等。

# 附 录 A(规范性附录)确定腧穴主治的依据、方法

# A1 确定腧穴主治的依据

以历代针灸文献为主要依据。古今针灸文献的选择以腧穴文献为主,治疗文献为辅。

通过研究将传世古代腧穴文献按其传承关系划分为四大分支,即“黄帝明堂经”系列、“甄权等诸家明堂 系列 窦太师针灸 系列 针灸聚英 系列 每一支文献均以源头文献为主要依据

现代腧穴文献,选用 版高等中医院校统编教材《针灸学》及对外培训教材《中国针灸学》为代表。

古代治疗文献以单穴方与其加减方为主 多穴方为辅 现代临床应用文献 选用单穴为主 或辅以个配穴,有明确的治疗病症、操作方法和疗程,且有较多的病例数和客观的疗效分析。

# A2 确定腧穴主治的方法

综合考察古今文献,进行正误、取精、补缺、规范的整理研究。当历代文献关于腧穴主治的记载有分歧时,注重多种方法的综合应用与多种文献的综合利用。基本思路为:考察单一腧穴时,注意与相关腧穴加以比较;考察文献时,既注重腧穴“主治”内容,又注意考察与主治直接或间接相关的所有方面。

# 附 录 B

(资料性附录) 腧穴基础主治的界定

针灸效应的产生取决于穴位和刺激两个方面。严格地说,还应当包括机体的反应性这一因素。从刺激方面来分析,同一穴位,如果采用不同的针法、不同的针刺层次、不同的刺激量,会产生不同的效应。以往的实验结果提示:针灸穴位在不同的刺激条件下可通过不同的机制表现出特异性与非特异性两种不同的效应。

腧穴主治的特异性主要取决于腧穴所在的部位,所有腧穴都可以治疗其所在部位及其相邻近部位的病症 因此位于不同区域腧穴的主治病症不同 另一方面四肢肘膝以下的特定穴除了上述近治作用外,还可治疗相关经脉、络脉及其所属脏腑的病症,表现出腧穴主治的经络特异性。

将针灸效应中主要由腧穴起作用的主治称为 基础主治 并将其条件定义为 以普通针法 常规刺激量与刺激时间、针灸单个穴位,针对一般敏感程度的人体所产生的治疗作用。本标准所规定的腧穴主治为基础主治,不包括基于特殊针灸法、特殊操作的腧穴治疗病症。

# 附 录 C(规范性附录)腧穴主治规范化表述

# C1 腧穴主治的表述体例

C11 以“常用于……”的表述形式概括腧穴的治疗、保健方面的应用。  
C12 主治病症按其内在联系分为若干组,不同组主治病症间用分号隔开,同一组主治病症间用逗号隔开 相关联症状 多为症候群 间用顿号隔开 新增的主治病症置于最后 之前用句号与其他主治病症相隔。  
C13 从古代文献筛选出的病症采用中医病名、症状名,但对于某些用现代医学病名表述更明确者,或基于现代针灸临床新总结的治疗病症,则采用现代医学病名。  
C14 古代针灸文献对于腧穴主治的表述,以症状描述为主,而目前关于症状名,还没有相应的国家标准或行业标准,此次腧穴主治病症的表述,优先采用国家有关诊疗标准术语的相关条目,现行标准没有收录的病症名称,参考各版高等中医院校统编《针灸学》教材确定。  
C15 十二经之五输穴、络穴,五脏六腑之俞、募穴,六腑下合穴等特定穴下,皆注明其特定属性,以反映其与腧穴主治的相关性联系。

# C2 腧穴主治表述体例的说明

古代针灸腧穴文献对腧穴主治的描述多为症状,而不是具体的病。即使某些腧穴的主治中出现病名,也多伴随具有辨证意义的特异性症状。古代医家总是不断地从对不同病的针灸治疗中概括、提炼出相关腧穴的特异性主治症 腧穴主治的演变遵循 从具体到一般 的总规律 即不断地从对所主治的一个个具体病中提炼出共通的基础症状或特异性症状,不论什么病只要见此症状,即可取此穴。即“有是症用是穴”。因此对于那些临床应用较广的腧穴,力求提炼出基础症状,而尽可能淡化具体病的列举。然而具体到每一个腧穴,因其主治范围大小不一,临床应用的频度不同,人们对其主治特点把握的程度也就不同,难以通过一次研究就达到理想状态。

同一个腧穴的临床应用,除了用于病症的治疗外,还用于疾病的预防以及保健、强身,这些正是针灸的优势所在,应加以总结并恰当地反映,因此在表述上不用“主治 病症”,而采用“常用于 ”来涵盖腧穴治疗与保健的双重应用。腧穴的保健作用一般只列举应用最广的,而针灸腧穴所产生的疾病预防作用一般与其所治疗的作用相对应,因此不专门列举腧穴的预防病症。
