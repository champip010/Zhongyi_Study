# 结直肠癌疼痛中西医结合诊疗指南编制说明

# 一、工作简况

# 1.任务背景

癌性疼痛是肿瘤常见的主要症状之一。据统计，约 1/3 接受癌症治疗的患者出现疼痛，超过2/3的进展期癌症患者出现疼痛，约50%疼痛为中度至重度，其中约 30%为重度疼痛。癌痛给患者造成极大的身心负担，严重影响了患者的生活质量。在结直肠癌疼痛的规范化治疗中，我们发现中药的使用较为普遍。西药治疗癌痛，疗效确切，起效快，但具有比较明显的毒副作用。中医药治疗疼痛历史悠久，古方众多。近年来国内大量文献报道了采用中医药内服、外用、针灸等方法开展对癌性疼痛的治疗，具有毒副作用小、使用安全、无成瘾性和无戒断性等优势。至今国内外尚无在癌痛规范治疗如何使用中药方面的共识或指南。如何体现我国中西医结合的治疗特色，做到既发挥中医特长，又根据目前的研究证据等级合理使用中药止痛，把止痛中药与癌痛三阶梯治疗的有机结合，根据疼痛性质合理使用中成药，是本诊疗方案的需要解决的问题。

# 2.任务来源

本任务来源于国家中医药管理局重大疑难疾病中西医结合试点项目（结直肠癌）—结直肠癌疼痛子项目组，由北京大学肿瘤医院、中国中医科学院西苑医院两家机构完成，立项时间是2020年3月9日。

# 3.指南起草单位

北京大学肿瘤医院、中国中医科学院西苑医院。

# 2.指南的关键技术内容

本指南的主要临床问题是中西医结合治疗结直肠癌疼痛是否优于单纯西医治疗？主要评价指标为降低疼痛评分、增加疼痛缓解率、降低不良反应、提高生活质量。本方案包括结直肠癌疼痛的病因及分类、结直肠癌疼痛的评估、结直肠癌疼痛的中西医结合治疗等方面。证据分级标准参考刘建平教授在《中医药循证医学》中提出的传统医学证据体的构成及证据分级的建议，本指南结合临床实际作适当修订，推荐意见分级包括：强推荐（或强不推荐）、弱推荐（或弱不推荐）、无明确推荐。结直肠癌疼痛的中医辨证分型分为湿热证、血瘀证、气滞证、阳虚证、阴虚证，主要方剂分别为槐角丸、血府逐瘀汤、六磨饮子、理中汤、芍药甘草汤。结直肠癌疼痛常用中成药分为活血消癥止痛、解毒消癥止痛、理气止痛，清热止痛、散寒止痛。常用剂型包括口服剂型、注射剂、外用剂型等。在临床应用时，应该首选口服剂型，在中医辨证前提下酌情选用。本方案推荐依据包括：NCCN成人癌痛指南（2020版）、北京市癌症疼痛治疗规范（2017版）、难治性癌痛专家共识（2017 年版）、中华人民共和国药典（2015 版）、中华人民共和国药典.临床用药须知.中药成方制剂卷（2010版）、中医经典著作、临床研究文献等。

# 4.指南问题

# （1）指南问题形成方法

临床问题论证会（附件6）

2020年12月27日于北京裕龙酒店进行了该指南临床问题专家论证会,指南工作组将问题清单递交给专家组,由专家组对指南问题进行共识。

1.原则

(1)采用名义组法共识会议，会议由第三方人员主持;(2)参与临床问题论证的专家为指南专家组成员(不少于2/3);(3)超过参会专家70%以上同意的指南问题,即为确定的指南问题;(4)根据实际情况，确定是否存在待确定的指南问题，原则上 50%-70%专家同意的问题为待确定指南问题;(5)会议投票最多不得超过3 轮。

# 2.流程

（1）指南主要起草人或秘书介绍指南问题形成的过程；（2）指南主要起草人解读指南问题清单；（3）专家组对指南问题进行投票；（4）工作组对投票结果进行统计，公布本轮投票已确定的指南问题和待确定的指南问题；（5）专家组对待确定的指南问题逐一进行发言讨论。

# 3.结果

共有28名专家与会,其中参与临床问题论证的专家为指南专家组成员(不少于2/3),每位专家基于其临床经验，对其合理性进行评价，评价标准为“合理——纳入指南”；“不确定——修改后再讨论”；“不合理——可不纳入指南”。通过专家讨论及论证表结果整合后,得出本指南要解决的最终问题:

1、中医药在结直肠癌疼痛治疗中的主要作用

3、中医药在结直肠癌疼痛治疗中的方法  
4、中医药在结直肠癌疼痛治疗的评价方法  
5、中医药如何与癌痛三阶梯治疗相结合治疗  
6、结直肠癌疼痛的中医辨证治疗  
7、根据疼痛病因使用中成药

一、无中医药治疗结直肠癌疼痛的文献，有针灸治疗结直肠癌术后恢复和奥沙利铂末梢神经毒性的文献，无结直肠癌疼痛的针灸文献。结直肠癌疼痛的主要中医辨证及代表方剂：未检索到关于结直肠癌疼痛的中医辨证的临床研究文献。所以针灸、中医辨证方药治疗结直肠癌疼痛证据不足。

二、中成药治疗结直肠癌疼痛的文献较多，包括中成药联合化疗，中成药联合镇痛西药，中成药单独应用等，结局指标包括疼痛评分、疼痛缓解率、不良反应发生率、生活质量。中成药治疗结直肠癌疼痛证据详细如下：

# 1、复方斑蝥胶囊

复方斑蝥胶囊，无专门针对结直肠癌疼痛的研究，有一篇随机对照研究中提示复方斑蝥胶囊联合化疗对217例中晚期恶性肿瘤具有减轻疼痛的作用，其中大肠癌疼痛患者29例。

# 2、复方夏天无片

复方夏天无片，无专门针对结直肠癌疼痛的研究，有一篇随机对照研究中提示对于轻度癌痛76例复方夏天无片与双氯芬酸纳肠溶片疗效相当，副作用减少，其中结肠癌19例。有一篇对照研究中提示胃肠道肿瘤疼痛患者30例，与常规化疗相比复方夏天无片联合化疗止痛效果明显，其中结直肠癌15例。

# 3、华蟾素

关于华蟾素治疗结直肠疼痛的研究15篇，其中华蟾素治疗癌痛Meta分析1篇，华蟾素胶囊联合化疗治疗晚期结直肠癌的系统评价与Meta分析2篇。许晶等做了华蟾素治疗癌痛有效性和安全性的系统评价和 Meta 分析，最终纳入 10 个研究，总样本量为 1 293 例，试验组 648 例，对照组 645 例，所纳入研究总体质量普遍偏低。Meta 分析结果显示: 华蟾素联合西药止痛药治疗癌痛在近期疼痛缓解效果、提高生活质量和便秘、恶心呕吐、头晕、嗜睡、厌食等不良反应发生方面优于单用西药止痛药，并且具有止痛起效时间短，维持时间长的优势，但不能降低排尿困难的发生率。金金等做了华蟾素胶囊联合化疗治疗晚期结直肠癌的系统评价与Meta分析，共纳入 9 项 RCTs涉及 658 例结直肠癌患者，试验组均为华蟾素胶囊联合化疗药物，对照组为单用化疗药物。Meta 分析结果显示，华蟾素胶囊联合化疗药物在近期有效率、患者生活质量、疼痛缓解率方面明显优于单纯使用化疗药物，并可降低白细胞毒性、胃肠不良反应、肝功能异常、手足麻木的发生率。彭莹莹等做了华蟾素胶囊联合奥沙利铂化疗方案治疗结直肠癌的Meta 分析，共纳入 8 个 RCT研究，505 例结直肠癌患者，Meta 分析结果显示华蟾素胶囊联合奥沙利铂化疗方案治疗结直肠癌的临床疗效、生活质量、疼痛缓解优于对照组，不良反应低于对照组。RCT研究12篇，共纳入406肠癌疼痛患者。其中3篇为化疗联合华蟾素胶囊对疼痛控制效果优于单纯化疗，7篇为西医镇痛治疗联合华蟾素优于单纯西医镇痛治疗，1篇为伊班膦酸联合华蟾素注射液镇痛效果优于单纯伊班膦酸骨转移疼痛，1篇为华蟾素注射液与美施康定对比镇痛效果相当，但生活质量更优。

4、元胡止痛片（颗粒）

元胡止痛片（颗粒），无专门针对结直肠癌疼痛研究，有一篇RCT研究提示元胡止痛胶囊联合吗啡缓释片对比单纯吗啡缓释片，镇痛效果相当，但元胡止痛片能够降低吗啡缓释片使用剂量及不良反应，具体病种不详。证据等级Ⅱ级，推荐意见为弱推荐。

# 5、桂参镇痛合剂

无专门针对结直肠癌疼痛研究，有3篇RCT研究针对癌痛的随机对照研究，提及肠癌或者胃肠道肿瘤。研究显示桂参止痛合剂具有明显镇痛作用，对中晚期癌性疼痛患者止痛效果优于强痛定片，且有改善肿瘤患者临床症状、提高生活质量的作用，无恶心呕吐、便秘副作用。

# 6、草乌甲素片（胶丸）

无专门针对结直肠癌疼痛研究，1篇单独应用草乌甲素治疗轻中度癌痛的随机对照研究，纳入肠癌13例，研究显示对于轻中度癌痛草乌甲素镇痛效果不劣于芬必得、曲马多，副作用更小，生活质量更佳。

# 7、新癀片

无专门针对结直肠癌疼痛研究，有一篇RCT研究联合美施康定治疗重度癌痛96例的随机对照研究，纳入结直肠癌8例，结果显示应用新癀片配合美施康定治疗重度癌痛患者可增加镇痛疗效，减轻便秘等副反应。有1篇病例观察单独应用新癀片治疗92例癌痛，纳入直肠癌4例，结果显示新瘤片对实体瘤所致的疼痛均有较好的效果，尤其是Ⅰ、Ⅱ级疼痛患者，不但起效较快，止痛维持时间较长，未发现恶心 、呕吐、腹痛、腹泻等副作用，但上消化道出血者禁用，胃、十二指肠溃疡患者慎用。

# 8、复方苦参注射液

关于复方苦参注射液治疗结直肠疼痛的研究7篇，其中治疗癌痛Meta分析3篇，治疗结直肠疼痛RCT研究4篇。李德辉等通过对纳入的 9 篇文献、997例癌痛患者进行 Meta 分析，结果显示：与单纯用奥施康定治疗癌痛比较，复方苦参注射液联合奥施康定的镇痛有效率增加，病人生活质量提高，恶心呕吐等不良反应发生率降低。黄奕雪等共纳入 21 篇随机对照试验，包括1903个患者，复方苦参注射液联合止痛药对于癌痛的疗效优于单纯使用止痛药，且能够辅助降低单纯使用止痛药的不良反应。陈芝强等纳入7个随机对照试验，共558名患者，Meta分析结果显示：复方苦参注射液联合唑来膦酸治疗骨性癌痛患者在疼痛缓解率方面优于单用唑来膦酸治疗，且联合用药未明显增加毒副反应。一篇研究显示复方苦参注射液联合化疗治疗结直肠癌骨转移疼痛与单纯化疗组比较，疼痛有效率、生活质量均占优，纳入大肠癌82例。一篇研究显示复方苦参注射液联合奥施康定治疗中晚期肠癌癌性疼痛患者能有效减轻疼痛程度和提高生存质量，效果优于单纯奥施康定治疗，不良反应发生率相似，纳入大肠癌64例。另一篇文献研究显示复方苦参注射液联合奥施康定治疗中晚期肠癌癌性疼痛疼痛评分、疼痛缓解率、生活质量均优于单纯奥施康定组，纳入大肠癌100例。一篇研究应用艾灸配合复方苦参注射液治疗的直肠癌性疼痛，疼痛缓解率、疼痛强度差、生命质量评分均高于西药止痛组，纳入直肠癌60例。

9、康莱特注射液（胶囊）

关于康莱特注射液（胶囊）治疗结直肠疼痛的研究9篇，1篇meta分析，8篇RCT研究。廖剑锋等做了康莱特联合化疗与单纯化疗治疗结直肠癌的meta分析，纳入9篇研究，共610例患者，结果显示，康莱特联合化疗组治疗结直肠癌的疼痛总缓解率优于单独化疗治疗组。其中1篇研究采用奥施康定联合康莱特注射液治疗中晚期癌痛患者，与单独使用奥施康定相比，不仅有更好的镇痛作用，并可明显改善患者的生存质量，纳入直肠癌10例。一篇研究采用康莱特注射液+最佳支持治疗生活质量和癌痛缓解率优于最佳支持治疗，纳入肠癌16例。一篇康莱特注射液+营养支持治疗+三阶梯阵痛治疗对比营养支持治疗+三阶梯阵痛治疗，治疗组与对照组比较疼痛缓解率、KPS评分、体重评分有明显提高。一篇康莱特注射液控制癌痛 及提高晚期癌症患者生存质量的多中心Ⅲ期研究，纳入14 家肿瘤医院 376 例晚期癌症，其中伴有癌症疼痛患者 328 例, 占 87.23 % ( 328/376)，研究显示康莱特注射液癌症疼痛控制总有效率达 80.49 % ( 部分缓解 56.10%,完全缓解 24.39 %) , 用药结束后疼痛缓解仍可维持 1～7 天以上, 且无成瘾性，纳入大肠癌28例。一篇研究显示康莱特与吗啡缓释片合用可明显提高重度癌性疼痛患者的镇痛效果，减少吗啡缓释片用量，改善患者生活质量。一篇研究显示最佳支持治疗联合康莱特能够有效改善老年晚期消化道肿瘤的生活质量、缓解疼痛，纳入大肠癌15例。一篇研究显示康莱特注射液联合FOLFOX4方案治疗晚期结直肠癌不仅能增加临床获益率，同时还能改善患者的身体状况和生活质量及缓解疼痛、降低不良反应的发生率，纳入大肠癌60例。一篇研究显示康莱特注射液联合FOLFOX4方案治疗晚期结直肠癌肝转移，联合化疗组在缓解患者的临床症状疗效较好, KPS评分和疼痛缓解率均优于单纯化疗组, 而且联合化疗组化疗的不良反应也相应较轻，纳入大肠癌66例。

# 10、湿润烧伤膏

RCT研究1篇，文献研究显示，湿润烧伤膏治疗放射性皮炎、疼痛缓解情况均优于红霉素软膏对照组，其中纳入直肠癌11例。
