# 《2 级高血压中西医结合诊疗指南》指南编制说明

# 一、工作简况

# 1.任务背景

高血压是临床常见慢性病，也是心脑血管病最主要的危险因素。2002—2015年间我国开展了三次高血压抽样调查（2002 年、2012 年、2015 年），粗患病率分别为 18.8%、25.2%、27.9%，人群高血压患病率总体呈上升趋势。据此推算，约四分之一的成人是高血压患者，总患病人数达 2.45 亿。2017 年，中国有 254万人死于高收缩压，其中 95.7%死于心血管病。2013 年由高血压带来的直接经济负担达 2103 亿元，占中国卫生总费用的 6.61%,给社会及经济带来沉重负担，高血压防治任务任重而道远。中西医结合是当前临床医学的发展趋势，推动了医学水平的整体提高。2 级高血压在中医学属于“眩晕”、“头痛”、“头风”和“风眩”的范畴，是高血压病控制的重点，是疾病进展的关键阶段。中西医结合治疗已经取得了较好的成绩，在医学领域占据重要地位。但目前在中西医结合治疗 2 级高血压方面缺乏必要的基于“中西医结合诊疗”理念的可用于指导临床诊疗的标准规范。为发挥中西医结合治疗 2 级高血压的优势，提高中医药防治 2级高血压的水平，规范中医药在治疗 2 级高血压中的应用，山东中医药大学附属医院在中华中医药学会的指导与领导下组织相关专家，参照国际最新的临床实践指南制订方法，并在相关法律法规和技术文件指导的框架下，以中西医结合诊疗理念为核心，以传统中医辨证论治及西药常规治疗为基本点，结合循证医学原理，在古今文献回顾分析、临床流行病学调查、专家咨询等系统研究工作基础上，对 2 级高血压的流行病学特点及危害、病因病机特点、证候类型、中医药干预措施、2 级高血压中西医结合诊疗等进行了梳理、归纳、总结，征询行业内部分专家意见，形成了《2级高血压中西医结合诊疗指南》研究框架，以期为从事高血压防治的中医、中西医结合临床医师提供指导性意见。

# （七）推荐意见

# 1.推荐意见形成方法

推荐意见考虑利弊权衡、证据质量高低、患者的偏好和价值观、成本（资源分配）等因素，由名义组会议专家综合权衡后作出推荐强度。本共识基于 GRADE网格，分三轮进行投票，达成共识规则如下：除了“中间”格以外的任何 1 格票数超过 50%，则达成共识，可直接确定推荐方向及强度；若“中间”格某一侧两格总票数超过 70%，则达成共识，可确定推荐方向，推荐强度为“弱”。其余情况视为未达成共识，推荐意见进入下一轮投票，投票不得超过三轮。最终，由共识秘书处统计并汇总推荐意见及强度。

# 四、与国内外同类指南的对比和最新指南采用情况

（一）描述国内外是否有已发布且正在实施中的同领域指南？本指南与其相比，有什么区别？

国内最权威的治疗高血压的指南为中华医学会发布的《中国高血压防治指南》2，最新版为 018 年修订版。该指南参考了世界卫生组织、中华医学会指南制订流程,对指南重要内容、证据级别及推荐类型进行了评估。在借鉴国际先进经验的基础上,结合我国高血压防治工作实践,充分应用中国证据,形成具有中国特色的高血压预防干预、诊断评估、分类分层、治疗管理指南。

在中医方面，由中华中医药学会发布的《高血压中医诊疗专家共识》，最新版为 2019 年版。该共识参考了《中国高血压防治指南》及《2018 ESC/ESH

Guidelines for the management of arterial hypertension》，以传统中医辨证论治与方证辨证理论为指导，在梳理历代高血压文献的基础上，结合现代循证医学理念，对高血压的关键病机、证候特点、经典名方辨证用药规律进行了总结讨论。

本指南与其相较，有共性的基础，但在传统辩证的基础上以及在现代医学基础之上，更加突出了中西医结合治疗高血压的优势点，强调中西医结合诊疗对高血压病的优化管理和治疗措施，更加体现中西医结合的优势和特色。

# （二）是否引用相关指南？引用的内容是什么？

本共识的部分内容借鉴和引用了 2018 年《中国高血压防治指南》，其中诊断标准与其保持高度一致；其次五大类降压药物的推荐；最后对高血压患者的靶目标管理、合并靶器官损害、危险因素的控制以及生活方式的干预也有适当的借鉴和引用。

# 五、与现行强制性国家标准或政策法规的关系

本项目工作组研究形成的《2级高血压中西医结合诊疗指南》与现行法律、法规和强制性标准没有冲突，并且在编制过程中严格遵循已有的国际、国内标准，使文本内容符合规范，言之有据。

# 六、代表性分歧意见的处理经过和依据

本共识制订过程中，无重大分歧意见。

（1）借助中医药学会等多个平台，利用传统和新媒体进行报导

本项目发起人在参加或组织相关领域的学术会议、国家级或省级中医药学会分会年会时，将采用主题报告、会议交流等多种形式，积极分享和推广将该团体标准。

由发起单位组织人员，积极撰写与该团体标准有关的学术论文，在专业杂志发表，提高其影响力和推广效果。
