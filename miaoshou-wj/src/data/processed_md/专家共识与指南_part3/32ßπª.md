# 中华中医药学会团体标准“2级高血压中西医结合诊疗指南”推广应用方案

# 一、该团体标准推广措施

1、借助中医药学会等多个平台，利用传统和新媒体进行报导

该诊疗方案公示后，对该诊疗方案进行发布。同时，请报刊媒体及新媒体对该诊疗方案的编制情况及意义进行报导宣传。此外，还可将该诊疗方案上传至网络，方便有需求者随时下载使用，以进一步促进其在全国范围内更快更好地的实施应用。

# 2、依托中华中医药学会、山东中医药学会推广带动

在该团体标准被公示后，依托中华中医药学会、山东省中医药学会的相关学术会议上发放纸质版及电子版团体标准，并进与该诊疗方案相关的专题学术报告。

3、依托本技术规范发起单位--医疗机构推广带动

山东中医药大学附属医院为国家高血压临床研究基地，同时，包括天津中医药大学在内的多家单位作为本项目的发起单位，均为三级甲等中医医院。在该团体标准公示后，将率先响应该团体标准，以带动全国中医医疗机构、综合医疗机构等全面实施。每年应用试点围绕该团体标准举办多次专题学术活动或参观交流活动。

# 4、依托发起人推广带动

本项目发起人在参加或组织相关领域的学术会议、国家级或省级中医药学会分会年会时，将采用主题报告、会议交流等多种形式，积极分享和推广将该团体标准。

# 5、发表相关的学术论文

由发起单位组织人员，积极撰写与该团体标准有关的学术论文，在专业杂志发表，提高其影响力和推广效果。

# 二、该团体标准推广的预期目标

1、本团体标准的使用省份在全国一半以上；  
2、开展学术论坛的标准培训，每年1-2 次，每次培训规模100-300 人

# 三、该团体标准的推广机制

1、组建该团体推广工作小组，负责该团体标准的推广工作；  
2、筹措专项经费，保证推广工作的顺利进行。
