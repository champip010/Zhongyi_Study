Key words] Medical devices of cardiac electrophysiology; Assessment of comprehensive value; Arrhythmia; Expert consensus

国家卫生健康委于2019年6月发布《医疗机构医用耗材管理办法(试行)》[提出，医疗机构对医用耗材临床使用的安全性、有效性和经济性进行监测评估时，应按照“合法、安全、有效、适宜、经济”的原则。截至目前，由于国家尚未出台更细致的指导性文件，各医疗机构均按自行设计的本院医疗器械及医用耗材遴选管理制度进行管理，模式多样，缺乏行业参考标准。

在心脏学领域，电生理技术是目前心内科用于对心律失常进行诊断和治疗的主要手段。作为有源侵入的Ⅲ类医疗器械，电生理器械质量与患者安全和手术质量息息相关。电生理产品构成复杂，一台完整的电生理手术通常需由设备、软件和耗材相互配合方能完成。根据国家医疗保障局更新的“医保医用耗材分类与代码数据库”[2要求，仅消融导管三级分类包含电定位治疗导管、磁定位治疗导管和冷冻治疗导管，每个分类下进一步按照单或双弯、是否带压力感应、是否带微电极、孔数等进行细分，每个细分类别下有不同的品牌和规格，因此电生理产品种类较多。而电生理整体手术价格较高，如何遴选出符合临床需求和医院管理要求、价格合理的产品是医疗机构面临的重要课题。为此，中国医学装备协会应用评价分会联合中山大学药学院医药经济研究所，组织70余位临床、采购、医务等相关部门及工程学、卫生技术评估(healthtechnology assessment，HTA)领域专家，通过总结分析国内外电生理产品注册管理要求、循证医学证据、HTA评估方法等，结合医疗机构的临床实践和医用耗材管理经验，在充分了解现阶段国内电生理技术应用现状的基础上，共同制定了“心脏电生理医疗器械综合价值评估专家共识”(简称“共识”），旨在以国内临床实践中心脏电生理医疗器械现有技术为切入点，基于医疗机构医疗器械管理视角，从临床价值、经济价值、患者价值以及管理价值4个维度建立心脏电生理医疗器械综合价值评估体系，为心脏电生理医疗器械的科学评估、遴选使用和管理提供参考依据。

# 1心脏电生理医疗器械适用范围

针对用于心脏介入消融(射频消融和冷冻消融)手术中使得心脏靶组织凝固性坏死，从而使其失去电生理传导功能的电生理医疗器械。临床中电生理医疗器械人体侵入程度高，接触血液循环系统，其预期功能的实现往往需要标测设备、消融设备、消融导管和标测导管4类核心产品模块的相互配合。根据《医疗器械分类目录》（国家药品监督管理局2017年第104号公告)要求，各核心产品模块的注册管理类别均为Ⅲ类，对应的心脏电生理产品医疗器械分类见表1。

“共识”适用于可以独立完成心脏介入消融手术的电生理产品套装，主要包括但不限于标测设备、消融设备、消融导管和标测导管4类核心产品模块，且产品套装中包括的产品模块应满足以下两个条件之一：①注册证中明确套装中各个产品可以配合使用；②不同品牌产品匹配使用时，有明确的技术协议，并且匹配性经过充分验证，以确保及时掌握匹配使用方的设计变更。其他配套产品，如房间隔穿刺术所需针和可调弯和固定弯鞘、心内超声导管等，可纳入该产品套装中一并进行评估。

# 2心脏电生理医疗器械综合价值评估

# 2.1临床价值评估

# 2.1.1适用范围

(1)特殊人群的适用性4。评估产品是否适用于常见的特殊人群(如心脏解剖畸形患者、孕妇、儿童、心功能不全的患者等)。评估指标采用主观评估的方式。依据临床实践经验对上述人群的适用程度进行主观评估。

(2)获批适应证范围[5]。评估消融导管获国家药监局批准适应证的范围，包括阵发性房颤、持续性房颤、室上速、房速和(或)房扑、室速和(或)室早。若经批准适应证为房颤，则包括阵发性房颤和持续性房颤。指标采用客观评估的方式。依据消融导管的上市时间以及注册证载明的适用范围，对产品所适用的心律失常疾病的数量进行等级划分。

(3)复杂心律失常的适用性5。评估产品对治疗复杂心律失常(如持续性房颤、疤痕性心律失常、心肌病、多源性的、外科术后等复杂心律失常)的适用性。评估指标采用主观评估的方式。依据临床实践经验对上述复杂心律失常的适用程度进行主观评估。

# 2.1.2 安全性

评估使用产品于术中及术后30d内或围术期的主要并发症的发生比例[6-10。主要并发症：①局部血管相关并发症，出血、血肿、动静脉瘘、假性动脉瘤等；②心脏相关并发症，脉瓣返流、心脏穿孔、心包积液、心包压塞、窦房结损伤、房室传导阻滞、心房食道瘘、肺静脉狭窄、左房僵硬综合征；③系统血管相关并发症(脑血管不良事件)，血栓、空气栓塞、卒中及短暂性缺血发作；④神经相关并发症，膈神经麻痹；③死亡。

# 2.1.3有效性

(1)手术即刻成功率。评估使用产品治疗心律失常的手术即刻成功率[10,14-16]。不同心律失常的即刻成功率可依据各医疗机构自行制定的临床规范或参考如下定义[17]：①房颤，消融术后达到相对应术式终点要求；②室上速，消融术后即刻旁道阻断，药物和(或)电生理刺激不再诱发出临床心律失常；③房速及房扑，对于局灶房速消融后起源点消失，对于折返性房速(房扑)消融术后消融线双向阻滞，临床心律失常消失，药物和(或)电生理刺激不再诱发出临床心律失常；④室速及室早，消融术后临床心律失常消失或显著(≥80%)减少，药物和(或)电生理刺激不再诱发出临床心律失常。评估指标采用客观评估的方式。依据所检索的公开发表的文献数据或基于病案系统数据开展真实世界研究，对适应证内使用产品进行消融手术的即刻成功率水平高低进行等级划分，同时对所依据文献数据或真实世界研究的证据质量高低进行等级划分。真实世界研究应遵守相关规范指南进行研究设计和统计分析以满足统计学要求、控制混杂效应。

(2)手术远期成功率。评估使用产品治疗心律失常的手术远期成功率[8,18-20]。不同心律失常的远期成功率可依据各医疗机构自行制定的临床规范或参考如下定义：①房颤，至少随访12个月，房性快速心律失常不再发生或发生减少(不服用抗心律失常药物情况下)；②室上速，至少随访6个月，无临床心律失常复发；③房速及房扑，至少随访6个月，无临床心律失常复发；④室速及室早，至少随访6个月，临床心律失常无复发或显著(≥80%)减少。评估指标采用客观评估的方式。依据所检索的公开发表的文献数据，对适应证内使用产品进行消融手术，单次手术后满足最少随访时限的成功率水平高低进行等级划分，同时对所依据文献数据的证据质量高低进行等级划分。

# 2.1.4 产品关键性能

(1)建模精细度。评估使用产品对心脏精细结构图像的还原能力，即呈现的心脏模型图像与标准CT或

MRI影像的重合程度(建模能力需由系统软件配合标测导管实现)。评估指标采用客观评估的方式。依据产品的物理性能参数判断是否可以进行三维心脏建模。同时，依据公开发表的文献数据或第三方权威检测报告，评估心脏模型是否可以高度还原CT或MRI影像，并对文献数据的证据质量进行等级划分[21]。

(2)标测准确性。评估产品识别电信号的能力，是否高效、真实准确、且抗干扰[22-24]。标测准确性越高，产品识别不同心律失常机制的准确性越高，可指导操作医生更加精准定位靶点(标测能力需由系统软件配合标测导管实现)。评估指标采用客观评估的方式。依据标测导管的电极数量进行等级划分。同时，依据公开发表的文献数据或第三方权威检测报告，评估产品与其他同类产品相比识别心律失常机制的相对优劣势，并对文献数据的证据质量进行等级划分。

(3)消融程度控制能力。评估产品是否具有对消融功率(W)、温度(℃)及合适的贴靠压力(g)的指导和调控能力，从而真实反馈消融效果(消融损伤控制能力需由系统软件配合消融导管实现)[25]。评估指标采用客观评估和主观评估相结合的方式，依据产品的消融指导参数数量和临床实践经验进行等级划分。

(4)产品稳定性。评估产品在累积使用一段时间后，标测系统的定位精度是否能够维持产品技术要求。评估指标采用客观评估的方式。依据第三方权威检测报告，对可保证产品稳定性的累积手术台数进行等级划分。

(5)影像整合能力。评估产品(系统软件)整合影像，包括X射线、超声和CT影像(或MRI影像)的能力，整合的影像种类越多则整合能力越强，如果能整合在一个屏幕上则最优。评估指标采用客观评估的方式。通过产品的影像整合功能演示，对产品可以整合影像类型的数量以及是否可整合在一个屏幕上进行等级划分。

(6)实时影像监测能力。评估产品实时影像监测的能力，即手术全过程中是否可通过心内超声导管在电生理导航系统上直视心脏结构(实时影像监测能力需由系统软件配合心内超声导管实现)。评估指标采用客观评估的方式。评估产品套装中是否包括心内超声导管，若包括则依据心内超声导管的成像类型和最远探查深度进行等级划分。

(7)产品智能化。评估产品是否有指导消融过程的智能化功能，如消融指数(ablation index)、损伤指数(lesion index)、灌注流自适应模式等。评估指标采用客观评估的方式。判断产品是否有指导消融过程的智能化功能，若包括，则依据公开发表的文献数据，评估产品该智能化功能的指导是否可以显著提高临床有效性，并对文献数据的证据质量进行等级划分。

# 2.1.5医护使用体验

(1)学习曲线。评估新术者通过使用产品可熟练掌握技术的时间[26。评估指标采用主观评估的方式。依据临床实践经验对产品在同类评估产品中的学习时间长短进行主观评估。

(2)手术操作的便利性。评估产品的特点是否可以降低手术操作难度，包括导管操作的便利性与稳定性(如导管的到位能力、力的扭矩及传递能力、控弯回弹性和弯型保持能力等)、鞘管可视化、弯形可调、模型成像清晰度、标测清晰度等。评估指标采用主观评估的方式。依据临床实践经验对使用产品的手术操作便利程度进行主观评估。

(3)对医护的职业保护。评估使用产品是否具有医护职业保护的优势，即无X射线也可完成手术[27-30]。评估指标采用客观评估的方式。①依据产品物理性能判断使用产品进行手术操作是否无需强制应用X射线，如使用心内超声导管；②依据公开发表的文献数据或基于病案系统数据开展真实世界研究，评估与同类产品相比术中射线暴露时间或暴露剂量的相对优劣势，并对证据质量进行等级划分；③真实世界研究应遵守相关规范指南[12,13]进行研究设计和统计分析以满足统计学要求、控制混杂效应。

# 2.1.6先进性

评估产品是否具备相对先进的技术证明(如，是否具备核心技术的发明专利认证、或产品通过创新产品注册程序或优先注册程序上市)，以及产品的技术是否是未来发展的方向[3l。评估指标采用客观评估和主观评估相结合的方式。依据生产企业递交的注册证、注册材料以及专利认证证明判断产品是否具备先进技术证明。同时，根据临床实践经验对产品可发展新的治疗策略或填补既往产品应用缺陷的能力进行等级划分。

# 2.2经济价值评估

# 2.2.1产品经济性

(1)电生理系统均次使用成本。评估医院购买和使用心脏电生理系统均摊到单台手术中的成本[32]。评估指标采用客观评估的方式。成本核算项目主要包括设备购置费单次分摊成本、开机成本(若有)、一次性耗材成本、可多次使用的耗材与部件单次分摊成本、设备定期维保单次分摊成本等。成本核算的步骤为：①根据设备可适用术式的年均台数以及设备折旧年限，估算设备可使用的总手术台数；②根据设备购置费以及设备可使用的总手术台数，计算设备购置费单次分摊成本；③根据设备定期维保的年均费用、设备折旧年限以及设备可使用的总手术台数，计算设备定期维保单次分摊成本；④根据各类可多次使用的耗材和部件的使用次数和成本计算其单次分摊成本；③将上述各项单次分摊成本、一次性耗材的成本以及开机成本(若有)求和。根据上述核算步骤计算所有评估产品的均次使用成本，并根据产品相比同类评估产品的成本节省程度进行等级划分。成本核算的计算[32为公式1：

# 2.2.2对医院运营效率的影响

(1)住院时间。评估使用该产品患者的总住院天数。评估指标采用客观评估的方式。基于病案系统数据开展真实世界研究，评估与同类产品相比住院时长的相对优劣势，并对证据质量1进行等级划分。真实世界研究应遵守相关规范指南[12-13进行研究设计和统计分析以满足统计学要求、控制混杂效应。

(2)手术时间。评估使用该产品的手术操作过程所需要的时间(min)，包括穿刺、解剖建模、标测、消融和验证等[15,27,37-38]。评估指标采用客观评估的方式。依据公开发表的文献数据或基于病案系统数据开展真实世界研究，对同一术式下，对产品相比同类产品手术时间的节省程度进行等级划分，并对证据质量进行评估。真实世界研究应遵守相关规范指南[12-13进行研究设计和统计分析以满足统计学要求、控制混杂效应。

(3)病种费用影响。评估使用不同产品，在特定付费制度下相关病种的次均费用，付费制度包括疾病诊断相关分组(diagnosis related groups，DRGs)、疾病诊断干预分组(diagnosis intervention packet，DIP)又称病种分值付费及单病种[39等，费用包含相关病种所有诊疗活动中由患者和医保为药品、耗材和各类医疗服务项目所支付的费用总额。评估指标采用客观评估的方式。依据病案系统数据或生产企业递交的测算报告，根据产品在相关疾病下是否低于支付标准或高于支付标准的程度来划分等级。

# 2.3 患者价值评估

# 2.3.1生命质量

评估产品对患者长期生命质量的改善程度(术后6个月)，如对患者的生理功能、社会功能以及心理健康产生的影响，可通过生命质量评估量表，如欧洲五维健康量表(EuroQol five-dimensionquestionnaire，EQ-5D)、健康调查简表(the MOSitem short from health survey，SF-36)等或心房颤动患者生活质量量表(atrial fibrillation effect onquality-of-life，AFEQT)等进行评估[40-42]。评估指标采用客观评估的方式。依据公开发表的文献数据，评估使用产品术后6个月以上相比术前是否可显著提高患者生命质量，并对证据质量进行等级划分。

# 2.3.2 患者体验

评估产品的特点能否提高诊疗全流程的患者体验，如降低患者因术前经食道超声心动图(trans-esophageal echocardiography，TEE)导致的疼痛，或术中对腹股沟、心脏等部位的疼痛。评估指标采用主观评估的方式。依据临床实践经验对使用产品提升患者体验的程度进行主观评估。

# 2.4管理价值评估

# 2.4.1产品质量认证

(1)生产企业质量管理认证。评估生产企业是否通过医疗器械质量管理体系认证，包括国际标准化组织(International Organization for Standardization,ISO)13485[43认证及国家标准《医疗器械质量管理体系用于法规的要求》(GB/T42061)[44医疗器械行业标准。评估指标采用客观评估的方式[39]。依据企业所提供的证明文件，对其是否通过医疗器械质量管理体系认证进行等级划分。

(2)产品质量认证。评估产品在中国、美国、欧盟、日本、韩国等主流市场的认证获得情况，辅助评估产品的安全性。评估指标采用客观评估的方式。依据企业所提供的产品上市认证证明文件，对产品所通过的国内外主流市场注册审核的数量进行等级划分[39]。

# 2.4.2设备安全质量

(1)平均无故障时间。评估产品在医院使用期间发生两次相邻故障之间的工作时间的平均值。评估指标采用客观评估的方式。依据医院管理数据或者生产企业提供的其他医院的设备使用数据，统计运行期间所有评估产品的正常工作时间和故障发生次数以计算平均无故障时间，并根据产品相比同类评估产品的时间提高程度进行等级划分。

(2)故障维修成本。评估产品在使用一定时间内(如1年)因发生故障而产生的成本，以此评估故障的严重程度。评估指标采用客观评估的方式。采用与平均无故障时间相同的数据来源计算所有评估产品由于故障而产生的年均维修成本，并根据产品与同类评估产品相比的成本节省程度进行等级划分。

# 2.4.3产品可及性

(1)产品线丰富度。评估生产企业是否可以覆盖1台电生理手术需要的所有产品类别(标测设备、消融设备、标测类导管耗材、消融类导管耗材、心内超声导管耗材等)。评估指标采用客观评估的方式。依据企业所提供的产品线相关材料，对企业所覆盖的产品类别数量进行等级划分。

(2)供货稳定性。评估企业既往对于本院和(或)同区域其他医疗机构的供应协议执行情况[45-48]。评估指标采用客观评估的方式。依据医院管理数据或企业提供的同区域内其他同等级医院的近3年供货记录，对订单响应速度和缺货频率进行等级划分。同时，依据企业所提交的获奖证明材料，判断企业是否获得供应商专项评奖。

(3)医疗机构设备覆盖数量。评估设备在特定区域内医疗机构常规使用的覆盖数量，间接衡量市场对产品以及企业供应能力的认可程度[49-50]。评估指标采用客观评估的方式。依据企业提供的证明文件，对设备近3年在全国公立医疗机构常规使用(不包括入院试用情况)的覆盖数量进行等级划分，医疗机构可根据本院所关注的重点区域(全国、同省市等)以及重点关注的医疗机构的类型(公立、私立、三甲等)对本指标的评估方法进行适当调整。

# 2.4.4技术服务支持

(1)技术人员服务质量。评估企业是否能安排技术人员提供术中技术支持、技术支持人员的专业度、以及本院和(或)同区域其他医疗机构的对既往技术服务支持的满意度。评估指标采用主观评估的方式。依据企业所递交的技术人员的数量、学历、经验等信息以及该企业既往对本院的技术支持情况，对企业技术人员的专业度和技术支持的满意度进行主观评估。

(2)售后服务及时性。评估企业在本院或同区域其他医疗机构对企业既往售后服务的响应时间以及整个售后的完成周期。评估指标采用客观评估的方式。依据医院管理数据或企业提供的同区域内其他同等级医院的售后记录，对售后响应时长和售后完成周期进行等级划分。

(3)售后服务质量。评估企业售后服务人员的专业度、本院或同区域其他医疗机构对于既往售后服务满意度。评估指标采用主观评估的方式。依据企业所递交的售后服务人员的数量、学历、经验等信息以及既往对本院的售后服务情况，对企业售后服务人员的专业度和售后服务的满意度高低进行主观分级判断。

(4)培训质量。评估企业所配备的面向医生团队内部培训师服务年限、培训师人员数量和资质经验、既往培训专业度等[39,51]。依据企业所提交的专职医生培训团队的配备数量、资质、经验相关文件，对培训师资进行客观评估。依据近3年医院及同区域其他医疗机构医护人员对于培训师和培训内容的专业性和实操培训的实施评价进行主观补充评估。

(5)培训持续性。评估企业是否可以连续3年、5年等根据医疗机构的需求提供持续培训[39,51]。评估指标采用客观评估的方式。依据企业所提交的专业教育培训合作记录，对培训的连续性以及培训范围进行等级划分。

# 2.4.5机构影响

(1)管理相融性。评估产品是否满足医院管理相关要求，包括是否省市平台挂网、是否有收费条目、是否纳入医保等[39,51]。评估指标采用客观评估的方式。依据本院相关管理制度以及产品信息，判断产品是否满足医院管理要求。

(2)学科建设与发展。评估产品的使用对相关团队科室建设的影响[39,51]。评估指标采用主观评估的方式。根据临床实践经验，对产品促进复杂手术开展以及学科发展的能力进行主观评估。

(3)现有诊疗程序的影响。评估使用产品是否会改变现有诊疗程序及影响程度[39,51]。评估指标采用主观评估的方式。依据医院相关诊疗流程以及产品信息，对现有诊疗流程对于产品的适用程度进行主观评估。

(4)额外人员及专家的投入或引进。评估使用产品是否需要额外的人员和专家，从而增加人力资源成本[39,51]。评估指标采用主观评估的方式。依据临床实践经验判断本院常规应用产品是否需要在相关术式中增加额外医护人员或者从院外引进专家进行指导操作。

“共识”由多领域专家采用德尔菲法和层次分析法，探索形成一套心脏电生理医疗器械综合价值评估量化体系，从临床价值、经济价值、患者价值及管理价值4个维度进行各个核心模块组件的关键价值点挖掘、指标细化以及评估方法制定，并通过多次专家论证保证指标框架、评估内涵、评估方式、评估标准及指标权重的科学性、合理性和可操作性。“共识”为医疗机构提供了心脏电生理医疗器械进行科学遴选以及“基于价值”
