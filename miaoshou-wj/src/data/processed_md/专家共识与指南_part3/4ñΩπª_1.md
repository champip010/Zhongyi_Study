# 《气滞证诊断标准》

# 推广方案

# 一、项目背景

气滞证是中医临床常见证候之一，多见于许多重大疾病和慢性疾病，并为其始发证和基本证，常伴随疾病始终，但诊断标准缺如，严重影响了证候诊断的规范性、一致性及可重复性。主要有以下几点问题：未建立科学的量化标准，临床诊断效能及实用性有待考证；不同疾病气滞证主次症状、条目权重、诊断阈值尚未确定。为避免个人因素对辨证的干扰，提高临床诊疗和临床科研辨证的准确性和一致性，结合国内外最新证候及相关术语研究进展，特制定本标准，指导临床与科研气滞证辨证的准确性和规范性。

为落实《中医药发展战略规划纲要（2016—2030 年）》提出的“健全中医药法律体系和标准体系”的重要任务，本标准在系统回顾既往气滞证相关诊断标准基础上，综合运用文献系统评价、德尔菲法、层次分析法、专家访谈、诊断性试验等方法，确立了气滞证诊断条目及其权重，修订了气滞证相关诊断标准，为气滞证的临床诊疗和科学研究提供指导。

# 二、推广目标

1、推广培训项目计划在2022-2023 两年内，拟对全国华北、华中、华南 3 个片区 5 个省、自治区、直辖市的精神科、脾胃科、乳腺科、妇科、心血管科及其他呼吸科疾病相关科室的临床医护人员，以及从事抑郁症、慢性非萎缩性胃炎等相关疾病研究的科研院所专业人员200 人次。

2、参加培训的学员能够掌握气滞证诊断标准，并运用于指导临床诊疗工作。

3、开展气滞证的临床流行病学调查。

# 三、推广内容

1、通过中华中医药学会等中医药学会平台发布和公告，使本标准具有权威性。

2、课题组将继续在国内外核心期刊、专业报纸上就标准研制的意义、临床适用性等问题发表1-2 篇相关文章。另将指导在读研究生将本标准的具体内容及应用范例等纳入学位论文中，并授权中国知网、万方等数据库公开发表，使读者对本标准有全面了解。

3、通过中医药相关网站、微信平台发布，提供用户免费下载权限，以方便用户更准确了解、应用本标准。

4、将在中华中医药学会中医诊断分会、中国中西医结合学会中医诊断分会、世界中医药学会联合会中医诊断分会举办相关的学术会议上，可利用制定本标准的首席专家和项目专家的学术影响力报告本标准具体内容、制定过程、应用范例等，使与会人员了解本标准及应用情况。同时将邀请媒体进行播报，以扩大本标准的影响力。

5、未来两年内，拟对全国华北、华中、华南3 个片区5个省、自治区、直辖市的 5 家临床研究分中心的临床医师和科研人员进行培训，累计达 200 人。同时拟对上述5 家临床调查研究单位科研人员就本标准的制定、意义、应用进行1 次专题培训，预计参会人员将超过100人。计划召开 1 次针对西医医院精神科临床医生的专题培训或研讨会，对本标准的中医辨证内容进行重点讲解，以进一步扩大本标准的应用面，预计参会人员将达到50 人。围绕本标准制定过程，拟开展层次分析法方法学培训 1 次，供科研人员及在读研究生参考，预计参会人员 150 人。

6、未来两年内，将在全国多家医院、相关科研院所开展标准适用性评价研究。

7、制作气滞证系列疾病诊断规范的短视频或软件，向患者和社会进行中医诊断规范化科普。

# 四、推广的管理机构

暨南大学、湖南中医药大学、广州中医药大学、中华中医药学会。
