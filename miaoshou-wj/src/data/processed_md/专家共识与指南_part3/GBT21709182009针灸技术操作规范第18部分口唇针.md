# 引 言

一般标准中所涉及的长度、宽度的计量都要求采用国际单位制，但是人体高矮胖瘦的差异很大，无法采用绝对的标准值描述针灸腧穴部位，只有通过等分折量的方法———骨度折量法描述腧穴部位，才能适用于所有人群和所有个体。1987年世界卫生组织在韩国汉城召开的国际会议上确定采用“寸”作为针灸经穴标准计量单位。因此，本标准的腧穴定位采用这种计量单位。

# 针灸技术操作规范 第18部分：口唇针

# 1 范围

GB／T21709的本部分规定了口唇针疗法的术语和定义、操作步骤与要求、注意事项和禁忌。  
本部分适用于口唇针疗法技术操作。

# 2 术语和定义

下列术语和定义适用于 GB／T21709的本部分。

2．1口针疗法 狅狉犪犾犪犮狌狆狌狀犮狋狌狉犲狋犺犲狉犪狆狔针刺口腔黏膜上的特定穴区以治疗疾病的方法。

2．2唇针疗法 犾犻狆犪犮狌狆狌狀犮狋狌狉犲狋犺犲狉犪狆狔针刺唇部穴位以治疗疾病的方法。

# 3 操作步骤与要求

# 3．1 施术前准备

# 3．1．1 针具选择

3．1．1．1 口针

宜选择直径为0．20mm～0．25mm，长度为15mm～50mm的毫针。

# 3．1．1．2 唇针

宜选择直径为0．20mm～0．30mm，长度为25mm～40mm的毫针。

# 3．1．2 穴位选择

根据疾病选取穴位或特定分区。附录 A、附录B分别给出了口针和唇针腧穴的命名与定位、示意图。

# 3．1．3 体位选择

口针操作可选择坐位或卧位，唇针操作宜选择卧位。

# 3．1．4 环境要求

应注意环境清洁卫生，避免污染。

3．1．5 消毒

3．1．5．1 针具消毒应选择高压蒸汽消毒灭菌法。宜使用一次性针具。

3．1．5．2 部位消毒

3．1．5．2．1 口针应用含有1％碘酊或0．5％碘伏的棉签或棉球对施术部位进行消毒。

3．1．5．2．2 唇针

应用含有75％乙醇或0．5％的碘伏棉签或棉球对施术部位进行消毒。

# 3．1．5．3 医者消毒

医者双手应先用肥皂水清洗，再用75％乙醇或0．5％碘伏棉球擦拭。

# 3．2 施术方法

# 3．2．1 口针疗法

3．2．1．1 进针

令患者张口，医者戴上无菌手套一手持无菌纱布捏住患者上唇或下唇，或用消毒棉签暴露施术部位，另一手持针柄将针刺入口腔黏膜穴位或特定分区。根据针刺的部位，选择合适的进针角度和深度。针刺手法，以病人耐受为度，可无针感。

# 3．2．1．2 留针

参见附录 C的 C．1．1。

3．2．1．3 出针

医者施术完毕，持针柄将针退出。

# 3．2．1．4 出针后处理

施术处若出血，宜用消毒干棉签或无菌纱球按压针孔至血止，然后嘱患者用0．9％生理盐水漱口。

# 3．2．2 唇针疗法

3．2．2．1 进针

医者一手将施术部位两侧皮肤提捏固定，另一手持针柄将针刺入。上唇部穴位沿人中沟向鼻中隔方向斜刺入6mm～15mm，以雀啄手法为宜；下唇部穴位平刺入10mm～30mm，可不施手法。

# 3．2．2．2 留针

参见附录 C的 C．1．2。

3．2．2．3 出针

医者一手固定施术部位两侧皮肤，另一手持针柄将针退出。

# 3．2．2．4 出针后处理

![](images/e7d3800e26ac81d5d44831c393990e73c68c13d3efda67384d03cb863af77109.jpg)

宜用消毒干棉签按压针孔。

# 4 疗程及适应证

4．1 口唇针的疗程参见第 C．2章。  
4．2 口唇针的适应证参见第 C．3章。

# 5 注意事项

5．1 初次接受治疗的患者，应首先消除其紧张情绪。  
5．2 如患者出现晕针，按照晕针常规处理法予以处理。  
5．3 不配合治疗的小儿，精神病患者，中、重度糖尿病患者，体质过度虚弱者和孕妇应慎用。

# 6 禁忌

6．1 口腔破溃、化脓、出血或肿瘤处。  
6．2 出血性疾病患者。  
6．3 严重高血压、心脏病患者。  
6．4 金属过敏者。

# （资料性附录）适应证、留针时间和疗程

# 犆．1 留针时间

# 犆．1．1 口针留针时间

可留针，留针一般不宜超过30min。

犆．1．2 唇针留针时间 以20min～30min为宜。

# 犆．2 疗程

宜隔日1次，5次为1疗程，疗程间宜间隔2d～4d。

# 犆．3 适应证

# 犆．3．1 口针适应证

疼痛性疾患，面神经麻痹、小儿麻痹后遗症、中风后遗症、急性腰扭伤、癫痫、痹症、急性结膜炎、口疮、产后缺乳、小儿惊痫、遗尿、呕吐、咳喘、牙龈肿痛、咽喉肿痛、鼻塞等。

# 犆．3．2 唇针适应证

疼痛性疾病、齿龈肿痛、口角流涎、面神经麻痹、脑血管意外、癫痫、精神分裂症等。
