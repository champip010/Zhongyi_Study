# 中医养生保健服务（非医疗）技术操作规范 小儿推拿

# 1 范围

本文件规定了小儿推拿的术语和定义、操作步骤与要求、操作方法、禁忌症、操作过程中可能出现的意外情况及处理措施。

本文件适用于中医养生康复服务（非医疗）从事小儿推拿养生保健的操作者。

本文件适用于在市场监督部门登记的经营范围为中医养生保健服务（非医疗）、提供中医养生服务的非医疗机构。

本文件适用于14周岁以下的小儿体质健康调理。

# 3 术语和定义

下列术语与定义适用于本文件。

# 3.1 术语

小儿推拿 Massage for children

# 3.2 定义

小儿推拿是在中医儿科学和中医推拿学的基础上形成和发展起来的，具有独特的理论体系。通过对小儿体表的特定穴位或部位，根据体质的不同采取不同的操作手法的一种养生保健方法。

# 4 操作步骤与要求

# 4.1 操作前准备

# 4.1.1 环境

环境清洁卫生，定期消毒，室内安静，避风、避强光，温度适宜，保持空气流通。

# 4.1.2 器具

普通诊疗床或儿童用诊疗床、清洁和消毒用品、推拿用介质。

# 4.1.3 操作者

操作者须热爱本职工作，认真负责，无传染病、精神病等身体健康状态下，能够胜任中医养生保健

服务工作的人员。具有中医药类相关学习背景，或者接受过中医养生保健专业培训并具备相关知识和技能，考试合格并取得小儿推拿资格证书。掌握从事中医养生保健服务相关技术操作规范和流程、技术风险防控、基本急救知识等，遵守卫生健康和中医药相关法律法规规章，遵守职业道德。

操作前，操作者应取下影响推拿的手和腕部饰物，修剪指甲并锉平。操作时双手温暖，精神专注，态度和蔼，争取患儿的配合。

# 4.1.4 操作部位选择

实施小儿推拿前，首先要使小儿全身放松，检查操作部位无骨折、无关节损伤或脱位，无皮肤破损、出血及皮肤发炎感染等影响操作的情况。根据不同体质选取适当的穴位和手法。上肢部穴位，可根据操作者习惯操作儿童左手或右手一侧的穴位；非上肢部位的双侧穴位，两侧均可选择操作。小儿推拿常用穴位定位及操作手法参见附录 B，穴位定位应符合 GB/T 12346的规定。

# 4.1.5 体位选择

# 4.1.5.1 小儿体位选择

应根据小儿年龄选择舒适、放松、能暴露操作部位、能配合推拿操作的体位，如坐位、仰卧位、侧卧位、俯卧位、抱坐位等。

# 4.1.5.2 操作者体位选择

坐位和站立位，以前者更为常用。

# 4.1.6 介质选择

常用的推拿介质可以分为：

# 4.1.6.1 粉类

由具有润滑特点且对人体安全的物质研磨成细粉所形成的介质。常用有:

# 4.1.6.1.1 食用淀粉

可润滑和保护皮肤，减少汗液对推拿操作的影响。四季均可使用，使用时应注意避免动作过大产生扬尘而被儿童吸入。

# 4.1.6.1.2 爽身粉

具有润滑皮肤和吸水性强的特点，质量较好的爽身粉可替代医用滑石粉，四季均可使用， 使用时应注意避免动作过大产生扬尘而被儿童吸入。

# 4.1.6.2 水类

由清洁水或其浸泡药物而成的介质。常用有:

# 4.1.6.2.1 水

清洁常温水，冬天可适当加热。具有清凉散热作用，适用于热性病证。

# 4.1.6.2.2 生姜水

取鲜生姜适量切碎、捣烂，用适量开水浸泡30分钟后，去渣取液应用。具有疏风散寒的作用，适用

于风寒病证。

# 4.1.6.2.3 葱白水

取葱白适量切碎、捣烂，用适量开水浸泡30分钟后，去渣取液应用。具有疏风散寒的作用，适用于风寒病证。

# 4.1.6.2.4 薄荷水

取鲜薄荷叶或干薄荷叶，置于容器中加盖用适量开水浸泡30分钟后，去渣取液应用。具有疏风散热的作用，适用于风热病证。

# 4.1.6.3 油类

由成份安全的动植物油脂类物质等为主构成的介质。常用有:

# 4.1.6.3.1 麻油

食用芝麻油。具有润滑除燥作用，适用于身体各部位推拿，也可用于刮法。

# 4.1.6.3.2 椰子油

食用椰子油。具有润滑除燥作用，适用于身体各部位推拿，也可用于刮法。

# 4.1.6.3.3 山茶油

食用山茶油。具有润滑除燥作用，适用于身体各部位推拿，也可用于刮法。

# 4.1.6.4 膏类

用动植物油、蜂蜜或羊毛脂、凡士林等为基质，加药物细粉或提取的有效成分混合而成， 具有相应药效的介质。常用有:

# 4.1.6.4.1 冬青膏

由冬青油、凡士林等混合配制而成。具有温经散寒的作用，适用于寒凝经络类病症。

# 4.1.6.4.2 健脾膏

由具有健脾作用的中药提取物与油脂混合调制而成。具有健脾开胃的作用，适用于脾胃功能虚弱的病症。

# 4.2 操作手法

# 4.2.1 小儿推拿常用手法

# 4.2.1.1 推法

# 4.2.1.1.1 直推法

用拇指螺纹面或桡侧缘（或用食、中两指螺纹面）,在选定的穴位上做单向直线推动。

# 4.2.1.1.2 分推法

用双手拇指桡侧缘或螺纹，或用双手食、中指螺纹面自穴位中间向两旁做分向推动。又称分法。

# 4.2.1.1.3 合推法

用双手拇指桡侧缘或螺纹，或用双手食、中指螺纹面自穴位两旁向中间推动。又称合法。

# 4.2.1.1.4 旋推法

用拇指螺纹面在穴位上做顺时针或逆时针方向旋转推摩。

# 4.2.1.2 揉法

# 4.2.1.2.1 指揉法

用拇指或食指，或用食、中、无名指螺纹面，吸定于一定部位或穴位上,做轻柔回旋揉动。

# 4.2.1.2.2 掌揉法

用掌根部分,吸定于一定部位或穴位上,做轻柔回旋揉动。

# 4.2.1.2.3 鱼际揉法

用手掌大鱼际吸定于一定部位或穴位上,做轻柔回旋揉动。

# 4.2.1.3 按法

用拇指或中指指端或掌心在一定的穴位向下逐渐用力按压。操作时手法要求轻柔、速度均匀协调、压力由轻到重。

# 4.2.1.4 摩法

用手掌掌面或食、中、无名指指面吸附于一定部位上，以腕部连同前臂做环形的有节律的抚摩。动作宜轻柔而有节奏。一般以按摩的速度和方向来区别补泻，如快摩为泻、慢摩为补；顺时针摩为泻、逆时针摩为补。此法在小儿多用于腹部。

# 4.2.1.5 运法

用拇指指面或桡侧面，也可用食指或中指指面，在操作部位上做弧形或环形的轻轻推摩。操作宜轻不宜重,宜缓不宜急。力度仅达表面皮肤，只在皮肤表面运动，不带动皮下组织。

# 4.2.1.6 掐法

手握空拳，拇指伸直紧贴于食指，用拇指指甲垂直用力按压穴位或部位，以指代针，不得抠动而掐破皮肤。适用于点状穴位，主要用于急救,如掐人中。本法也可用于疾病的防治，一般轻掐后加揉，形成掐揉复合手法，如掐揉五指节。操作时要逐渐用力，以达到深透为止，不宜反复长时间应用，注意在重掐时不得掐破皮肤。

# 4.2.1.7 搓法

在夹持的基础上来回搓动为搓法。操作时，两手掌夹住所取的肢体或部位，相对用力搓摩，或同时做上下往返的移动。两手用力相等、速度均匀，搓动快、移动慢。切忌用生硬粗暴蛮力，以免搓伤皮肤与筋脉。主要适用于四肢和胁肋部。

# 4.2.1.8 摇法

被动环转运动肢体的方法称摇法。操作时，一手握持住肢体或关节的近端，一手握持住关节的远端，

顺着关节运动的方向，做一定幅度的摇动，如摇肘关节。操作者两手要协调配合，注意用力要平稳，宜缓不宜急，幅度应由小到大；不得超出关节生理活动的范围，更不能反关节摇动。主要适用于关节部位，一般根据病症情况决定摇动的次数。

# 4.2.1.9 捏法

以拇指与食、中两指相对用力捏拿皮肤。捏法主要用于脊柱部，故称为捏脊。

捏脊疗法以拇指桡侧缘顶住皮肤，食、中两指前按，三指同时用力提拿肌肤，双手交替向前捻动，从尾椎骨端捏至大椎穴，本法又称翻皮肤；或用食指屈曲，用食指中节桡侧缘顶住皮肤，拇指前按，两指同力提拿肌肤，双手交替向前捻动，从尾椎骨端捏至大椎穴。

# 4.2.1.10 拿法

捏而提起谓之拿。用拇指指端和食、中两指指端，或用拇指与其他四指相对，提拿一定部位和穴位，做一松一紧的拿捏。

# 4.2.1.11 拍法

以半握虚掌拍打体表。注意用力由轻到重，轻重适度。也常用作推拿治疗后的结束手法。

# 4.2.1.12 擦法

用手掌面、大鱼际或小鱼际着力于选定部位上进行直线来回摩擦。

# 4.2.1.13 捣法

用中指指端，或屈曲的食指、中指指间关节着力，做有节奏的叩击动作。

# 4.2.2 小儿推拿复式手法

# 4.2.2.1 飞经走气

用右手拿住小儿左手四指，再用左手四指从肘关节曲池按之、跳之至手腕部总筋，反复几遍后，左手拇、食二指拿住手腕部内外侧不动，然后右手屈伸搓揉小儿四指。

# 4.2.2.2 摇抖肘法

先以左手拇、食、中三指托小儿之肘，再以右手拇、食两指叉入小儿虎口，同时用中指按定天门穴，然后屈小儿之手上下摇之。

# 4.2.2.3 水底捞月

一手握持手掌，另一手拇指自小指根起，沿小鱼际推至小天心，转入内劳宫处，做捕捞状，后一拂而起。亦可将凉水滴入小儿掌心，以拇指或中指端旋推，边推边吹凉气。又称水中捞水、水里捞月、水中捞明月。

# 4.2.2.4 打马过天河

先运内劳宫，然后再用左手拿小儿两指，用右手食、中、无名指沿天河打至腕横纹止；或用食、中指沿内关、间使、天河水一起一落打至洪池穴止，边拍打边随之吹气。

# 4.2.2.5 黄蜂入洞

用食、中两指指端紧贴在小儿两鼻孔下缘处，以腕关节为主动带动着力部分做反复揉动或振颤。

# 4.2.2.6 运水入土

用大拇指外侧缘自小指指腹部的肾经穴起，沿手掌的尺侧和掌根部，运至大拇指指腹的脾经穴。

# 4.2.2.7 运土入水

用大拇指外侧缘自大拇指指腹的脾经穴沿小儿手掌的掌根部和尺侧运向小指指腹部的肾经穴。

# 4.2.2.8 按弦搓摩

在小儿身后，用双掌在小儿两腋下胁肋处，自后上而前下搓摩至肚角处。

# 4.2.2.9 二龙戏珠

以左手拿小儿之手，使掌心向上，前臂伸直，右手食、中两指面自小儿总筋处，以指头交互向前按之，直至洪池为一遍，5遍左右。最后一手拿捏阴池、阳池5～6次，另一手持小儿左手食指、无名指指端摇动20～30次。

# 4.2.2.10 双凤展翅

用双手食、中两指夹小儿两耳向上提几次后，再按掐眉心、太阳、听会、牙关、人中、承浆等穴。

# 4.2.3 小儿推拿取穴方法

# 4.2.3.1 自然标志取穴

可根据固定标志取穴，如外劳宫位于手背二、三掌骨交接处的凹陷中；四横纹位于掌面食、中、无名、小指第一指间关节横纹处。或根据活动标志取穴，如委中：屈膝，膝后腘横纹之中间，两筋凹陷处。

# 4.2.3.2 手指同身寸取穴

穴位的位置，将食指、中指、无名指和小指并拢，以中指中节横纹处为准，四指横量作为3寸。

# 4.2.3.3 简便取穴

例如：内劳宫位于掌心,握拳时中指落着点即是该穴。

# 4.3 小儿推拿操作顺序

一般有三种方式，可根据实际情况选择应用:

4.3.1 先操作手部穴位，再依次操作头面、胸腹，下肢、腰背部穴位。   
4.3.2 先操作主穴，后操作配穴；先采用轻手法，后用重手法操作。   
4.3.3 根据症状轻重缓急，决定推拿的操作顺序。

# 4.4 小儿推拿操作时间、频率、力度

# 4.4.1 小儿推拿操作时间

小儿推拿操作时间可根据不同年龄、不同体质和手法特性等情况而定。一般每穴操作时间为1\~2分钟；总的治疗时间为20～30分钟；对于年龄小、体质弱、采用刺激性较强的手法者可适当缩短操作时间。若以 1 岁为标准，每穴推2分钟左右(强刺激手法除外)，每个主穴推300次左右。小于1岁或体质较弱者，

推拿的时间可适当缩短，次数可适当减少；大于1岁或体质强壮者，时间适当延长，推次适当增加。

# 4.4.2 小儿推拿操作频率

小儿推拿操作频率视不同症状而定。一般以7～10天为1周期，2个周期之间可间隔2～3天。特殊情况应根据体质调整推拿操作频率。

# 4.4.3 小儿推拿操作力度

小儿推拿操作时要用力均匀、动作轻柔、深透平稳；年龄小、体质弱者，手法宜轻；年龄较大、体质强者，手法可适当加重，以小儿能耐受为宜。

# 5 操作方法

操作方法参见附录A。

# 6 禁忌症

6.1 各种皮肤病，以及皮肤有破损(发生烧伤、烫伤、擦伤、裂伤等)、不明肿块以及有瘢痕、正在出血和内出血的部位等；  
6.2 有明显的感染性症状，如骨与关节炎症、化脓、骨髓炎、蜂窝组织炎、丹毒等，以及肿瘤、外伤骨折、脱位等；  
6.3 严重的心、肺、肝、肾等脏器病症者;  
6.4 某些传染病，如水痘、猩红热、肝炎、肺结核等;  
6.5 出血性症状，如免疫性血小板减少症、血友病、过敏性紫癜等。

# 7 操作过程中可能出现的意外情况及处理措施

# 7.1 意外情况

实施小儿推拿过程中可能出现皮肤破损、肠损伤、哭闹等或操作后症状不缓解、甚至加重等意外情况。

# 7.2 处理措施

# 7.2.1 皮肤破损

若用力不当导致小儿皮肤破损，较轻者可保持局部清洁并避免在破损处操作，较重者应做局部消毒处理，预防感染。

# 7.2.2 肠损伤

腹部推拿操作用力不当可能导致小儿之肠痉挛，严重时可能导致肠套叠或肠梗阻，应立即到相关科室进行检查及处置。

# 7.2.3 其他

在推拿过程中小儿出现烦躁、恐惧、哭闹、极度不配合的情况下，应暂停推拿，耐心与小儿沟通和

交流，取得小儿信任后即可继续实施推拿；操作后症状不缓解甚至加重应去相关医疗机构进行相关检查；凡小儿出现抽搐、昏迷等危重症状，均应及时就医抢救，暂不作推拿。

双手掌快速摩擦感到发热发烫，将双手掌心内劳宫轻覆于小儿双眼上，待热感不明显时，再重复上述操作，如此反复 6 次。

# A.6.7 摩腹

以手掌面或食、中、无名、小指螺纹面着力，顺时针方向摩小儿腹部 3～5 分钟。

# A.6.8 拿揉颈项部

以拇指与食、中二指对称用力，拿揉小儿颈项部 1 分钟。

# A.6.9 捏脊

以拇指桡侧缘顶住小儿背部皮肤，食、中两指前按，三指同时用力提拿肌肤，双手交替向前捻动，从尾椎骨端捏至大椎穴 3～5 遍，本法又称翻皮肤；或用食指屈曲，用食指中节桡侧缘顶住小儿背部皮肤，拇指前按，两指同力提拿肌肤，双手交替向前捻动，从尾椎骨端捏至大椎穴为一遍，共 3～5 遍。

# A.6.10 操作时间

20\~30 分钟。

# A.6.11 适应症

小儿近视、用眼过度、视疲劳、远视储备不足等。

# A.7 健脾益肾，促进生长

# A.7.1 补脾经

以一手拇、食二指捏住小儿拇指，使之微屈，再以另一手拇指螺纹面或拇指绕侧沿小儿拇指桡侧缘，自指尖推向指根；也可以拇指螺纹面，旋推小儿拇指螺纹面 100～300 次。

# A.7.2 补肾经

以拇指螺纹面或拇指桡侧面着力，沿小儿小指掌面自指尖向指根方向直推，也可旋推小指末节螺纹面旋推 100～300 次。

# A.7.3 捻五指

以拇指与食指螺纹面相对用力，依次捏住小儿治疗部位（拇指、食指、中指、无名指、小指）从指根到指尖，做相对用力的捻动（紧捻慢移）3\~5 遍。

# A.7.4 按揉增高穴

以拇指螺纹面着力，按揉小儿手掌第四、五掌骨之间，握拳时小指尖所触之处下 0.5寸和下 1.5寸（共 2 穴）100\~300 次。

# A.7.5 掐揉五指节

依次用拇指甲掐小儿手背五指第一指间关节横纹处 3～5 次、以拇指端揉此处 30\~50次。

# A.7.6 摩腹

以手掌面或食、中、无名、小指螺纹面着力，顺时针方向摩小儿腹部 3\~5分钟。

# A.7.7 按揉足三里

以双手拇指螺纹面着力，按揉小儿下肢外膝眼下 3 寸，胫骨外侧约一横指处 100～300次。

# A.7.8 按揉涌泉

以拇指螺纹面着力，推揉小儿足掌前 1／3 与后 2／3 交界中间凹陷处 100～300 次。

# A.7.9 捏脊

以拇指桡侧缘顶住小儿背部皮肤，食、中两指前按，三指同时用力提拿肌肤，双手交替向前捻动，从尾椎骨端捏至大椎穴 3～5 遍，本法又称翻皮肤；或用食指屈曲，用食指中节桡侧缘顶住小儿背部皮肤，拇指前按，两指同力提拿肌肤，双手交替向前捻动，从尾椎骨端捏至大椎穴为一遍，共 3～5 遍。

# A.7.10 操作时间

20\~30 分钟。

# A.7.11 适应症

小儿五迟、五软，身高、体重生长发育明显迟于正常同年龄、同性别儿童等。

# A.8 镇静安神，健脑益智

# A.8.1 按揉百会

一手扶儿头部，另一手拇指端着力，按揉小儿头顶正中线与两耳尖连线之交点处 30～50 次。

# A.8.2 按揉四神聪

一手扶儿头部，另一手拇指端着力，按揉小儿头部百会穴前后左右各旁开 1寸处（共 4 穴）各 30\~50次。

# A.8.3 四大手法

A.8.3.1 开天门：以双手拇指螺纹面着力，自小儿眉心直推至前发际 30～50 次；

A.8.3.2 推坎宫：以拇指螺纹面着力，自小儿眉心向眉梢分推 30～50次；

A.8.3.3 运太阳：以拇指或中指端着力，揉小儿两侧眉梢后凹陷处 30～50次；

A.8.3.4 揉耳后高骨：以两中指螺纹面着力，按揉小儿耳后入发际高骨下凹陷处 30～50次。

# A.8.4 清补脾经

将小儿拇指伸直，自指根至指尖来回推 100～300次。

# A.8.5 清肝经

以拇指桡侧面着力，自小儿食指掌面指根直推至指尖 100～300次。

# A.8.6 捣揉小天心

以中指端着力，捣小儿手掌面大小鱼际交界凹陷处 30 次，然后再揉此处 100～300 次。

# A.8.7 按揉内劳宫

以拇指或中指螺纹面着力，按揉小儿掌心中第二、三掌骨之间，即握拳屈指时，中指尖端处 100～300 次。

# A.8.8 掐揉五指节

依次用拇指甲掐小儿手背五指第一指间关节横纹处 3～5 次、以拇指端揉此处 30\~50 次。

# A.8.9 摩腹

以手掌面或食、中、无名、小指螺纹面着力，顺时针方向摩小儿腹部 3\~5分钟。

# A.8.10 按弦走搓摩

在小儿身后，用双掌在小儿两腋下胁肋处，自后上而前下搓摩至肚角处 50\~100次。

# A.8.11 按揉心俞、肝俞、肾俞

以拇指螺纹面着力，依次按揉小儿背部第五胸椎、第九胸椎和第二腰椎棘突下旁开 1.5 寸处，各100～300 次。

# A.8.12 操作时间

20\~30 分钟。

# A.8.13 适应症

小儿注意缺陷多动症、烦躁、易怒等。

# 附录 B（资料性）小儿推拿常用穴位

# B.1 头面部穴位

# B.1.1 太阳

定位：眉梢与眼外角（目外眦）中间，向后约 1寸凹陷处。  
操作手法：直推法、揉法。  
操作次数：每分钟揉 100～300次。

# B.1.2 人中

定位：人中沟上 1/3 与中 1/3交界处。  
操作手法：掐法。  
操作次数：掐 5～10次。

# B.1.3 天柱骨

定位：颈后发际至大椎穴沿颈椎脊突成一直线。  
操作手法：直推法、揉法、擦法。  
操作次数：每分钟揉 100～300 次，直推 30～50次，擦 30～50次。

# B.1.4 天门

定位：自眉心至前发际成一直线。  
操作手法：直推法。  
操作次数：每分钟推 30～50 次。

# B.1.5 坎宫

定位：两眉上，自眉头至眉梢成一线。  
操作手法：分推法。  
操作次数：每分钟分推 30～50次。

# B.1.6 天庭

定位：头部正中线，入前发际 0.5寸处。  
操作手法：掐法、揉法。  
操作次数：每分钟揉 100～300次，掐 3～5次。

# B.1.7 山根

定位：两目内眦中间，鼻梁上低洼处。  
操作手法：掐法。  
操作次数：掐 3～5 次。

# B.1.8 迎香

定位：鼻翼旁 0.5寸，鼻唇沟凹陷中。  
操作手法：揉法、推法。  
操作次数：每分钟揉 100～300次，推 30～50 次。

# B.1.9 耳门

定位：耳区，耳屏上切迹与下颌骨髁突之间的凹陷中。  
操作手法：按法、揉法。  
操作次数：每分钟揉 100～300次，按 3～5次。

# B.1.10 百会

定位：头顶正中线与两耳尖连线之交点处。  
操作手法：按法、揉法。  
操作次数：每分钟揉 100～300次，按 3～5次。

# B.1.11 高骨

定位：耳后入发际，乳突后缘下陷中。  
操作手法：掐法、揉法。  
操作次数：每分钟揉 100～300次，掐 3～5次。

# B.2 手部及上肢穴位

# B.2.1 脾土（经）

定位：拇指螺纹面或拇指桡侧自指尖至指根呈一直线。  
操作手法：旋推法、直推法。  
操作次数：每分钟推 100～300次。

# B.2.2 胃经

定位：拇指掌面近掌端第一节。  
操作手法：直推法。  
操作次数：每分钟推 100～300次。

# B.2.3 大肠

定位：食指桡侧缘，自食指端桡侧边缘至虎口成一直线。  
操作手法：直推法。  
操作次数：每分钟推 100～300次。

# B.2.4 小肠

定位：自小指端尺侧边缘至指尖成一直线。  
操作手法：直推法。  
操作次数：每分钟推 100～300次。

# B.2.5 板门

定位：大鱼际隆起处。  
操作手法：揉法、推法（横纹推向扳门或扳门推向横纹）。  
操作次数：每分钟揉 100～300次，推 30～50 次。

# B.2.6 内劳宫

定位：手掌心，位于第二、三掌骨之间。简便取穴：小儿握拳屈指时，中指尖端处。  
操作手法：揉法、运法。  
操作次数：每分钟揉 100～300次，运 30～50 次。

# B.2.7 一窝风

定位：屈腕，手背掌根中凹陷处。  
操作手法：揉法、掐法。  
操作次数：每分钟揉 100～300次，掐 3～5次。

# B.2.8 少商

定位：拇指末节桡侧，指甲根角侧上方 0.1寸。  
操作手法：掐法。  
操作次数：掐 3～5 次。

# B.2.9 肝经

定位：食指末端螺纹面或食指掌面，由指尖至指根呈一直线。  
操作手法：直推法。  
操作次数：每分钟推 100～300次。

# B.2.10 心经

定位：中指末端螺纹面或中指掌面，由指尖至指根呈一直线。  
操作手法：掐法、直推法。  
操作次数：每分钟推 100～300次，掐 3～5次。

# B.2.11 肺经

定位：无名指末端螺纹面或无名指掌面，由指尖至指根呈一直线。  
操作手法：直推法、旋推法。  
操作次数：每分钟推 100～300次。

# B.2.12 肾经

定位：小指末端螺纹面或小指掌面，由指尖至指根呈一直线。  
操作手法：直推法、旋推法。  
操作次数：每分钟推 100～300次。

# B.2.13 四横纹

定位：掌面食、中、无名、小指关节第一指间关节横纹处。  
操作手法：掐法、直推法。  
操作次数：每分钟推 100～300次，掐 3～5次。

# B.2.14 内八卦

定位：以掌心为圆心，从圆心至中指根横纹中点连线的 2/3为半径作圆。  
操作手法：运法。  
操作次数：每分钟运 50～100次。

# B.2.15 曲池

定位：屈肘，在肘窝桡侧横纹头至肱骨外上髁中点处。  
操作手法：拿法、按法、揉法。  
操作次数：每分钟揉 100～300次，按、拿 3～5次。

# B.2.16 老龙

定位：距中指指甲根正中约 0.1寸处。  
操作手法：掐法。  
操作次数：掐 3～5 次。

# B.2.17 二扇门

定位：掌背无名指与中指、中指与示指的指根夹缝间。  
操作手法：掐法、揉法。  
操作次数：每分钟揉 100～300次，掐 3～5次。

# B.2.18 二人上马

定位：手背无名指与小指掌指关节后凹陷处。  
操作手法：揉法。  
操作次数：每分钟揉 100～300次。

# B.2.19 外劳宫

定位：手背二、三掌骨交接处凹陷中。  
操作手法：揉法。  
操作次数：每分钟揉 100～300次。

# B.2.20 三关

定位：前臂桡侧边缘，自腕横纹直上至肘横纹成一直线。  
操作手法：直推法。  
操作次数：每分钟推 100～300次。

# B.2.21 六腑

定位：前臂尺侧边缘，自腕横纹直上至肘横纹成一直线。  
操作手法：直推法。  
操作次数：每分钟推 100～300次。

# B.2.22 天河水

定位：前臂掌侧正中，自腕横纹中点至肘横纹中点成一直线。  
操作手法：直推法。  
操作次数：每分钟推 100～300次。

# B.3 躯干部穴位

# B.3.1 天突

定位：胸骨切迹上缘凹陷正中。  
操作手法：按法、揉法。  
操作次数：每分钟揉 100～300次，按 3～5次。

# B.3.2 膻中

定位：胸骨上，两乳头连线中点。  
操作手法：分推法、按法、揉法。  
操作次数：每分钟揉 100～300次，按 3～5次，分推 50～100次。

# B.3.3 中脘

定位：脐上 4 寸。  
操作手法：按法、揉法、分推法。  
操作次数：每分钟揉 100～300 次，按 3～5 次，分推 50～100次。

# B.3.4 脐中

定位：肚脐中。  
操作手法：揉法、摩法、直推法。  
操作次数：每分钟揉 100～300次，摩 3～5分钟，推 50～100次。

# B.3.5 腹

定位：腹部。  
操作手法：摩法、分推法。  
操作次数：摩 3～5 分钟，分推 50～100次。

# B.3.6 天枢

定位：与脐平，脐旁 2 寸。

操作手法：按法、揉法。  
操作次数：每分钟揉 100～300次，按 3～5次。

# B.3.7 肚角

定位：脐下 2 寸，旁开 2 寸至两旁大筋（相当于石门穴）。  
操作手法：按法、拿法、揉法。  
操作次数：每分钟揉 100～300次，按、拿 3～5次。

# B.3.8 丹田

定位：腹部脐下，脐中线直下 2 至 3 寸处。  
操作手法：揉法、摩法。  
操作次数：每分钟揉 100～300次，摩 3～5分钟。

# B.3.9 大椎

定位：脊柱区，第七颈椎棘突下。  
操作手法：按法、揉法、直推法。  
操作次数：每分钟揉 50～100 次，按 3～5 次，直推 50～100 次。

# B.3.10 肺俞

定位：第三胸椎棘突下旁开 1.5寸。  
操作手法：按法、揉法、分推法。  
操作次数：每分钟揉 100～300次，按 3～5次，分推 50～100次。

# B.3.11 脊柱

定位：大椎至长强成一直线。  
操作手法：直推法、捏法。  
操作次数：捏 3～5 次，直推 50～100次。

# B.3.12 七节骨

定位：第四腰椎至尾骶骨端（长强穴）成一直线。  
操作手法：推法。  
操作次数：每分钟 50～100次。

# B.3.13 龟尾

定位：尾椎骨端（相当于长强穴）。  
操作手法：揉法。  
操作次数：每分钟揉 100～300次。

# B.4 下肢及足部穴位

# B.4.1 百虫

定位：膝上内侧肌肉丰厚处。  
操作手法：按法、揉法。  
操作次数：每分钟揉 100～300次，按 3～5次。

# B.4.2 足三里

定位：外膝眼下 3 寸，胫骨外侧约一横指处。  
操作手法：按法、揉法。  
操作次数：每分钟揉 100～300次，按 3～5次。

# B.4.3 三阴交

定位：内踝直上 3 寸，胫骨内侧缘凹陷中。  
操作手法：按法、揉法，向上或向下直推法。  
操作次数：每分钟揉 100～300 次，按 3～5 次，直推 50～100次。

# B.4.4 委中

定位：屈膝、膝后腘横纹之中间，两筋凹陷处。  
操作手法：拿法。  
操作次数：拿 3～5 次。

# B.4.5 涌泉

定位：足底部，足掌前 1／3 与后 2／3交界中间凹陷处。  
操作手法：掐法、揉法，或向足大趾方向直推法。  
操作次数：每分钟揉 100～300 次，掐 3～5次，推 50～100次。
