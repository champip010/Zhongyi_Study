# 引 言

一般标准中所涉及的长度 宽度的计量都要求采用国际单位制 但是人体高矮胖瘦的差异很大 无法采用绝对的标准值描述针灸腧穴部位,只有通过等分折量的方法———骨度折量法描述腧穴部位,才能适用于所有人群和所有个体。 年世界卫生组织在韩国汉城召开的国际会议上,确定采用“寸”作为针灸经穴标准计量单位。因此,本标准的腧穴定位采用这种计量单位。

3.2

天部、人部、地部 tian-ren-dineedlingdepths

腹针将在腹壁的进针深度分为天、人、地三个层次,天部是指刺至腹部真皮层和皮下;人部是指刺至腹部浅筋膜和脂肪层,位于天部与地部之间;地部是指刺至腹部肌层。

3.3

腹纵线 abdominalvaerticalline人体仰卧时,从中庭穴至曲骨穴的水平纵线。

3.4

腹横线 abdominalhorizontalline人体仰卧时,由神阙穴通过两个天枢穴延伸的水平横线。

3.5

# 定位穴点 fixedpoint(s)

腹针中具有标志作用的穴点,由部分传统经穴与腹针特定穴点组成,有规律地分布于以神阙穴为中心的上下左右,定位穴点同时也具有治疗作用。

注:定位穴点的名称、代码、部位与取穴示意图参见附录 。

3.6

# 对应穴点 correspondingpoint(s)

在两个相邻的定位穴点之间取穴 依据病变部位及病症不同 选取相应的对应穴点 也是腹部针的主要治疗点。

在治疗疼痛性疾病时,常使用以下刺法,其中主穴是指病灶在腹部的对应穴点。

# B1 三角刺

以主穴为顶点,或上下,或左右各距 寸 寸,加刺两针,使三针形成等腰或等边三角形的多针刺法。此法适宜于肩关节疼痛、膝关节疼痛等疾病。

# B.2 三星刺

以主穴为中点,向两侧各距 寸 寸,加刺两针,使三针形成直线排列的多针刺法。此法适宜于坐骨神经痛等疾病。

# B3 梅花刺

以主穴为中点,上下左右各距 寸 寸加刺一针,共 针,形成梅花图案的多针刺法。此法适宜于腰痛等疾病。

# 附 录 C

(资料性附录) 腹针处方原则

腹针处方是按照中医“君臣佐使”配方原则并结合针刺深度进行组方,具有一病一方的特点。君穴、臣穴是根据病变脏腑主要病机所选的腧穴或穴点,应针刺到地部;佐穴是根据病变的次要病机所选的腧穴或穴点,应针刺到人部;使穴是根据病变部位所选的腧穴或穴点,应针刺到天部。

腹针进针时,一般按“先上后下、先内后外”的顺序。
