# 妇科科门诊服务务指南

# 1 范围

本文文件明确了妇妇科门诊服务务的术语和定定义、服务原则、服务内容容、基本要求求及服务质量量评价。

本文文件适用于开开展妇科门诊诊工作的医疗疗机构。

# 4 服务原则则

# 4.1 规范服服务

科学布布局，设计合合理，由执业专专业技术人员员严格按照诊诊疗流程，为为就诊患者提提供规范服务务。

# 4.2 文明服服务

按文明明礼仪要求，坚坚持以人为本本，以患者需需求为导向，充充分告知，自自主选择，为就就诊患者提供供热情、真诚、周到到的服务。

# 4.3 高效服服务

充分利利用医疗机构现有的医疗设设备、科学技技术和人才资资源，通过信信息化手段，数据共享，实现“一站式”就诊诊，为就诊患者提供安全、高效、便捷捷的服务。

# 5 基本要求求

# 5.1 总体布布局

5.1.1 按照照《医疗机构构基本标准（（试行）》（（2017 版）要要求，根据诊诊疗服务需求合理划分候诊诊区、诊室、检查室室、治疗室、手术室等区域域。

5.1.2 三级级医院和有条条件的二级医医院可在基本本设置基础上上设置门诊独独立手术室（人人流室、宫腔腔镜手术室、宫颈疾疾病手术室、隔离手术室等等）、专科 B 超室、阴道道镜室、盆底底康复治疗室室等。

5.1.3 可依依据病人不同同需求设置普普通门诊、专专家门诊、特特需门诊。

5.1.4 可设设置包括但不不限于以下亚亚专业门诊：

a) 宫宫颈疾病门诊诊；  
b) 妇妇科内分泌门 诊；  
c) 更更年期门诊；  
d) 计计划生育门诊诊；  
e) 流流产后关爱门诊；  
f) 盆盆底疾病门诊诊。

5.1.5 应设设置标识明显显、数量适宜宜的卫生间。  
5.1.6 应设设置绿色通道道，予残疾人人、军人、老老年人等特殊殊人群优先服务。  
5.1.7 宜设设置咨询服务务台。  
5.1.8 宜设设置母婴室。  
5.1.9 宜设设置工作人员员更衣室。

# 5.2 设施配配备

# 5.2.1 基础础

5.2.1.1 应应设置明亮适适度的室内照照明，并符合合节能环保的的要求。  
5.2.1.2 应应设置符合节节能环保要求求的空调设备备。  
5.2.1.3 应应配备满足诊诊疗服务的电电脑、打印机机、网络终端端等硬件及软件设备。  
5.2.1.4 应应设置意见箱箱、投诉电话话、多媒体液液晶屏幕、自自助服务终端、宣传栏等。  
5.2.1.5 应设置供服服务对象等候候的桌椅。  
5.2.1.6 应设置无障障碍通道与栏栏杆等无障碍碍设施。  
5.2.1.7 应配备应急急照明系统。。  
5.2.1.8 应设置数量量适宜的垃圾圾桶。  
5.2.1.9 可摆放数量量适宜的盆栽栽植物。  
5.2.1.10 宜设置排排队叫号设施施。  
5.2.1.11 宜设置饮饮水设施。  
5.2.1.12 宜设置手手机充电设施施。

# 5.2.2  诊疗

配备备应与医院级级别、科室功功能相适应。 三级医院和 有条件的二 级医院应配备备超声诊断仪仪、宫腔镜、腹腔镜、阴道镜、高高频电波刀等等。

# 5.2.3  消防

应符符合GB 500166的规定。

# 5.3 人员要求

![](images/c2b8024af890e33d003e768a9842a34ff9d8798a89f3cbb93193b179e6d49313.jpg)

# 5.3.1  业务素质

5.3.1.1 应根据医疗疗需要及工作作量，按《医医疗机构基本本标准（试行行）》（20177 年版）要求求，合理配备备不同类别别、级别与数数量的专业技技术人员。

5.3.1.2 宜参加相关关妇女心理健健康指导培训训，掌握基本本心理问题处处理方法。

# 5.3.2  仪表仪容

5.3.2.1 应着装统一一，工作服干干净整洁，佩佩戴工作牌， 不应穿短裤裤，拖鞋。

5.3.2.2 不留指甲及及美甲，不配配戴首饰。男 医务工作者宜宜穿西装衬衣衣打领带，女女医务工作者者妆容得体，不应披散散头发。

5.3.2.3 精神饱满 ，态度亲和，，仪态大方， 举止文明。

5.4 制 度建设

# 5.5 文 明服务制度

细化化文明服务相相关要求，其其中包括服务务态度、服务用语、着装要要求、形象要要求、工作准准则等内容。

# 5.6 预约诊疗制度

明确确预约时间、预约渠道，预约方式，预约诊疗服服务项目等内容容。

# 5.7 首 诊负责制度

明确确首诊科室和和医生的工作作职责，细化化首诊流程。

# 5.8 会诊制度

明确确会诊提出的的条件、时限限、会诊类型型，会诊程序。

# 5.9 转诊、、转科交接管管理制度

明确转转诊、转科的条件、时限，细化转诊、、转科交接流流程及转诊指指征管理要求求。

# 5.10 门诊诊就诊患者健康教育制度

明确健健康教育内容、形式、流程程等。

# 5.11 诊疗疗应急管理制 度

明确诊诊疗突发事件件定义、分类、职责、管理理要求和处置置流程。

# 5.12 人员员考核与奖惩惩制度

细化考考核项目、考考核内容、评分分标准、考核核办法、奖惩惩办法等。

# 5.13 查对对制度

明确操操作前、中、 后应查对的 内容、要求及及方法。

# 5.14 病历历质量管理制度

明确监监控组织，细化考核、质控控和监管等内内容。

# 6 服务内容容及要求

# 6.1 挂号

6.1.1 应按按预约诊疗制制度要求通过过医疗机构官官方网站、APPP、微信公众众号、电话、医疗机构现场、自助设备等渠道道，提供一种 或多种挂号服服务方式。6.1.2 宜根根据挂号的时时间及排序预预估患者就诊诊时间，实行行分时段预约诊疗服务。6.1.3 挂号号单上宜提示示需要到达候候诊区的时间间及候诊区相相应的地点和位置。

# 6.2 咨询

6.2.1 应配配备熟知医院院环境布局、服务流程、学科特色、检查诊疗项目特殊要求、专家特长的工作人员，提供专专业咨询服务。  
6.2.2 咨询询过程中涉及及隐私保护、就诊患者情情绪、家属陪陪伴等问题，应采用一对一一咨询方式，必要时丈夫或家属属可陪同共同咨询。

# 6.3 候诊

6.3.1 应有有明显的路径径指引标识，标识清晰、醒目，导向向易懂，字体规范。  
6.3.2 应在在候诊区设置置就诊注意事事项、常见妇妇科疾病知识识宣传栏或影视简介。  
6.3.3 应在在候诊区进行行候诊排队提提示。

# 6.4 诊疗

6.4.1 应按按首诊负责制制度要求提供供诊疗服务。  
6.4.2 应详详细询问现病病史、既往史史、药物过敏敏史、生育史史、家族史等等，通过体格检检查及妇科检检查，结

合实验室室检查、医学学影像检查等等，给出专业业、有效的治疗方案或指导导建议。

6.4.3 应向就诊患者者或其家属说说明病情及治治疗方式、特特殊治疗及处处置，并获得得其同意，应应记录诊疗内容并存档档。

6.4.4  医务人员完成成诊疗工作后后应主动指引引就诊患者进进入下一诊疗疗环节。

# 6.5 缴 费

# 6.5.1  支付方式

宜设设置网络支付付、现场支付付、自助支付付等多种支付付渠道，可采用用包括但不限限于以下方式式进行支付：

a) 医保卡；  
b) 银行卡；  
c) 支付宝；  
d) 微信；  
e) 现金等。

# 6.5.2  缴费凭条

缴费费凭条中应对对缴费项目明明细及金额有有详细说明或或清单。

# 6.5.3  缴费发票

缴费费同时可按需需生成缴费发发票。

# 6.6 检 查

6.6.1  检查单上应 明确检查时间间、地点、检检查注意事项项及特殊要求求。

6.6.2  需要特定条件件的检查，例例如空腹，休休息等，宜在在检查单上给给出温馨提示示， 明确检查查前的注意事事项。

6.6.3  检查前需口服服药物或特殊殊注意等，宜宜明确注意事事项及服药方方法。

6.6.4  检查为有创操操作，应明确确告知需要的的注意事项及及可能发生的的后果，充分分让就诊患者者知情，必要要时应根据据相关规定与与就诊患者或或其委托的家家属（代理人人） 、法定监护护人等签署知知情同意书。。

6.6.5  检查需预约时时，宜做到合合理安排检查查时间，并标标记告知就诊诊患者预约时时间。

# 6.7 会 诊

6.7.1 应按会诊制度度要求，对不不能确诊或久久治无效的就就诊患者，应应请上级医师师会诊，病情情复杂者可作作为疑难病病例提请科室室或小组讨论论。

6.7.2 涉及多学科会会诊时，由科科主任（门诊诊组长）向门门诊办公室提提出，门诊办办公室及时组组织，申请多多学科诊疗疗模式。

# 6.8 转 诊

6.8.1 应按转诊、转转科交接管理理制度要求，及时传递病病历与相关信信息，为就诊诊者提供连续续医疗保健服服务。

6.8.2 危急重症患者者转移时按转转诊、转科工工作流程， 在在就诊患者安安全情况下向向其他科室或或医院进行转转诊。

# 6.9 心理健健康

针对就就诊患者心理理问题，按照心心理评估-心心理疏导-综合合干预流程提提供辅导服务务。

# 6.10 教育育与随访

宜通过过门诊随访、电话随访、微微信、短信推推送随访等方方式提供随访访服务及健康康教育。

# 6.11 投诉诉监督

6.11.1 应应提供投诉电话、投诉邮箱箱、意见箱、、网上投诉等等监督渠道。  
6.11.2 应应按投诉监督管理制度要求求对投诉予以以答复，并告告知投诉人处处理时限及处处理结果。

# 7 服务质量量评价

# 7.1 评价原原则

客观公公正，民主公开。

# 7.2 评价方方式

7.2.1 医疗疗机构自我评评价。  
7.2.2 政府府相关部门评评价。  
7.2.3 就诊诊患者及家属属评价。  
7.2.4 第三三方评价。

# 7.3 评价指指标

7.3.1 就诊诊患者满意度度。  
7.3.2 家属属满意度。  
7.3.3 疾病病诊断准确率率。  
7.3.4 疾病病治疗有效率率。  
7.3.5 有效效投诉办结率率。

# 7.4 评价方方法

7.4.1 意见见征询(现场调查、电话访访谈、网上调调查)。  
7.4.2 检查查考核。
