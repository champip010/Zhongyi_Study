# 针灸技术操作规范 第3部分：耳针

# 1范围

本文件规定了耳针的操作步骤与要求、注意事项与禁忌。  
本文件适用于耳针技术操作。

# 2规范性引用文件

本文件没有规范性引用文件。

# 3术语和定义

下列术语和定义适用于本文件。

3.1耳针ear acupuncture采用毫针、压丸、埋针、刺血、电针、按摩等一系列操作刺激耳穴以防治疾病的一类方法。

3.2耳穴毫针法ear acupuncture therapy使用毫针刺入耳穴以防治疾病的一种方法。

3.3耳穴压丸法ear plaster therapy使用特定丸状物贴压耳穴以防治疾病的一种方法。

3.4耳穴埋针法ear needle-embedding therapy使用掀针埋入耳穴以防治疾病的一种方法。

3.5耳穴刺血法ear bloodletting pricking therapy使用一定针具点刺耳穴出血以防治疾病的一种方法。

;.6耳穴电针法ear electro-acupuncture therapy使用电针刺激耳穴以防治疾病的一种方法。

3.7耳穴按摩法ear massage therapy使用手或耳穴按摩棒刺激耳穴以防治疾病的一种方法。

# 4操作步骤与要求

# 4.1施术前准备

# 4.1.1针具及压丸(器具)选择

针具针身应光滑、无锈蚀，针尖应锐利、无倒钩，压丸应圆润、大小适宜、不易碎、无毒。按摩棒应不 易折断、无毒，施术端应圆钝、光滑、大小适宜。

# 4.1.2耳穴选择

根据病情，并结合中医理论、现代医学及临床经验选择耳穴。

# 4.1.3体位选择

选择患者舒适、医者便于操作的体位。

# 4.1.4环境要求

应注意环境清洁卫生，避免污染。

# 4.1.5消毒

4.1.5.1针具消毒：应选择一次性无菌针具。   
4.1.5.2部位消毒：应用75%乙醇或碘伏在施术部位擦拭。   
4.1.5.3医者消毒：医者双手先按七步洗手法洗净，再用75%乙醇擦拭。

# 4.2施术方法

# 4.2.1耳穴毫针法

医者一手固定耳郭，另一手拇、食、中指持针刺入耳穴。针刺方向视耳穴所在部位灵活掌握,针刺深度宜0.1cm～0.3cm,以不穿透对侧皮肤为度。针刺手法与留针时间应视患者的病情、体质及耐受度综合考虑。宜留针15 min～30 min,留针期间应间断行针1次～2次。出针时一手固定耳郭，另一手将针拔出，应用无菌干棉球或棉签按压针孔。

# 4.2.2耳穴压丸法

医者一手固定耳郭，另一手用镊子夹取耳穴压丸贴片贴压耳穴并适度按揉，根据病情嘱患者定时按压。宜留置2d～4d。耳穴压丸贴片应按照附录A制备。

# 4.2.3耳穴埋针法

医者一手固定耳郭，另一手用镊子或止血钳夹住撤针针柄刺入耳穴，用医用胶布固定并适度按压，根据病情嘱患者定时按压，宜留置1d～3d后取出掀针，应消毒埋针部位。

# 4.2.4耳穴刺血法

医者佩戴一次性乳胶或薄膜手套,刺血前宜按摩耳郭使所刺部位充血，医者一手固定耳郭，另一手持针点刺耳穴,挤压使之适量出血。施术后以无菌干棉球或棉签压迫止血并消毒刺络部位。

# 4.2.5耳穴电针法

用酒精棉球消毒耳穴部位,医者一手固定耳郭，另一手拇、食、中指持针刺入耳穴。针刺方向视耳穴所在部位灵活掌握，针刺深度宜0.1cm～0.3cm,以不穿透对侧皮肤为度。并根据病情选择好所需的波形和频率,将电针仪的一对输出导线的正、负极分别连接在两根毫针柄上（一对导线的正、负二极宜接在同侧耳郭，二枚以上者宜远距离相接配对),再拨动电位器开关,逐渐调高输出电流至所需的刺激量。电流刺激量应根据病情而定,宜留针15 min～3O min,出针时一手固定耳郭，另一手将针拔出,用无菌干棉球或棉签按压针孔。

# 4.2.6耳穴按摩法

医者用手或耳穴按摩棒对准耳穴部位,施以一定力度及手法，时间宜5min～10 min。应按照附录B具体操作。

# 5注意事项

5.1 施术部位应防止感染。  
5.2 紧张、疲劳、虚弱患者宜卧位针刺以防晕针。  
5.3 湿热天气、耳穴压丸、耳穴埋针留置时间不宜过长,耳穴压丸宜2d～3d,耳穴埋针宜1d～2d。  
5.4 耳穴压丸、耳穴埋针留置期间应防止胶布脱落或污染，对普通胶布过敏者宜改用脱敏胶布。  
5.5 耳穴刺血施术时，医者应戴医用手套避免接触患者血液。  
5.6 妊娠期间慎用耳针。

# 6禁忌

6.1脓肿、溃破、感染、冻疮、瘢痕局部的耳穴禁用耳针。  
6.2凝血机制障碍患者不应使用耳穴刺血法。  
6.3不能配合施术的患者，禁用耳针、电针及埋针等操作。

# 附录A

（规范性）耳穴压丸贴片的制备

将医用胶布剪成约0.6cm×0.6 cm大小，上置压丸制成耳穴压丸贴片,压丸直径约0.2cm,应清洗消毒,宜选用植物种子,如王不留行子、白芥子、急性子、莱菔子、油菜籽等;或选用聚苯乙烯珠、磁珠等。目前，临床上广泛使用的是王不留行子和磁珠。

# 附录B

# （规范性）

耳穴按摩手法

耳穴按摩手法主要包括按揉法、点按法、按掐法、提捏法、擦法、摩法。依据耳穴所在部位选择适合的按摩操作手法。

# B.1按揉法

是按法与揉法的复合动作,用按摩棒或手指罗纹面置于耳穴部位，前臂和手指施力，进行节律性按压揉动。

# B.2点按法

是以按摩棒或食、中指指端的罗纹面着力，垂直向下按压。

# B.3按掐法

是指术者手握空拳,拇指伸直，指腹紧贴在食指中节桡侧缘，拇指指甲垂直于耳穴，逐渐用力，达到“以指代针"之效。

# B.4提捏法

是指术者肩、肘关节放松,腕关节略背伸，用拇指与食、中两指相对用力挤压并向外提拉患者耳郭。

# B.5擦法

是用按摩棒或手指指腹贴附于耳郭表面，做较快速的往返直线运动，使之摩擦生热。

# B.6摩法

是以单指(拇指、食指、中指)或食指、中指、无名指与小指并拢，指掌自然伸直,腕关节略屈，以指面附着于耳郭部位，做环形而有节律的抚摩。
