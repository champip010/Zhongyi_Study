# 卵巢癌中西医结合诊疗指南编制说明

# 一、工作简况

# 1.任务背景

在妇科恶性肿瘤中，卵巢癌的死亡率最高，严重威胁妇女健康。其起病隐匿，70%卵巢癌确诊已属中晚期，上皮癌在卵巢恶性肿瘤中占90%。近年我国开展了众多高水平的循证医学研究，摸索中医药与手术、化疗、靶向治疗等结合的治疗规律。中西医结合治疗卵巢癌在临床中的良好疗效充分说明：两种治疗方法可以互相扬长避短，优势互补，有利于肿瘤患者获益。

由于目前中西医结合治疗卵巢癌的水平参差不齐,迫切需要一部有实际应用价值的中西医结合治疗卵巢癌的指导性文件，最大程度提高临床疗效。故采用现代循证医学原则并结合中医临床特点，编纂指南，供从事中医、西医及中西医结合工作的医师在临床实践中参考使用，对中西医结合诊疗上皮性卵巢癌的临床行为的科学化、规范化、合理化。

# 2.任务来源

本任务来源于中华中医药学会《卵巢癌中西医结合诊疗指南》制定计划，由中国中医科学院广安门医院负责组织完成。立项时间是2022年8月。

# （3）访谈提纲：

1.您认为中医在卵巢癌治疗中如何参与？2.您认为中医药治疗对患者术后的生存质量有何影响？ 3.您认为中医药治疗对围化疗期患者生存质量有何影响？4.您认为中医药单药或合并西医规范治疗对患者的生存期有何影响？安全性如何？5.您认为中医药治疗还对困扰卵巢癌患者的那些疾病或症状有改善？

# （4）访谈结论

1.中医在卵巢癌的手术、化疗、靶向、维持治疗阶段均可参与。  
2. 中医药治疗能改善患者术后淋巴囊肿、伤口难愈和腹胀腹痛的症状  
3.中医药治疗能改善围化疗期患者的骨髓抑制、脱发和消化道不良反应。  
4.中医药维持治疗能提高患者的生存质量，改善PARP抑制剂或TKI引起的不良反  
应。  
5.中医药还能治疗癌因性疲乏和焦虑抑郁。

# 2.临床调研

# （1）调研方案确定的过程及方法

充分了解一线临床医生的实际诊疗情况和诉求，并在此基础上形成的临床问题，可为指南或共识的制定提供方向性指导，使指南更具有临床价值。因此，在《卵巢癌中西医结合诊疗指南》制定中，项目组在全国范围开展临床调研研究，作为形成临床问题的关键步骤之一。

项目组基于国内临床医生面临的实际情况，开展临床调研研究。工作组利用问卷星发放电子问卷20份，对一些问题进行了调研，具体见附件一。调研问卷采用半结构化设计，问卷设计应紧密围绕指南的主题。本指南的主题是卵巢癌的中西医结合诊疗，因此临床调研应中西医并重，了解中医和西医的治疗情况，在此基础上重点关注临床医生在治疗卵巢癌时遇到的实际问题，如希望中西医结合的时机、患者对中医药的接受程度、临床用药依据、选择干预措施时考虑的因素、临床诊疗中存在的问题、对指南或共识的建议等。

# （2）调研对象分布情况

临床调研涉及11个地区（北京、上海、广东、山东、内蒙古、黑龙江、河北、陕西、贵州、江苏、甘肃），选择地域内有代表性的19家医院进行调研，大多数为三级甲等医院。选择有卵巢癌诊疗经验的中医、中西医结合及西医临床医生进行调研。本研究共调研20名临床医生，其中主任医师14名，副主任医师4名，主治医师2名。

# （3）调研结论

1）临床医生的诊疗应基于目前最佳证据，同时结合个人经验及患者意愿进行。  
2）在指南或共识内容上，应注重古今医案、医家经验对指南的指导作用。  
3）在卵巢癌的手术和化疗阶段，中医改善生存质量，维持治疗阶段，中医能预  
防复发转移。  
4）突出临床实用性。  
5）重视证据，但不局限于证据。

6）指南或共识可涉及卵巢癌的中西医诊断、围手术期并发症（伤口不愈合、肠梗阻、淋巴囊肿）中医药管理、围化疗期（消化道反应、骨髓抑制、周围神经损伤、脱发）的中医药管理、维持治疗相关不良反应的中医药管理（PARPi、小分子TKI）及焦虑、癌因性疲乏中医管理等方面内容。  
7）突出辨证要点，突出中医特色疗法。  
8）指南发布后注意指南的宣贯和再评价。

# 1.证据检索

# （1）检索方法

文献检索范围为中国知网（CNKI）、中国生物医学文献数据库(CBM)、万方数据库（Wan fang）、维普数据库（VIP）、Pubmed、Embase、The Cochrane Library，收集自建库至2022年5月中医药干预卵巢癌患者的Meta分析、系统评价、随机对照试验等，检索策略参照中华中医药学会《中医药整体证据研究的标准化操作规程》（2020年3月版）。文献语种为中文和英文。采取主题词和自由词相结合的方式检索，检索词根据不同的问题设置。

# 1.推荐意见形成方法
