# 《冠状动脉微血管病中医诊疗指南》推广方案

近年来，随着循证医学、中医药学和介入性心脏病学的迅速发展，冠状动脉微血管病的临床意义日益受到人们的高度重视。虽然在医学文献中出现冠状动脉微血管病的报道已有几十年的历史，但有关此病的中医诊断和治疗建议一直未形成共识。大量的研究表明，中医药在防治冠脉微血管病中具有一定疗效，但尚无专门针对冠状动脉微血管病的中医药诊疗指南或专家共识，我国中医药临床从业者对于此病的病因、发病机制、临床分型、诊断、治疗和预后等诸方面仍有进一步认识空间。因此，由刘红旭教授、林谦教授、韩晶岩教授、吴永健教授、毛静远教授、安冬青教授等人牵头，向中华中医药学会团体标准办公室提出申请，旨在服务临床医师，为冠状动脉微血管病的中医诊疗提供参考。为了增强本《指南》的影响力，工作组特制定了《指南》宣传推广方案，旨在让临床医生更快速全面地了解指南的相关内容，从而进一步指导冠状动脉微血管病的临床诊治。本推广方案具体内容如下：

# 1、同期发表学术期刊论文

《冠状动脉微血管病中医诊疗指南》（报批稿）及编制过程的相关材料正式提交标准化办公室并审核通过后，工作组将在《中医杂志》、《中国中西医结合杂志》、《中西医结合心脑血管病杂志》等学术期刊同期发表《冠状动脉微血管病中医诊疗指南》（杂志版），以方便全国广大医务工作者下载查阅，从而进一步扩大本指南在国内的影响范围。

# 2、学术论坛专题报告

首都医科大学附属北京中医医院心血管科每年会定期举办或承办多个学术会议或学术论坛，工作组计划以此学术平台为基础，邀请专家进行专题报告，向参会专家及临床医师介绍指南的制定背景、过程及内容等。具体学术会议或论坛名称如下：

（1）中国介入心脏病学大会（CIT）中西医结合论坛（2）冠心病学科交叉暨介入治疗大会（CMIT）中西医结合论坛（3）北京中医药学会学术年会（心血管专业委员会）（4）世界中医药学会联合会介入心脏病专业委员会学术年会

# 3、以医院为单位进行宣讲培训

指南工作组计划组织专家以线上或线下讲课的形式，以医院为单位，向北京地区及全国具有代表性的三级甲等中医医院或中西医结合医院，对指南内容进行宣讲培训。

目前暂定的医院名单主要包括：上海中医药大学附属曙光医院、上海中医药大学附属岳阳中西医结合医院、上海长征医院、广西中医药大学第一附属医院、山西省中医院、中日友好医院、中国中医科学院西苑医院、中国中医科学院望京医院、中国医学科学院阜外医院、中国医学科学院协和医院、天津中医药大学第一附属医院、天津市第三中心医院、长春中医药大学附属医院、北京中医药大学东直门医院、北京中医药大学东方医院、北京中医药大学第三附属医院、江西中医药大学附属医院、江苏省中医院、安徽中医药大学第一附属医院、河南中医药大学第一附属医院、陕西中医药大学附属医院、首都医科大学附属北京安贞医院、湖北省中医院、湖南中医药大学第一附属医院、新疆维吾尔自治区中医医院、黑龙江省中医院、福建省人民医院等。

# 4、举办继续教育项目

指南工作组拟向北京中医药学会申请举办《冠状动脉微血管病中医诊疗指南》推广应用培训班，组织专家从中西医两个角度对“冠状动脉微血管病”的定义、分类、诊断、治疗、《冠状动脉微血管病中医诊疗指南》制定的背景、制定流程、条目等内容进行全面、系统的讲解。
