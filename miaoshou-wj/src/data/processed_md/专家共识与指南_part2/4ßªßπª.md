# 推广应用方案

# 一、指南的实施单位

本指南发布后，拟先行在北京大学肿瘤医院和中国中医科学院西苑医院实施，在两院开展结直肠癌疼痛临床诊疗工作，规范结直肠癌疼痛中西医结合诊疗，提升结直肠癌疼痛中西医结合治疗水平，从而提高结直肠癌疼痛患者的生活质量。将来推广到更多的全国肿瘤专科医院、中医医院、中西医结合医院，进一步提升了结直肠癌疼痛单病种的诊疗水平。
