# 1.工作简况

# 1.1 任务来源

为了贯彻落实 2015 年 3月国务院印发的《深化标准化工作改革方案》，国家标准委员会办公室决定开展团体标准试点工作。试点工作期间，为促进国内外股骨头坏死中医辨证标准方案的规范化，为股骨头坏死中医临床实践提供可靠的辨证标准，提高股骨头坏死中医药临床医疗及科研研究水平，从而促进中医医疗和学术进步的进程。立项开展了股骨头坏死中医辨证标准的制订工作。本标准由中华中医药学会提出并归口，中国中医科学院望京医院承担起草，项目负责人陈卫衡。本项目计划于 2019年2月完成。

# 1.2 协作单位

中国中医科学院望京医院为本项目的主要完成单位，协作单位有中国中医科学院中药研究所、浙江省中医院、武汉市中医院、上海中医药大学附属曙光医院、洛阳正骨医院、江苏省中医院、广州中医药大学第一附属医院、甘肃省中医院、福建中医药大学。

# 1.3 主要工作过程

2017年在中华中医药学会组织下，于6月成立了以中国中医科学院望京医院为主要起草单位，广州中医药大学第一附属医院、大连大学附属中山医院、福建中医药大学、浙江省中医院、中日友好医院、上海曙光医院、中国中医科学院中药研究所、上海新华医院、郑州大学第一附属医院、甘肃省中医院、武汉市中医院、洛阳正骨医院、江苏省中医院、福建中医药大学参加起草的标准工作组，工作组起草人由陈卫衡，何伟，童培建，詹红生，林娜，李盛华，董晓俊，刘又文，沈计荣组成。6月～8月工作组完成文献研究报告：统计分析股骨头坏死辨证术语；文献计量研究方法搜集统计分析中医药疗法治疗股骨头坏死的辨证标准文献。2017年8月由本标准主要起草单位与其他30家协作单位共同召开了《股骨头坏死中医辨证标准》项目启动会，规范股骨头坏死中医辨证标准术语；探讨中医药疗法治疗股骨头坏死的辨证标准内容。2018年1月～3月，工作组成员开始起草股骨头坏死中医辨证标准，专家咨询探讨结合现有标准起草《股骨头坏死中医辨证标准》草案。2018年3月初项目工作组于北京再次召开专家论证会，项目组采用股骨头坏死专题会议方式广泛征求同行对《股骨头坏死中医辨证标准》草案建议并对标准草案进行修改。2018年7月项目工作组于北京再次召开《股骨头坏死中医辨证标准》初稿讨论会，征求对《股骨头坏死中医辨证标准》初稿修改意见。2018年11月项目工作组于北京召开《股骨头坏死中医辨证标准》同行专家评审会，广泛征求同行意见。项目工作组先后通过5轮现场及1轮电子问卷供发放183份问卷，收回180份问卷，供108位中医专家与72位西医专家参加问卷调查，专家积极系数98.36%。项目工作组按照“循证”等原则逐条讨论了专家们的意见，提出了采纳、不采纳的意见及理由，以之为依据，对征求意见稿进行修改。2019年2月完成《股骨头坏死中医辨证标准》，提交中华中医药学会审批发布。

# 2.标准编制原则和确定标准主要内容的依据

# 2.1 标准编制原则

本标准编制遵循“科学性、实用性、规范性”原则，按照“能够为中医行业内实际应用，能被行业外广泛接受和认可，并与国际诊疗标准接轨”的要求。

# 2.1.1 科学性

科学性是编制本标准的前提，也是保障标准质量的基础。本标准在编制过程中，严格遵循科学性原则。

本团队采用了具有中医药特色的“文献研究法”、“专家问卷调查法（Delphi 法）”、“专家会议法”三法合一的研究方法，保证了诊疗标准的科学性。

采用了中医药标准化研究团队中医循证标准研制的“文献研究法”，包括“文献检索”、“文献综述”、“文献评价”3 个方面的内容。其中“文献检索”按国际通行要求方法进行，要求查找古今中外与股骨头坏死相关文献；“文献综述”提出了保证标准研制质量的工作要求，对其中的临床文献按不同类别提出了文献评价方法并逐条文献进行了评价评分。专家问卷调查采用国际广泛应用的 Delphi 法为基础加以改良，结合中医药行业的具体情况，采用了具有代表性权威性的调查专家遴选办法、各轮问卷的形成方法、答卷数据处理的统计分析方法，使之成为有效凝聚专家共识的方法。

按中医药标准化研究中规定的“专家会议法”要求，邀请了以中医骨科学专家为主，相关药学、方法学专家等组成的专家论证组召开专家论证会，就项目工作组通过文献研究、专家问卷调查形成的标准草稿，特别是其中存有争议、有待讨论、商榷的内容，请专家们给出较客观的和专业化的意见，形成本标准标准初稿。

# 2.1.2 实用性

本标准制订的目的主要是促进国内、外股骨头坏死中医辨证标准的规范化，为股骨头坏死中医临床实践提供可靠的辨证标准，提高股骨头坏死中医药临床医疗及科研研究水平，从而促进中医医疗和学术进步的进程。使医护人员全面了解相关知识、易于实际操作，能在股骨头坏死的医疗工作中得到广泛的实际应用。

在标准研制过程中，查找了相关的现代大量的中医与中西医结合临床报道、学术著作与教材等，将相关论述经文献综述形成的参评因子组成调查问卷，调查了分布于全国各地区股骨头坏死的专家，集中他们的意见，再经过专家论证和同行业专家广泛征求意见，使本标准从研制过程到结果保证了其实用性和可操作性。

# 2.1.3 规范性

本标准在研制过程中，均按照国家中医药管理局政策法规与监督司、中华中医药学会的要求。所采用的技术方法，包括文献检索和文献评价方法、专家问卷调查方法、专家论证会方法等，均按照国际比较公认的方法进行。保证了本标准的研制方法，包括技术方法及形成的标准规格体例、名词术语、语言文字等的规范性要求。

# 2.2.2 确定标准主要内容的方法和依据

本标准的制定主要采用“文献研究法”、“专家问卷调查法（Delphi 法）”、“专家会议法”三法合一的研究方法。制定过程经历了三次专家论证会，六轮问卷调查，一次行业专家征求意见。综合专家意见反复修改完善，最后形成送审稿。

# 2.2.2.1 文献研究

（1）相关教材书籍：通过对超星数字图书馆及中国中医科学院图书馆所藏书籍进行查阅，利用纳排标准对书籍进行整理，检索到中医骨伤科学、中医骨病学教材等书籍。

（2）股骨头坏死中医辨证标准：ZY／T001.1-94《中医病证诊断疗效标准》；国家药品监督管理局修订，并于 2002 年发布的《中药新药临床研究指导原则》(试行)；北京市中医药管理局、北京中医协会联合修订，并于 2007年发布的《北京地区中医常见病证诊疗常规》；ZYYXH/T374-2012《成人股骨头缺血性坏死》；

GB/T 1.1—2009 《标准化工作导则——第 1部分：标准的结构和编写》。

（3）期刊文献：通过文献计量研究方法搜集统计分析《中国学术期刊全文数据库》（CNKI）、《中国生物医学文献数据库（CBMdisk）》、《中文科技期刊数据库（全文版）》、MEDLINE数据库中收录的涉及中医证型的中医药辨证治疗股骨头坏死的文献。

制定一套明确的文献纳入与排除标准，对文献进行筛选，阅读符合标准的文献。纳入文献主要选择：已发布的标准、规范、标准及教材，以及随机对照研究、半随机对照研究、名老中医专家经验等。排除文献为不足以影响临床的理论探讨，设计不好及写作较差的临床报道，非名老中医的、未取得广泛共识的自拟方的临床报道。

（4）检索结果：共检索到中文文献 8586篇，剔除重复文献后所得到的文献数量 1581 篇，阅读题目和摘要，剔除综述、动物实验研究、理论探讨、会议通知，得到的文献数量为 662 篇，全文阅读后，共纳入符合标准文献 147篇。

# 2.2.2.3 专家论证会

初稿完成后，项目组于 2018 年 7 月项目工作组于北京再次召开专家论证会，邀请 30 位中医骨科学、中药学、方法学等专家对初稿论证。会议就工作组提出的标准初稿和相关问题进行了研讨，对其中的技术内容进行充分论证，对初稿提出了进一步修改的意见。对于初稿中的若干具体内容，专家们通过讨论基本上达成共识，提出了许多有价值的修改意见。2018 年8-10 月项目工作组在修改初稿的基础上形成了最终的征求意见稿。

# 2.2.2.4 同行征求意见

# 3.与相关法律、法规和强制性标准的关系

本项目工作组研究形成的《股骨头坏死中医辨证标准》与现行法律、法规和强制性标准没有冲突，并且在编制过程中严格遵循已有的国际、国内标准，使文本内容符合规范，言之有据。

# 4.重大意见的处理经过和依据

在制定本标准过程中，项目组成员将难以解决的问题加以汇总，形成调查问卷，以电子问卷形式发送给相关专家，采用德尔菲(Delphi)法进行调研，并对专家意见进行了结果分析，采纳评分较高的意见。对于仍然无法解决的问题，项目组则通过专家共识会的方式，取得专家的一致性意见。编制过程中，项目组力求规范中涉及到的内容都有高质量的证据支持，让我们制订的规范标准每一项条目都有据可依。

# 《股骨头坏死中医辨证标准》专家调查问卷总结

对象：先后通过5轮现场及1轮电子问卷供发放183份问卷，收回 180 份问卷，供 108 位中医专家与 72 位西医专家参加问卷调查，专家积极系数 98.36%。调查专家专家名单如下：朱俊峰、周明旺、赵德伟、张善星、张明雪、张伯松、詹红生、袁普卫、阎作勤、许鹏、邢更彦、谢雁鸣、王智勇、王天芳、王庆普、王和鸣、童培建、田华、孙伟、沈计荣、商洪才、彭江、卢敏、刘又文、刘献祥、刘保一、林娜、李子荣、李振华、李学民、李盛华、李少华、李劲峰、冷向阳、康鹏德、靳英辉、金今、焦锋、纪泉、黄野、黄相杰、黄枫、胡永成、何伟、韩永台、杜炯、董晓俊、陈卫衡、陈棉智、陈继营等专家。

对专家答卷的统计分析，采用 Excel表格录入数据，按照数据统计结果分析汇总专家意见，总结讨论所有专家提出的意见。专家们基本认可问卷中的内容，对一些具体问题提出了自己的意见。专家对各项问题的赞同率：被调查专家对股骨头坏死中医辨证标准均给出了赞同意见。专家提出的主要问题有：有专家对辨证标准的部分细节提出了如提高各证型之间的辨识度，与影像分期结合易于西医使用等改进建议。专家对问题认定的重要程度：被调查对象对问卷中所设定的问题大部分都给出了“很重要”的评价。专家对问题的熟悉程度：绝大多数的被调查专家对问卷中的各个专业问题都是熟悉及比较熟悉。

具体专家意见如下：

（1）多数专家认为股骨头坏死中医证型应分为 4型。（2）选择气滞血瘀、痰瘀阻络、肝肾亏虚、经脉痹阻专家人数较多。（3）多数专家认为股骨头坏死中医证型应包括主症、次症、舌象、脉象。（4）股骨头坏死草案调查结果显示四型（气滞血瘀、痰瘀阻络、经脉痹阻。肝肾亏虚）（早中晚三期，主症、次症）股骨头坏死中医证型选择人数最多。（5）四型（气滞血瘀、痰瘀阻络、经脉痹阻。肝肾亏虚）（早中晚三期，主症、次症）股骨头坏死中医证型初稿同意人数79.16%。

（6）四型（气滞血瘀、痰瘀阻络、经脉痹阻。肝肾亏虚）（早中晚三期，主症、次症）股骨头坏死中医证型征求意见稿同意人数 80.00%。
