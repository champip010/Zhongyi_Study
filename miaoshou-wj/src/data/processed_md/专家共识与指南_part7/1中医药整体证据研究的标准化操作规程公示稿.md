# 引 言

循证医学对临床疗效研究证据评价与推荐的指导作用已得到了国际医学界的广泛认可。中医药临床疗效证据的循证医学证据总结亟需依托规范的技术方法和标准化操作规程。对中医药数千年来的经验进行系统整理和严格评价，实现对浩瀚文献的去粗存精、去伪存真，这是中医药继承与发扬的必由之路。针对中医药领域现代高质量随机对照试验等证据较少，古籍记载与专家经验、共识等其他证据对临床仍有重要指导作用的现状，本项目提出中医整体证据的理论，综合集成中医古籍、现代文献、名医经验、指南共识、教材专著等多来源证据形成证据体，进而制定规范中医药现代临床研究证据、专家经验与共识性证据、中医古籍证据挖掘等整体证据的一系列研究方法的标准操作规程，并涵盖实验室研究证据等支持性证据的系统收集、整理、分析和评价的过程，从而快速高质量地形成可供国际范围内应用的中医药临床循证医学证据，有利于推动中医药临床实践跟上现代循证医学的步伐，有利于解决中医临床教材循证资料缺乏，医师可借鉴最佳证据少的问题。对发展循证中医药学，巩固与提高我国在国际中医药教育、临床和科研中的地位具有重要的意义。

本标准提出中医药整体证据研究的标准化操作规程总体原则，明确了中医药整体证据的内涵及其整体证据研究的主要内容，将为中医药证据的系统研究、中医药领域的循证临床实践指南制修订、高等教育与继续教育、以及药物研发提供参考。

# 第 1 部分：总则

# 1 范围

本标准规定了中医整体证据的构成和研究涵盖的内容。

本标准适用于中医药循证临床实践指南的制修订、需要提供整体证据的行为规范指南等。

# 2 术语和定义

下列术语和定义适用于本标准。

# 2.1

临床研究证据 Evidence of clinical study

以人为研究对象、有特定研究目的和研究设计的临床研究，主要目的在于评价某种干预措施的临床疗效评价与安全性。

# 2.2

专家共识 Expert consensus

采用一种形式，通过一定的方法集合不同个体提出的多种建议或意见，并最终达成共识。

2.3

专家经验 Expert experience某领域内的专家个人对某些科学问题的经验（常见的是对某病的诊疗经验）。

中医古籍证据 Evidence of classical Chinese medicine literature中国古代各种医籍（主要指1912年以前）中记载的中医药诊疗经验、认识或理论。

# 2.5

整体证据 Whole evidence

针对某一疾病（或健康状态），系统整理的包括古籍证据、专家经验、共识性证据、临床研究证据和临床前实验室药理研究证据等多来源证据的整体，较传统的基于现代临床研究的循证证据范围更广。

# 3 整体证据研究方法的指导原则

3.1 遵循科学发展规律，促进中医药诊疗经验的去粗存精、去伪存真。  
3.2 符合国际临床研究、疗效评价、临床推荐的基本原则。  
3.3 充分考虑中医药学科发展阶段与临床诊疗特色。  
3.4 考虑疾病的国际治疗进展与中医药需求。  
3.5 系统整理及严格评价不同来源、不同类型证据。  
3.6 对中医药的临床疗效进行科学、客观、定量呈现。

# 4 整体证据方法研究与应用的基本内容

4.1 建立和完善中医药整体证据概念与方法。  
4.2 中医古籍证据的检索与评价方法。  
4.3 现代临床研究证据的检索与评价方法。  
4.4 临床前实验室药理研究证据的检索与汇总方法。

# 第 2 部分：古籍证据梳理的标准化操作规程

# 1 范围

本标准规定了中医整体证据中古籍证据的构成和研究涵盖的内容。  
本标准适用于中医药古籍经验证据的系统梳理和挖掘等。

# 2 术语和定义

下列术语和定义适用于本标准。

# 2.1

# 病（中医） Condition

即疾病，是致病邪气作用于人体，人体正气与之抗争而引起的机体阴阳失调、脏腑组织损伤、生理机能失常或心理活动障碍的一个完整的生命过程。

# 2.2

条文 citation中医古籍中关于疾病认识、诊疗等信息分条说明的文字。

# 3 确定疾病相关的古籍检索词

# 3.1 疾病相关古籍检索词的范围

在确定检索词之前，应明确临床需求，构建临床问题，结合古籍特点，确定检索词。疾病相关的古籍检索词通常包括：疾病相应的古代病名；该疾病关键临床症状和体征相应古籍描述的词语。如果该疾病具有较为明确的古代病名，则直接检索全部对应的古代病名；如果该疾病的古代病名尚不明确，则列举该疾病关键临床症状和体征的名词。

# 3.2 确定疾病的古代病名

# 3.2.1 收集古籍病名

收集古籍病名的途径主要包括：

a) 古籍词典或中医辞典：查找相关名词的同义词或近义词；  
b) 历史沿革的相关文献：检索该疾病认识、命名、治疗的相关文献，补充可能病名；  
c) 古籍条文：通过阅读相关条文，补充该疾病相关的古代病名。

# 3.2.2 古籍病名的相似度分类

将上述途径收集的古籍检索词按其与疾病现代诊断相似度分为“与疾病现代认识相符”和“不确定或内涵太宽广”两类。在中华医典中预检索，记录每个检索词命中的古籍条文数，并试读部分命中的条文以判断其与疾病现代认识的相似度。

# 3.2.3 确定临床症状和体征的古代名词

a）根据现代临床诊断标准列举该疾病的症状和体征；b）选择较有把握确诊该疾病的特异性临床症状、体征及病史特征组合，并转换成古代可能描述的名词或词组；c）转换名词或词组时应查找古籍词典，明确其确切含义；d）参考相关书籍和期刊文献，补充可能的检索词；f）在中华医典预检索，记录命中条文数，并试读部分命中的条文以判断其与疾病现代认识的相似度。

# 4 条文的纳入排除标准

a)全面回顾该疾病的诊断标准，参考以症状体征为主要诊断条件的标准；b)确定特异性症状、体征可能有相应的古籍描述词语，通过这些词语可以判断该描述很可能是该疾病（即纳入标准）；c)确定可能作为鉴别诊断的症状、体征相应的古籍描述词语，通过这些词语可以判断该描述与该疾病不相关（即排除标准）；d)通过这些特异性和鉴别性词语的合理组合，拟定古籍条文可用的纳入排除标准；e)对古籍条文中症状和体征进行编码，如：0=未提及该症状或体征，1=可能是该症状或体征，2=条文描述了该症状或体征；f)在初始条文库中，抽取部分条文，应用上述拟定的纳入排除标准进行试判断筛选，根据测试结果，调整纳入排除标准。

# 5 《中华医典》第五版的检索及条文筛选

# 5.1 《中华医典》第五版界面

中华医典（湖南电子音像出版社）软件主界面左侧为书目结构树，右上方为检索操作区，右下方为检索命中条文显示区。可在左侧的书目结构树中选择特定的书目类型或范围进行检索。

# 5.2 检索步骤

以中华医典收录的全部古籍中检索“咳嗽”为例。

示例1：

a) 选定搜索节点：根据需要在左侧目录结构树选择节点并勾选“从选中节点中搜索”，或者不勾选直接从整个中华医典内搜索；b) 选定检索范围：推荐“目录”（在结构树中包含书名、章节标题或内容等进行检索）或“正文”（在正文中进行检索）；c) 检索及结果：检索框中输入“咳嗽”，单击“搜索”后有显示查找的结果数的弹出窗口，关闭窗口则书目结构树区则会列出本次检索结果；

d) 条文查看：在书目结构树区域点击任意书目或章节标题，右侧的条文显示区则会展现正文的结果，正文最上一行红色字体显示的是条文的出处。

e) 结果合并及记录：在合并目录检索和正文检索的命中条文,并删除重复条文后，记录以上操作结果。

# 5.3 筛选确定初始条文库的条文

基于纳入排除标准和预检索的结果，删除与该疾病不相关检索词及其条文，确定初始条文库的条文，再筛选出与疾病现代认识高度相符的条文进行数据分析。如果没有高度相符的条文，可选择大部分描述相符的条文。在正式筛选前进行预筛选，两人独立筛选，两人意见不同时找第三方解决。

# 6 古籍数据处理

# 6.1 Excel 管理古籍条文

# 6.1.1 中华医典结果导出

中华医典可选择导出检索结果，也可重复手动复制条文到Excel中。每次只能复制当前页面显示的条文。

# 6.1.2 Excel整理及核对

检查确认已将在中华医典检索所得的全部条文汇总整理到Excel中，此处建议双人核对。对于重复的条文，同样导入到Excel文件中，但需明确标记。

# 6.1.3 Excel变量设置

除书名、章节、条文、检索词等变量外，还要设置条文序号、书籍编号、权重、条文分类、关键词、术语标注、作者、朝代等变量。

# 6.2 古籍条文查重原则

为了方便排除重复条文，Excel中的数据可按书名、章节和检索词进行排序，条文重复情况判定标准见附录A；有些条文里，一个症状名后附一系列方剂，每个方剂在Excel中需分开录入；当条文中仅提及方名但未描述组成，且该方剂组成有多个不同版本时，标记提及同一方剂的条文编号。此外，需使用统一的术语详细记录判定重复条文的原因。

# 6.3 数据编码

# 6.3.1 数据编码注意事项

采取双人独立对条文阅读并进行编码，然后核对两人编码的结果，对编码不同的条文，通过讨论和商量确定最终编码，并记录讨论结果。

# 6.3.2单个症状和体征的编码

根据需要选择单个症状和体征作为分类标准，在Excel表中将其作为单独变量并进行编码。例如：如果选择的疾病有急性与慢性之分，那么“急性/慢性”就是一个分类标准，编码参考如下：0=没有提及急性或慢性、1=急性、2=慢性。

# 6.3.3 编码检索词

为了记录每个检索词最终获得的条文数量，对命中每条条文的检索词进行编码。因为同一条条文可通过多个检索词检索到，所以将每个检索词作为单一变量标记，记录命中的条文。使用数字编码：0=该检索词未命中本条文、1=该检索词命中了本条文。

# 6.3.4 统一书名

收集目标条文时，往往会发现同一本书因出版年份、印刷版次或出版社的不同出现两个或两个以上的书名。因此需对书名进行标化，以避免数据分析产生缺漏或错误。

# 6.3.5 确定书本出版日期

书本出版日期是指写作年代或出版年代。对于已经确定书籍的成书或出版年代，这些可以直接采用；对于没有明确的，可选择作者死亡的年份、王朝灭亡的年份或根据参考资料推测最可能的年份。

# 6.3.6 标化或编码成书朝代

根据书名的整理结果，以及参考中华医典的成书朝代划分标准进行成书朝代的标化或编码。中华医典的成书朝代划分标准如下：

a) 唐代以前（618年前）  
b) 唐代及五代十国（618-960）  
c) 宋金（960-1271）  
d) 元（1271-1368）  
e) 明（1368-1644）  
f) 清（1644-1911）  
g) 民国（1912-1949）  
h) 现代（1950 至今）

# 6.3.7 编码方剂

如条文中出现方剂，则要对每个方剂进行对应编码。方名、组成都相同，或者组成相同方名相似的可用统一编码。而方名相同但组成不同则分配不同的编码，并在备注中用方名加数字区分，如归脾汤 1、归脾汤 2。

# 6.3.8 编码中药

将方剂按组成进行拆分，每味中药占据一个单元格。同一中药的不同名称，如薯蓣和山药，根据中国药典使用的中药名进行标化，保留原始记录。如药名对应不同的几味中药,例如：地黄、熟地黄和生地黄，则分别进行编码，分析时记录总体应用频数及各分类频数。

# 7 古籍数据分析

Excel筛选、编码完毕后导出到SPSS进行数据分析，分析前清理数据并检查数据的准确性，制定统计分析计划。

采用SPSS统计分析软件的语法（syntax）功能，并于开始分析前，设计好统计语法，记录分析的目的、分析的时间和分析语法。古籍条文常用数据分析有：

a) 对纳入条文中提及的方剂和中药进行频数分析；

b) 根据相似度评分逐步选择相似度高的条文进行分析；  
c) 根据分析的需要，使用“重新编码”（recode）功能，合并或生成新的变量；  
d) 方剂组成的聚类分析。

# 8 古籍证据评价

针对古籍的证据评价，结合古籍的记载特点，考虑从古医籍来源的权威性、医家的资质、古籍记载的形式和内容的完善程度、古籍应用的传承性等方面进行评价，获得古籍证据评价结果，进而结合现代研究的证据进行临床决策。

A.1 一个检索词，如“肺胀”的条文在同一本书中可能出现两次。如果两处条文的内容相同，则明确标注为“与第XX条文相同”，且不需进行数据提取。

# 3 术语和定义

下列术语和定义适用于本标准。

# 3.1

Meta 分析 Meta-analysis

在系统评价中使用统计学方法来合并纳入研究的结果。有时会被误认为是系统评价的同义词，系统评价通常包括Meta分析。

# 3.2

随机对照试验 Randomized Controlled Trial, RCT

将两种或两种以上的干预措施(可包括对照干预措施或无干预措施)随机分配给受试者进行比较的试验。大多数情况下，一种干预措施被分配到一个单独的个体，但有时也被分配到某个特定群体里中的个体(例如，一个家庭中的个人)，或干预措施分配到一个单独的个体自身(例如，干预措施在同一个体干预的先后不同顺序或干预措施分配在同一个体自身的不同身体部位)。

# 3.3

# 非随机对照试验 Non-randomized study

任何评估干预措施(危害或益处)有效性的但没有使用随机化将受试单位（个体或群体）分配到不同比较组的定量研究。这包括队列研究、病例对照研究、前后对照研究等。

# 3.4

偏倚风险 Risk of Bias, RoB

偏倚风险是指结果或推论中发生系统性错误或偏离真实值的风险，通常指在研究的设计、实施、分析和报告过程中产生的系统误差。

# 3.5

系统评价 Systematic review

对一个明确提出的临床问题使用系统的、明确的方法来检索、筛选和客观地评价相关研究，并收集和分析纳入研究的数据的一种综述。可用或不用统计学方法(Meta分析)分析和合并纳入研究的结果。

# 4 疾病认识的梳理方法

# 4.1 疾病的中医认识

中医对疾病的认识包括病因病机、辨证分型、诊断、治疗、预防调摄等信息，可参考教科书、专著和指南进行相关内容的梳理。其中，教科书包括国家制定的供中医和中西结合专业本科生和研究生使用的规范教材。

中医指南或中西医结合指南则通过数据库检索，检索时检索词应包括：“指南”“标准”“共识”“路径”以及疾病名称、中医（含中药、其他中医疗法、预防调摄）等。具体中医的疾病认识梳理的表格记录示例见附录A。

关于病因病机的源流，除了参考教科书、专著和指南，必要时可参考《中华医典》梳理相关内容。最终，病因病机的源流需通过专家咨询达成共识。

# 4.2 疾病的西医认识

西医的疾病概述包括西医流行病学、危险因素、诊断、防治、并发症等信息，可通过检索教科书、指南、文献数据库、政府网站等参考资料来收集以上信息。

流行病学资料的内容应当包括患病率、发病率、疾病负担、直接经济负担、间接经济损失等。西医疾病概述参考资料整理记录参照附录B。

在确定现代研究检索词前，应基于研究的目的构建临床问题，从临床研究的PICO核心要素结合研究类型制定纳入排除标准，然后进行检索筛选和评价。

# 5.1 收集病名检索词

# 5.1.1英文病名检索词

# 5.1.1.1 英文电子数据库检索

通过英文电子数据库检索收集英文病名有以下几种方式：

a) Cochrane Library 数据库：参考系统评价和研究方案检索策略中的“病名”检索词，或参考Cochrane Review Group相关文献使用的检索策略；b) PubMed 数据库：在MeSH 主题词检索框输入疾病名称，根据需要选择是否拓展检索词，注意最后一次MeSH主题词修订时间；c) Embase 数据库：在Emtree 主题词检索界面中选择“Advanced”，在检索框中输入检索词，在“Mapping”中根据需要选择是否拓展检索，单击“Search”。

查找非Cochrane 系统评价中的病名相关检索词；查找病名相关文献中的关键词；用病名名称的拼音进行检索。

# 5.1.2 中文病名检索词

# 5.1.2.1 检索相关文献

如果疾病在古代有相应病名，则要查找该疾病在古籍和现代中医类文献中出现的中医病名，具体操作请参考5.1。除此之外，参考研究疾病相关系统评价获取检索词，查找现代中医、中西医、西医教科书，也是收集该疾病的现用病名的有效途径。

# 5.1.2.2 中文电子数据库查找

通过中国生物医学数据库（SinoMed）的主题词功能，查找相应的主题词及上下位词，维普、知网等数据库通过查找同义词及检索历史模块，确定相应的近义词或同义词。

# 5.1.2.3 其他查找方式

使用上述关键词、同义词或近义词进行检索；文献下载筛选后，查看文献的摘要、全文、参考文献补充病名检索词。

# 5.2 收集干预措施和研究类型的检索词

收集干预措施和研究类型的确定操作与病名检索词一样，英文检索词可参考附录C，中文检索词参照附录D。需注意在不同的数据库时需根据数据库特点调整整个检索式的格式和结构。

# 5.3 检索词筛选

查找到的所有检索词都要进行记录，创建检索词筛选列表，然后根据查找结果命中数以及抽查准确性确定纳入与排除的情况。如果一个词包含另一个检索词，则应当将另一个检索词排除。如无法判断，用“AND”逻辑式连接这两个检索词进行预检索。如果检索数量与其中一个检索词的相同，则证明该检索词包含另一个检索词；如果检索数量小于任一检索词的结果，则两个检索词是交叉关系，均无需排除。

检索式检索则是将检索词之间用布尔逻辑运算符“OR”“AND”“NOT”连接。与分步骤检索一样，同类检索词之间用“OR”连接，不同分类检索词用“AND”连接。又因为运算顺序（ ）> NOT> AND > OR，所以要先将同类检索词用括号括起来再与其他类检索词“AND”连接。一般不建议使用“NOT”，因为数据库定义的主题词未必符合研究目的，不利于检索结果的全面性。

a）通过预检索检测检索策略的全面性，是否存在文献漏检的情况。

b）在相关出版物的参考文献中查漏补缺。如果预检索未能检出出版物引用的文献，可在PubMed或SinoMed中检索该文献以确定它是否被标引。若已被标引，但预检索未能检出，进一步完善检索策略。

c）对检索策略进行修改时，要将修改后的检索策略另存为新文件，并标注修正的原因。每次检索词的预检索都要记录检索结果并根据结果的数量及准确性判断是否保留。

# 6.4 文献题录导出

# 6.4.1 基本步骤

文献题录导出基本步骤如下：

a) 检索结果按发表时间排序，以防因网络稳定性或其他问题导致导出题录的重复或缺  
漏；b) 将每页显示设置为数据库允许的最高上限，尽可能减少页面切换；c) 全选当前页的所有文献结果，按顺序翻页重复此操作直至达到数据库每次导出题录  
的数量上限；d) 点击含“导出”二字的操作选项，在文献导出格式中均选择“EndNote”并单击“导  
出”，SinoMed数据库除外；e) 重命名导出生成的TXT文件，并保存在本地计算机；f) 将原已选文献的记录清除，翻页重复以上操作直至导出所有题录。

每次导出题录数量的上限：中国生物医学文献数据库及中国知网不超过500条，万方数据知识服务平台及重庆维普数据库不超过50条。

# 6.4.2 SinoMed 导出内容设置

a) 单击检索结果列表右上方的“结果输出”；b) 在“选择输入方式”项目中选择“保存”；c) 在“选择保存格式”项目中选择记录号，并填写与所需输出结果对应的记录号；d) 在“选择保存格式”项目中选择“自定义”，将弹出子项目“选择字段”，勾选：标题、摘要、关键词、作者、作者单位、出处，然后单击“确定”。

# 7.1 导入题录

# 7.1.1 导入设置

EndNote导入文献的方法有很多，此处主要介绍TXT格式转换的题录导入方法。打开或新建Endnote Library，单击“File”> “Import File”将弹出相应窗口，根据需求设置“ImportFile”弹出窗口的选项。弹出窗口各项说明如下：

a) 单击“Edit”> “Preferences”；b) 在弹出窗口“EndNote Preferences”选择“Display Fields”模块以设置Library界面显示内容；

c) 推荐的界面显示字段主要包括： Author、Year、Title、Journal、Abstract、Rating等；d) 研究者可根据Author、Title、Year等主要字段，通过不同的组配方法完成文献的除重e) 因为各数据库导出的题录在标点符号、空格或字母大小写上有所差异，EndNote系统自动查重无法检测，所以在经过系统自动除重后，可选择进一步手动查重。

# 8 编码及数据提取

# 8.1 题录编码

# 8.1.1 按治疗措施分类编码

根据文献报道的治疗措施进行分类编码。

示例7：

a) E，与中医治疗不相关的题录；  
b) H，中药治疗类的题录；  
c) A，针灸及其他于穴位相关的治疗类题录；  
d) O，其他如太极、导引等中医疗法治疗类的题录；  
e) B，中医疗法联合西医疗法的治疗类题录；  
f) U，根据标题和摘要仍无法确定分类的题录。

# 8.1.2 按研究类型分类编码

根据文献类型进行分类编码。  
示例8：a) S，系统评价；  
b) G，综述；  
c) R，随机对照试验；  
d) C，对照试验。

更具体的题录编码细则请参考附录E。在进行全文下载、浏览后再最终判断题录编码，尤其是U编码的题录。最终纳入的文献题录用“Y”标记。

# 8.2 数据提取

文献数据提取一般耗时较长，通常需提前制定工作计划，以提高工作效率，避免工作重复。根据研究目的、纳入排除标准和纳入文献数据特征建立数据提取表，然后按照以下几个注意事项提取数据：

a) 特殊语种：所有数据应以原始语言提取，对于非英语或非中文的文献，可请专人协助数据提取；b) 单元数据：提取数据时确保每个单元仅输入一个数据以便必要时排序，如果该变量无数据，则输入'NS'（NotSpecified）；c) 方剂名及中药名：方剂名称及其组成中药名需完全按照文献中的描述提取，且使用与原文献相同的语言,必要时可对其进行编码和分类，以便于统计分析；d) 检索日期：为便于识别数据来自原始记录和后续检索的记录，标注提取或检索日期；e) 数据提取操作：两人各自独立同时进行数据提取，提取完成后进行结果对比，检查录入数据一致性，若有异议必要时请第三人共同协查。

# 8.3 数据编码

在现代和古籍文献数据分析中，常采用数据编码系统地处理数据，将其拆分成较小的可分析单元，以便于统计分析。编码簿创建需做到以下几点：

a) 列出纳入分析的所有变量类型及其所有可能的分类；b) 变量名位于首列，命名尽可能简短；c) 对每个变量进行赋值（通常为数值），提前定义缺失值的编码，如NA（NotApplicable）；d) 将编码簿作为单独文档保存为Microsoft Excel工作表，使用前复制该文件。

# 9 评价

# 9.1 偏倚风险评估（RoB）

随机对照试验的偏倚风险评估采用Cochrane系统评价手册的偏倚风险评估工。Cochrane偏倚风险评估工具包括以下7个条目：

a) 随机序列的产生(选择偏倚)；  
b) 分配隐藏(选择偏倚)；  
c) 研究参与者和受试者采用盲法(执行偏倚)；  
d) 结果评估者采用盲法(观察偏倚)；  
e) 结果数据的完整性(失访偏倚)；  
f) 选择报道(报告偏倚)；  
g) 其他。

根据附录H中详细描述的评价标准，评价各条目可能存在偏倚为“低风险”或“高风险”或“不清楚”，并给出判断依据。以文字、表格或图示方法显示所有纳入文献的评价结果。

针对非随机对照研究的偏倚风险评估，针对不同研究类型，偏倚风险工具较多，如Newcastle-Ottawa scale[1]，Downs-Black工具[2]，ROBINS-I等[3]，部分工具尚处于初步使用和完善阶段，具体内容参见相关的说明手册及国内学者的解读[4,5]。

# 9.2 证据质量评价与结果的呈现

# 9.2.1 研究要素的重要性评价

a) 以研究问题和结局指标重要性制定PICO评级表格，参考示例见附录I；  
b) 组建评价小组，该小组应该由临床专科人员、方法学研究人员和临床咨询专家组成；  
c) 评价小组所有成员独立地评价PICO评级模板中各条目重要性；  
d) 随后汇总成员组意见，通过会议对做出最后的决定。

# 9.2.2 证据总结表创建

证据总结表可在GRADEpro网站完成。步骤如下：

a) 登陆其网址http://www.gradepro.org；b) 选择“新建”或从RevMan“导入”项目来制作证据总结表；c) 根 据 分 级 标 准 得 出 结 论 ， 具 体 操 作 可 参 考 GRADEpro 手 册（http://www.guidelinedevelopment.org/handbook）；d) 证据总结表范例可见附录J，而证据总结表相关定义和规范表达可参照附录K。

# 10 临床前实验室药理研究证据梳理与汇总

# 10.1证据梳理

查找研究疾病相关文献明确发病机制和通路。若发病机制通路未知或不明确，则明确治疗药物的作用途径及其代表性作用机制，例如皮质类固醇和抗肿瘤坏死因子药物。明确中药作用途径的检索步骤：

a) 检索中医、西医全国统编教材(供本科生或研究生使用)和《中药大辞典》。  
b) 明确中药的植物学名、拉丁文名和活性成分（分类归纳）。  
c) 使用相关检索词检索PubMed 和EMBASE。

将检索结果根据以下纳入标准进行筛选：①动物模型实验研究，如烟雾诱发的小鼠COPD模型；②细胞学分子学实验研究，如肺上皮细胞因子通路；③近期发表在高影响因子期刊上的研究；或研究疾病领域专家发表的相关综述。

# 10.2证据总结

主要包含以下内容：

a) 描述中药名及其活性化学成分，例如甘草主要活性成分是三萜类化合物，这包括甘草甜素、黄酮类、生物碱和其他化学成分如甘草次酸等；

b) 归纳总结与研究疾病治疗相关的活性成分作用机制，如在急性肺炎小鼠动物模型中，黄酮类化合物甘草甙、甘草苷以及甘草素分别以30mg/kg灌胃均可减少支气管肺泡灌洗液中嗜中性粒细胞、巨噬细胞和淋巴细胞的数量；c) 根据纳入研究亮点和重点进行讨论，例如某中药现有实验研究证据阐明的某些药理作用和机制在某临床试验中也得到了印证。

# 11 整体证据汇总及推荐

# 11.1整体证据汇总

将中医指南或教科书推荐的方药及针灸等疗法、古籍中常用的方药及针灸疗法、现代临床研究中常用的方药及针灸等疗法的证据进行客观的汇总呈现。

# 11.2证据推荐

结合目前的指南、古籍及现代研究证据，综合考虑卫生资源配置、利弊权衡、患者的意愿等因素，同时结合医生的经验，做出临床推荐。当推荐意见不一致时，可采用德尔菲法或名义群体法等共识方法达成一致，也可参照应用GRADE网格对推荐意见达成共识。

# G.1 中国生物医学文献数据库(SinoMed)

a) 通过网址http://www.sinomed.ac.cn/进入SinoMed系统主页，点击页面右上角“请  
选择数据库”的下拉框中，选择“中国生物医学文献数据库”。b) 可以在“高级检索”模块直接输入检索式进行检索，也可以直接在“快速检索”模  
块分步骤检索。c) 该数据库将默认“快速检索”在“全字段”及“主题词”检索。

G.2 万方数据知识服务平台

a) 通过网址http://www.wanfangdata.com.cn/index.html进入万方数据知识服务平  
台主页；b) 单击检索框旁的“高级检索”进入新网页后选择“专业检索”，根据需要选择文献  
类型，在检索框内按万方检索语法编辑检索式；c) 平台限制每次检索框内输入的字符不超过800个。

G.3 中国知网数据库

a) 通过网址http://www.cnki.net/进入CNKI文献服务系统主页；b) 点击检索框右侧的“高级搜索”；c) 根据需要在“跨库”限定检索的子数据库范围，如学术期刊、博硕、会议、报纸、  
年鉴、专利、标准、成果等；d) 检索途径建议使用“专业检索”功能；e) 根据研究目的及检索框下方提示选择相应运算符，推荐运用“SU”主题字段检索；f) 检索框限定最多仅能输入500个字，超出限制的检索式要进行拆分，干预措施检索  
式可拆分，但病名不建议拆分；g) 拆分的检索式通过检索框右下方的“结果中检索”实现AND逻辑关系连接。

G.4 重庆维普数据库

a) 通过网址http://www.cqvip.com/进入CQVIP文献服务系统主页；b) 进入“高级搜索”使用“直接输入检索式”模块；c) 根据CQVIP检索表达式语法编辑病名、干预措施和研究类型的检索式（“\*”代表“并且”，“+”代表“或者”，“-”代表“不包含”）；d) 建议选择“题名或关键字”（即“M”）作为检索字段。该数据库检索式检索无字符限制。
