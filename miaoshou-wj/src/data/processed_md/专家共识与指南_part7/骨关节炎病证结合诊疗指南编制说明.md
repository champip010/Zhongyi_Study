# 《 骨 关节炎病证结合诊疗指南》

# 编 制说明

# 《骨关节炎病证结合诊疗指南》标准编制组二零一九年六月

# 一、概述

# （一）任务来源

中华中医药学会团体标准制修订资助。

# （二）目的意义

风湿病属于临床常见、多发﹑疑难病。风湿性疾病包括 100 余种疾病，临床表现复杂多样，往往侵害人体多个系统，每种疾病发病率不一，男女比例各不相同，常见风湿病的患病率为：类风湿关节炎 0.3～0.4%；强直性脊柱炎 0.3%；系统性红斑狼疮 75～115/10 万人；骨关节炎 40 岁人群中占 10%～20%，60～70 岁者75%；如果把每种疾病的发病率汇总，风湿病的患病人数当以亿计，该类疾病对人类健康造成严重的危害已引起国内外专家的高度重视。我国风湿病患病基数大，患病人群构成复杂，边远地区和贫困人群因医疗水平或经济原因，很多风湿病患者没有接受规范治疗，严重影响患者预后。目前风湿病发病原因不明，可能与感染、遗传、雌激素水平等有关，环境因素（如寒冷、潮湿等），以及劳累、营养不良、外伤、精神刺激等均可以诱发。风湿病治疗所使用的药物包括免疫抑制剂、生物制剂、激素、非甾体消炎药等。早期、积极联合应用常规改善病情的抗风湿药，以及生物制剂的介入，使患者预后有很大改善。然而，长期、联合应用改善病情抗风湿药物出现诸多不良反应（如肝肾损害、骨髓抑制、皮疹、消化系统反应等），使用生物制剂的巨大经济负担，现有药物疗效的局限性，使很多患者的疾病控制不佳。

中医药对风湿病的研究已有数千年的历史，历代医家具有创新和发挥，历史医籍皆有论述，且形成了系统、完整的理论体系。早在《黄帝内经•痹论》指出“风、寒、湿三气杂至合而为痹”，最早提出了痹病的病因，随着自然气候、人类生活饮食习惯的改变及对疾病认识的发展，逐渐发现运气偏胜感受湿热邪气而可发为湿热痹，也可由寒湿痹日久不愈，或素体阴虚阳胜，或久用过用温燥药，寒从热化而成，因此热痹逐渐增多。从病因，病机分析，患者外感风寒湿热之邪，合而致痹；精神情志损伤，导致气血阴阳失调而致痹；饮食失宜，嗜食肥甘厚味，辛辣之品，损伤脾胃而致痹；劳逸过度耗伤人体的精血而致痹。本虚多为气阴不足，肝肾亏虚，标实为风寒湿邪，痰瘀痹阻。近几十年来，中药、中西药联合治疗是中医治疗风湿病的主要方法。中医治疗作为风湿病治疗不可或缺的组成部分，目前仍以整体观念、辨证论治、个体化诊疗为主要的诊治观念。中医治疗方法，包括复方汤剂、中成药、单味药、中药外治，缓解风湿病病情的疗效是有目共睹的。风湿病多以四肢关节多发肿痛等局部体表病变为临床特点，中药内服对改善关节局部症状起效慢，而中医外治法在很大程度上提高了治疗的效果，在风湿病的治疗过程中具有不可替代的作用。

中华中医药学会风湿病分会成立于 1983 年，由路志正担任第一届主任委员，王承德教授任第二届、第三届主任委员，姜泉教授任第四届主任委员。在历届主委领导下，在朱良春、谢海州、王为兰、陈之才、张沛虬、李济仁、娄多峰等老专家的支持指导下，始终坚持将提高临床疗效与人才培养作为风湿病分会的主要任务，重视中医药继承发展和科技创新，引领全国中医风湿病学术发展。经 30余年努力，专业委员遍布全国各省市，主办了 19 届全国风湿病学术大会，7 届国际风湿病学术研讨会和“海峡两岸风湿学术研讨会”，扩大了中医风湿病研究在国内外的学术影响。学会专家承担了国家科技部七五至十二五攻关（支撑）计划、国家自然科学基金、国家中医药管理局、各省市级课题等，在全国各省市建立了学术分支机构，指导和引领了各地风湿病学科的发展，在风湿病学界具有重要的学术地位和影响；举办 10余期全国学习班，各地学习班 20余期，培养了大批专科人才；主办《风湿病与关节炎》杂志，报道中医风湿病临床科研动态；研制5个痹病系列药，并在临床推广，目前仍然服务于临床，提高了全国各地中医风湿病的整体医疗和科研水平。

本病中医药治疗较为棘手，需要辨病和辨证有机结合起来，既要重视疾病发生发展的过程，又要立足疾病各个阶段和不同时期辨证施治，才能整体把握疾病预后、转归和治疗。而在中医药治疗风湿病临床中，存在着缺乏临床规范化应用方案、缺少高级别循证医学证据、尚无公认的临床诊疗指南等问题。病证结合的临床诊疗指南制定研究是中西医两种医学体系有机结合的具体体现，以促进广大中医及中西医结合工作者的普遍认可和在临床诊疗和研究中得以广泛应用，推动中医诊疗规范化的进程。因此，亟需制订符合基于循证医学理念的适用中医药临床使用的病证结合诊疗指南。

# 二、编制依据和原则

# （一）文献检索范围

本部分工作委托中国中医科学院中医药信息研究所完成。文献涵盖了风湿病分会建立以来的 33 年的中医药治疗骨关节炎的文献，检索数据库包括 CNKI，VIP，万方，PUBMED 等，重点检索年限：2006 年-2017 年。

结果共检得中文文献 3059 篇，英文文献 234 篇。初筛后共得到中文文献篇2717，英文文献167 篇进行文献质量评价。

# （二）文献评价取舍的原则

（1）文献资料应尽可能符合循证医学原则者。  
（2）有临床参考价值的病例组的文献资料。

（3）国家科技部“五年计划”课题、国家自然科学基金课题、国家中医药管理局课题等国家级课题中相关的文献资料。

（4）著名中医风湿病专家的文献资料。  
（5）文献资料的真实性强，临床研究设计科学。  
（6）文献资料的疗效可靠，使用安全。  
（7）资料要少而精，要具有代表性。  
（8）符合中医药辨证、西医疾病诊断等。

# （三）文献推荐意见和证据分级的定义

采用国际通用 GRADE分级及推荐。

# （四）专家咨询认证

指南初步形成后，我们采取了德尔菲法和头脑风暴法相结合的原则，制定了专家咨询调查表，咨询调查的主要内容是骨关节炎临床常见的证候、证候要素、治法和方药、治疗策略等。在对专家咨询调查表的分析处理的基础上起草《中医骨关节炎诊疗指南（草稿）》并组织相关专家进行四轮论证会议进行论证评价。考虑到中医老专家经验在中医药治疗中的特殊作用，我们又增加了一轮中医风湿病病学会顾问论证过程，力求指南的制订能体现学术传承及循证为主，经验为辅，专家共识的学术理念。

适用于中医药治疗风湿病的临床诊断、治疗、预防和调摄。
