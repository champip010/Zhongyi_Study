# 前 言

本标准按照 GB/ T 1 .1—2009 给出的规则起草。

本标准由中华中医药学会提出并归口。

本标准参考 2012年发布的《成人股骨头缺血性坏死（ZYYXH/T374-2012）》、1994 年发布的《中医病证诊断疗效标准》、2002 年发布的《中药新药临床研究指导原则（2002年版--试行）》、《北京地区中医常见病证诊疗常规》与“三期四型”的辨证思路。早期股骨头坏死分气滞血瘀和痰瘀阻络为二型，中期为经脉痹阻型，后期为肝肾亏虚型。

本标准起草单位：由中国中医科学院望京医院负责起草，中国中医科学院中药研究所、浙江省中医院、武汉市中医院、上海中医药大学附属曙光医院、洛阳正骨医院、江苏省中医院、广州中医药大学第一附属医院、甘肃省中医院、福建中医药大学参加起草。

# 引 言

本次修订的《股骨头坏死中医辨证标准》(以下简称《标准》) 以指南的形式发布，主要针对股骨头坏死制订中医辨证标准，供股骨头坏死中医药临床医师，骨伤科讲师与科研工作人员使用。主要目的是推荐有循证医学证据的股骨头坏死中医辨证标准，指导临床医务人员在辨证基础上规范使用中医药，有利于中西医结合临床，科研和教学工作。

股骨头坏死采用中医药治疗已经在相关的西医共识和指南中得到推荐，早中期的疗效优势也得到逐步认可，但股骨头坏死的辨证分型方法众多，缺乏较为统一和规范的辨证标准。中医辨证是中医开展临床治疗和科研教学的前提，必须制订相关标准。国家中医药管理局和国家药品监督管理局分别于 1994、2002 年制定发布的《中医病证诊断疗效标准》和《中药新药临床研究指导原则》中，前者将股骨头坏死的中医证型分为气滞血瘀型，风寒湿痹型，痰湿型，气血虚弱型，肝肾不足型等 5 型，后者则分为筋脉瘀滞型，肝肾亏损型 2 型，但仅气滞血瘀型，风寒湿痹型，痰湿型，肝肾不足型在临床中较为常用。而中华中医药学会 2002年发布的《成人股骨头缺血性坏死（ZYYXH/T374-2012）》标准中，又将其分为气滞血瘀证，筋骨劳损证，寒湿凝滞证，骨蚀痰湿证，内损酒精证和气血两虚肝肾亏虚证等六型，从文献计量研究发现，虽然此标准发布已有十余年，但仅气滞血瘀证临床中常用，其余五种证型临床应用较少。由此可以看出，中医药治疗股骨头坏死亟需适用于中医药临床的股骨头坏死辨证标准，提高股骨头坏死中医药临床医疗及科研研究水平。本标准所列证型为临床常见证型，临床工作中如遇到不符合本辨证标准的特殊证型，可根据具体情况按照中医理论进行辨证。股骨头坏死的中医辨证标准将在临床推广应用中不断完善，在未来循证医学证据的支持下定期修订。

# 3 术语和定义

下列术语和定义适用于本文件。

# 3 .1

股骨头坏死（Osteonecrosis of the Femoral Head，ONFH）

股骨头坏死（ONFH）又称股骨头缺血性坏死（Avascular Necrosis of theFemoral Head，ANFH），是指由于股骨头血供中断或受损，引起骨细胞及骨髓成分死亡与随后的修复，继而导致股骨头结构改变、塌陷，关节功能障碍的疾病。本病可分为创伤性和非创伤性两大类，前者主要是由股骨颈骨折，髋关节脱位等髋部外伤引起，后者在我国的主要原因为皮质类固醇的应用及酗酒。

# 3 .2

中医辨证 Traditional Chinese Medicine Syndrome Differentiation中医辨证是在中医理论指导下，对四诊所收集资料进行辨别、分析、综合和

判断其证候类型的思维过程，即确定现阶段属于何证的思维加工过程。它是将患者的体质、正气强弱、周围环境和疾病特点综合考虑的一种诊断方法，是中医治疗的首要依据。

# 4 股骨头坏死中医辨证标准

# 4.1 早期

# （1）气滞血瘀证

多见于早期（ARCO Ⅰ—Ⅱ）创伤性股骨头坏死。主证：①髋部疼痛，痛如针刺，痛处固定；②关节活动受限。次证：①面色暗滞；②胸胁胀满疼痛；③舌紫/青/暗或有瘀斑；④脉弦或涩。

判定标准：具备主证 2 项与次证1项或主证1项与次证2 项，即可判定为本证。

# （2）痰瘀阻络证

多见于早期（ARCO Ⅰ—Ⅱ）非创伤性股骨头坏死。主证：①髋部疼痛，或有静息痛；②关节沉重。次证：①胸脘满闷；②形体肥胖；③舌胖大苔白腻或舌紫/青/暗或有瘀斑；④脉弦涩/滑或沉涩/滑。

判定标准：具备主证 2 项与次证1项或主证1项与次证2 项，即可判定为本证。

# 4.2 中期

# 经脉痹阻证

多见于中期（ARCO Ⅱ—Ⅲ）股骨头坏死。主证：①髋痛至膝，动则痛甚；②关节屈伸不利。次证：①倦怠肢乏；②周身酸楚；③舌暗或紫；④脉涩而无力。判定标准：具备主证 2 项与次证 1 项或主证 1 项与次证 2 项，即可判定为本证。

# 4.3 晚期

# 肝肾亏虚证

多见于晚期（ARCO Ⅲ—Ⅳ）股骨头坏死。主证：①髋部疼痛，下肢畏寒；②下肢僵硬，行走无力。次证：①腰膝酸软；②下肢痿软无力；③头晕或健忘；④舌淡苔白；⑤脉沉而无力。判定标准：具备主证 2 项与次证 1 项或主证 1 项与次证 2 项，即可判定为本证。
