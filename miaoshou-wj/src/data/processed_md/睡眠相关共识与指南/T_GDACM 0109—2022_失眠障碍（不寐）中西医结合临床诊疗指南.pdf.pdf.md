# 引言

随着经济发展、生活节奏的加快，人们生活压力越来越大，失眠发生率越来越高，已成为全球第二精神障碍类疾病。中国睡眠研究会研究报告称，我国成年人失眠发生率达38.2%。长期失眠伴有明显的焦虑甚至抑郁情绪，严重者因失眠导致冲动、自伤甚至自杀；失眠同时也是诸多躯体疾病诸如心脑血管、内分泌疾病等的独立预测因子。综合性医院就诊病人中，40%伴有不同严重程度的失眠，广泛分布在耳鼻喉科、消化科、神经科、内分泌科及心血管科等，如未得到规范治疗，不但增加临床风险且消耗大量的医疗卫生资源。 》

虽然镇静催眠和精神类药物是治疗失眠的一线药物，在改善睡眠质量、提高睡眠效率等方面作用显著，但长期应用或滥用镇静催眠药物会引起药物依赖、成瘾、反跳、需要终生服药等棘手的临床问题，因此一部分患者有明显的惧药心理，导致依从性差。此外，目前对失眠的诊治并未分级、分人群诊治，对于轻度或阈下失眠应用镇静催眠药，容易导致过度治疗、认知损害等问题，若不治疗，则患者痛苦，并可能由轻度失眠进展为中、重度睡眠障碍和精神障碍；对于重度失眠及慢性顽固性失眠患者，常常因其社会心理学因素不明确，而延误了治疗，导致预后不良、迁延难愈、临床疗效不佳。对于老年性患者、青少年患者、孕产期哺乳期患者，镇静催眠药物临床使用受限且疗效不理想。

中医治疗失眠障碍强调整体观念，个体化辨证论治，患者具有较好的治疗体验，依从性高。目前，已有很多的临床研究证实中医药治疗失眠疗效显著，安全性高，且无依赖性。而非药物治疗，尤其是睡眠认知行为治疗、具有中医特色的调息治疗等，能够提高患者对于睡眠生理及失眠的认识，调整患者对于睡眠的恐惧和焦虑，帮助患者自我调节睡眠卫生习惯，可以产生持久的疗效且安全性高。中医药及非药物治疗失眠障碍己成为当前临床应用和医学研究的热点。

睡眠医学属于新兴学科，全国的睡眠领域专家较少，从事睡眠疾病诊治的临床专家和研究者广泛分布在神经科、心理科、呼吸科、麻醉科及中医科等众多学科领域当中，专业背景多样；失眠患者也广泛于这些科室就诊或转诊。然而，目前对于失眠的中西医结合诊疗策略缺乏统一的认识，因此，整合多学科理论，对中西医结合诊疗失眠的理论进行进一步探索和补充，充分挖掘和发挥结合西医、中医及具有中国特色睡眠认知行为治疗的优势，形成《失眠障碍中西医结合临床诊疗指南》，对于提高及优化失眠障碍诊疗规范具有非常重要的意义。

《失眠障碍中西医结合临床诊疗指南》整合现代医学和中医药理论，充分发挥中西医结合诊治的优势和特色，全方位、全链条、更精准、更个体化地指导失眠的诊治，具体特点为：

a）根据失眠严重程度建立了分级、分阶段治疗失眠的中西医结合诊疗策略。匹兹堡睡眠质量指数（Pittsburghsleepqualityindex，PSQI）<15分为轻、中度失眠障碍，口服中药汤剂：PSQI≥15分为重度睡眠障碍，口服中药汤剂联合镇静催眠药。体现出不同失眠严重程度的精确性治疗，更符合临床需要，细化了既往诊疗模式：  
b）基于失眠错综复杂的病因，本指南重视失眠患者的睡眠质量、心身症状及社会心理影响因素；针对顽固性、反复发作的失眠障碍患者，重视其人格特点、家庭环境等影响因素，给予相应的心理干预推荐;  
c）补充及整合了传统疗法和心理治疗，重视患者睡眠自我管理，结合中国传统情志理论并参考西方的各项心理治疗技术，推荐患者进行调息训练、认知团体治疗、中医心理干预等，探索出了具有中国特色的失眠临床精准诊疗模式，进一步提高了患者睡眠质量，以达到起效更快、维持疗效佳。

本指南是基于循证依据和专家共识进行制定。

# 3 术语和定义

下列术语和定义适用于本标准。

3.1

# 不寐insomnia

脏腑机能紊乱，气血阴阳失调，导致不能获得正常睡眠的疾病。

3.2

# 失眠insomnia

尽管有合适的睡眠机会和睡眠环境，依然对睡眠时间和（或）质量感到不满足，并且影响日间社会功能的一种主观体验。

注：失眠的主要症状表现为入睡困难（入睡潜伏期超过30min）、睡眠维持障碍（整夜觉醒次数≥2次）、早醒、睡眠质量下降和总睡眠时间减少（通常少于6.5h），同时伴有日间功能障碍。失眠引起的日间功能障碍主要包括疲劳、情绪低落或激惹、躯体不适、认知障碍等。

3.3

# 调息治疗pranayama

通过深慢吸气、呼气方式，以及有意识的屏息来放松身体，缓解情绪，从而促进睡眠的方法。

3.4

睡眠认知行为治疗cognitivebehavioraltherapyforinsomnia

学习并维持适应性的睡眠习惯，减少自主或认知上的唤醒，建立关于睡眠的恰当信念和态度,寺续进行健康睡眠实践的教育。

# 3.5

焦虑认知治疗cognitivetherapyforanxiety

通过治疗师带领患者，了解焦虑相关障碍常见的表现和引起焦虑的负性思维，分析焦虑症状形成和持续存在的条件刺激因素，并且传授患者学习放松方法，尤其应对焦虑急性发作的治疗方法。

# 3.6

# 抑郁认知治疗cognitivetherapyfordepression

通过治疗师带领患者了解抑郁相关表现，帮助患者和亲属识别并应对抑郁情绪，建立对抑郁发作的科学认知，并且向患者及家属传授治疗过程中如何应对消极观念行为等风险情况，遵医嘱服药的必要性，形成治疗同盟等的治疗方法。

# 4诊断

# 4.1中医诊断标准（标准）

中医诊断标准参照ZY/T001.1-1994中医病证诊断疗效标准制定。

# 4.2西医诊断标准（标准）

西医诊断标准参照2014年美国睡眠医学会（AmericanAcademyofSleepMedicine）出版的第三版《国际失眠障碍分类（InternationalClassificationofSIeepDisorders）·失眠（Insomnia）》制定，包括慢性失眠障碍和短期失眠障碍。

# 4.3鉴别诊断（指南）

参考《中医失眠障碍综合防治指南》(2019年版)。

# 4.4病史采集（标准）

需对失眠患者的睡眠卫生习惯（睡前行为、睡眠觉醒节律、夜间症状、日间活动和功能）和一般情况（用药史、既往史、手术史、应激因素、婚育史、家族史等，女性患者需了解其妊娠、月经史）采集信息，从躯体、生理、心理、精神、药物五大方面（5P病因）问询，以提炼出患者失眠的易感因素、诱发因素和维持因素（3P模型），依据中医望闻问切四诊收集中医症状和体征，并结合精神心理检查综合评估失眠的核心病因。

# 4.5睡眠质量评估（标准）

睡眠质量评估包括睡眠量表评估和睡眠客观检查。

# 4.5.1睡眠量表评估

睡眠量表评估宜采用以下量表:a）匹兹堡睡眠质量指数（PittsburghSleepQualityIndex，PSQI）量表b）失眠严重程度指数（InsomniaSeverityIndex，ISI）量表

# 4.5.2睡眠客观检查

睡眠客观检查宜使用多导睡眠监测（Polysomnography，PSG）

# 4.6心身症状评估（临床建议）

心身症状评估宜采用康奈尔医学健康指数（CornellMedicalIndex，CMI）问卷进行评估。

# 4.7心理评估（临床建议）

心理评估可从情绪症状、个性特点、人际关系三方面评估患者的心理状况。

# 4.7.1情绪症状评估

情绪症状评估宜采用以下量表进行评估：

a） 焦虑自评量表（Self-RatingAnxietyScale，SAS）b） 抑郁自评量表（Self-RatingDepressionScale,SDS）

# 4.7.2个性特点评估

个性特点评估宜采用以下量表进行评估：  
a） 明尼苏达多项人格测试（MinnesotaMultiphasicPersonalityInventory，MMPI)  
b） 防御方式问卷（DefenseStyleQuestionnaire,DSQ）

# 4.7.3人际关系评估

人际关系评估宜采用以下量表进行评估：  
a）父母养育方式评价量表（EgnaMinnenavBarndomsUppfostran，EMBU）  
b）Olson婚姻质量问卷

# 附录B

# （资料性）睡眠认知行为治疗内容设置

# B.1睡眠刺激控制

睡眠刺激控制的目的是结合时间（卧床时间）和环境（床和卧室）因素刺激快速入睡，并建立规律的睡眠-觉醒生物钟节律。

a） 仅在感到有困意时卧床睡觉。  
b） 如无法入睡（例如，20分钟后）便起床，更换房间，如再次感到睡意再返回卧室。  
c） 减少所有干扰睡眠的活动，不吃东西，不看电视，不听广播，不在床上计划或解决问题。  
d） 每天早晨按时起床，无论前夜睡眠时间的长短。  
e） 避免日间小睡。

# B.2睡眠限制

a）每天定时记录上床、起床时间和估计睡眠总时间，并计算睡眠效率。（睡眠效率为睡眠时间占全部卧床时间的百分比）。  
b）如果本周平均每晚的睡眠效率达到90%以上，则下周可提早15～30分钟上床；如果睡眠效率在80%～90%间，则下周维持原来上床时间；如睡眠效率低于80%，则下周上床时间要推迟15～30分钟。  
c）根据上述原则，通过周期性促进睡眠效率，直至达到足够的睡眠时间。  
d）注意事项：鼓励患者要坚持做睡眠日记，严格按照自己的睡眠效率调整上床时间。

# B.3睡眠卫生

睡眠卫生教育的目的是提供干扰睡眠抑或促进睡眠的生活方式（饮食，运动，药物使用）和环境因素（光线，噪音，温度）方面的信息。a） 就寝几小时前避免使用兴奋剂（如咖啡因，尼古丁，浓茶）。b） 在就寝时间避免饮酒，这会导致睡眠片段化。c） 20点后避免剧烈运动。d） 就寝前至少有1小时的时间放松。e） 保持卧室安静，黑暗和舒适。f） 保持有规律的睡眠时间表。

# B.4睡眠认知调整

a）帮助患者了解睡眠知识，纠正患者如“打鼾无害”、“睡前喝酒有助睡眠”、“能记住梦是良好睡眠的标志”、“睡觉不择时间都可以”等常见不良认知。  
b）帮助患者认识到自己对于睡眠的错误认知，以及对失眠问题的非理性信念与态度，使患者重新树立起关于睡眠的积极、合理的观点，从而达到改善失眠的目的。
