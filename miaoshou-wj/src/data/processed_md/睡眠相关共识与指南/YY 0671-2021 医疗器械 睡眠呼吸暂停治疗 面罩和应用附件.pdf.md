# 引 言

睡眠呼吸暂停是正常呼吸在睡眠期间发生有临床意义的间歇性缺失。近年来随着对睡眠呼吸暂停风险意识的显著提高，睡眠呼吸暂停治疗设备的使用已经越来越普遍。本标准涵盖了设备使用过程中用以保护患者的面罩和其他应用附件的基本安全和基本性能要求。

# 5 结构要求

# 5.1 面罩接头

面罩接头如果是圆锥的，则应是符合YY/T1040.1或YY1040.2要求的15mm或22mm外圆锥接头。

非圆锥面罩接头不应与符合YY/T1040.1或YY1040.2要求的圆锥接头相配合，除非其满足YY/T1040.1或YY1040.2关于锁接、断开锁接和泄漏的要求。

通过检查和功能测试来检验是否符合要求。

# 5.2 生物相容性

在正常使用中，与患者或患者气体通道接触的部件或材料，应按GB/T16886.1的规定进行评价。

对于插入鼻腔或口腔的部件或材料应按黏膜接触类进行评价。

对于不插入鼻腔或口腔的部件或材料(例如：面罩弯头、管路、衬垫、支撑架),气体通道材料应按与皮肤接触类进行评价。

对于面罩材料，包括预期接触患者头部的头带，应按与皮肤接触类进行评价。

所有材料应按GB/T16886.1中的分类规定视为持久接触类型注：分类为持久接触类型是因为睡眠呼吸暂停治疗设备和附件的累积使用时间大于30天。

面罩或附件不能使用天然乳胶。

面罩或附件的制造商应在风险管理过程中阐述由于物质析出或泄漏至气体通道带来的风险。应特别注意致癌的、致突变的或有生殖毒性的物质。

面罩或附件，如果含有致癌、致突变或有生殖毒性的邻苯二甲酸酯(盐),则应在面罩或附件本身或者在包装上标明含有邻苯二甲酸酯(盐)。如果面罩或附件的预期用途包括对儿童、孕妇或哺乳妇女的治疗，则在风险管理文档中应包括使用邻苯二甲酸酯(盐)的具体理由。含有此类邻苯二甲酸酯(盐)的面罩或附件的随附文件应包含这些患者群体使用后的剩余风险，如适用，还应包含适当的预防措施。

通过GB/T16886.1的应用，检查随附文件，检查风险管理文档中对致癌、致突变或有生殖毒性的物质的识别及使用的理由，来检验是否符合要求。

# 5.3 \*避免重复呼吸的保护

# 5.3.1 正常状态下的保护

应提供在正常状态下将重复呼吸风险降至最低的方法。这些方法可以通过面罩或其他附件，或者睡眠呼吸暂停治疗设备实现。

在正常状态下，在如下测试条件下，CO相对浓度增加不应超过20%：

最小额定压力下；压力为5hPa(5cmHO)；和压力为10 hPa(10cmHO)。按附录F所述测试方法来检验是否符合要求。

# 5.3.2 单一故障状态的保护

对于覆盖口部的面罩，应设计为尽可能减少单一故障状态下的重复呼吸。  
在单一故障状态下，在如下测试条件下，CO相对浓度增加不应超过60%：一呼吸管路堵塞情况下；和一呼吸管路的设备端与大气相通。  
注：对于仅覆盖鼻部的面罩，在单一故障状态下，患者可通过口部进行正常呼吸。  
按附录F所述的测试来检验是否符合要求。

# 5.4 清洗、消毒和灭菌

面罩和所有附件，无论单一患者重复使用还是多患者重复使用，都应设计成使沾污特性最小，同时易于操作者清洗。

用于多患者重复使用的面罩及所有附件包括其部件均应制造成可以清洗和消毒或清洗和灭菌的样式。

面罩和所有附件及其部件的清洗和消毒的处理或(重复)处理方法应使用能体现它们的预期使用寿命的可清洗或清洗与消毒周期的次数来确认。

在面罩及其所有附件和部件的随附文件中，关于灭菌处理或重复处理的操作说明应符合YY/T0802—2020 和GB/T19974的要求。标识为无菌的面罩及其所有附件应使用GB/T19974中适当的，经过确认的方法进行灭菌。

非无菌器械的包装系统，应设计成使预期在使用前灭菌的产品保持预期的清洁度。

通过检查随附文件和评审处理方法有效性的确认来检验是否符合要求，包括面罩、附件和部件在重复处理后符合其性能规范的验证。

# 5.5 单一故障状态下的呼吸

对于覆盖口部的面罩，应提供在单一故障状态下限制吸气和呼气阻力的措施。流量为50L/min时，气阻不应超过10hPa(10cmHO)（在患者连接口处测量）。

睡眠呼吸暂停治疗设备通常结合不同制造商生产的面罩和应用附件一起使用。然而，大多数睡眠呼吸暂停治疗设备的患者端压力是可调的，所有的连接附件和面罩位于压力控制区域以外。因此，这些面罩及附件对患者接收的治疗压力有直接的影响。睡眠呼吸暂停治疗设备有益于患者通过增加咽喉横截面积和降低上呼吸道的压缩性这样一种重要原理来实现。睡眠呼吸暂停治疗设备用于通过面罩和附件向患者传递治疗压力。

连接到睡眠呼吸暂停治疗设备的面罩和附件的气阻取决于呼吸气体流量，尤其是比较高的吸气峰值流量可能会引起患者连接口和患者呼吸道之间明显的压降，因此，患者不能获得所需的治疗压力，阻塞性呼吸暂停的概率增加，治疗目的就会达不到。而且，睡眠呼吸暂停治疗设备一方面依靠对睡眠呼吸暂停治疗设备的设计从而将室息的风险最小化，另一方面依靠患者的保护机制对单一故障作出反应患者觉醒以避免可能造成的伤害。因此，本标准延伸了睡眠呼吸暂停治疗设备的性能标准，以确保传送治疗压力和防止室息。

这些要求和测试方式为使用者和操作者选择合适的睡眠呼吸暂停治疗设备、面罩、附件及连接件提供指导，同时也为确保面罩和附件与睡眠呼吸暂停治疗设备之间的相容性提供指导。

附录G（规范性附录）振动和噪声

# G.1 原理

振动和噪声测试的目的是测量面罩和所有附件产生的噪声，不包括呼吸管路或睡眠呼吸暂停治疗设备产生的噪声。

# G.2 设备

使用下列物品：

a） 被测面罩或附件；  
b) 制造商推荐的呼吸管路和睡眠呼吸暂停治疗设备；  
c） 符合GB/T3785.1规定的1级声级计的麦克风；  
d） 符合ISO3744：2010规定的带有频率A加权特性和时间F加权特性F的声级计。

# G.3 步骤

按如下所述进行测试：

a）封住面罩或附件的患者端，将面罩和所有附件放置在声反射面。  
b）按照制造商提供或建议的方法安装呼吸管路。  
c） 如果设备提供湿化器，测试过程应包含湿化器。  
d）采用合适的方法隔离测试区外的呼吸管路和设备的声音，这样，来自呼吸管路和气流的声音就不会影响面罩和任何附件的声音测量。

注：隔离是为了确保呼吸管路或设备发出的噪声或气流产生的噪声不影响面罩和附件的声音测量。

e）设置设备使其患者连接口处达到10hPa(10cmHO)的持续压力。  
f）在ISO 3744:2010中7.2.3和8.1.1规定的反射面上近似自由场区域内，以面罩和附件的几何中心为球心画一个半径为1m的半球，并按照ISO 3744：2010中8.2.1的要求，使用声级计的麦克风，在这个半球上测量10个点的声压，该声级计要符合GB/T3785.1中规定的1类设备的要求。  
g） 测试过程中确保外部噪声的A计权本底水平背景声压级至少比被测值低6dB。  
h）按ISO3744：2010中8.2.4的规定，计算测量表面上的A计权平均声压级水平。  
i）按 ISO3744：2010中8.2.5 的规定，计算A计权声功率级水平。
