# 保健用品功能学评价指导原则及试验要求第15部分：改善睡眠、醒脑通窍保健类功能学评价动物试验

# 1范围

DB61/T999的本部分规定了改善睡眠、醒脑通窍保健类保健用品的功能学评价动物试验的试验项目、试验方法、观察指标、结果判定和注意事项。本部分适用于陕西省改善睡眠、醒脑通窍保健类保健用品的功能学评价动物试验。

# 2术语和定义

下列术语和定义适用于本文件。

2. 1

睡眠 sleep

是一种与觉醒相对的、自发性的、周期性的和可逆性的感觉减退、意识减弱甚至消失，机体与外界的联系减弱，失去对环境精确适应能力的生命状态。

2.2记忆memory是指识记、保持、再认识和重现客观事物所反映的内容和经验的能力。

2.3翻正反射rightingreflex是指动物四脚朝上后，主动恢复正常体位的过程。

# 3试验项目

# 3.1 适用原则

宜采用镇静试验、催眠试验评价受试品改善睡眠的功能；宜采用学习记忆试验评价受试品醒脑通窍的功能。

# 3.2镇静试验

镇静试验应以活动计数为指标记录试验动物的自主活动，反映受试品的镇静功能。本试验宜选用的试验方法包括：

a) 计算机视频跟踪技术测定法；  
b) 红外线自发活动记录仪记录法；  
c) 走动时间和举双前肢试验；  
d) Vidiomex.V图像分析法;  
e) 开放场试验;  
f) 探测身体辐射热分析法；

1 拮抗中枢兴奋试验。

# 3.3 催眠

# 3.3.1对阈下剂量催眠药物的协同作用

应采用小鼠为试验动物，观察受试品对腹腔注射阈下剂量镇静催眠药物（如戊巴比妥钠）的协同眠作用；应记录翻正反射消失和恢复的时间，以睡眠潜伏期、睡眠时间和睡眠率为指标反映受试品的协同催眠作用。其中，各指标应按照以下计算方法确定：

a) 睡眠潜伏期：睡眠潜伏期=翻正反射消失的时间-给镇静催眠药的时间;  
b) 睡眠时间：睡眠时间=翻正反射恢复的时间-翻正反射消失的时间;  
c) 睡眠率：睡眠率=翻正反射消失的动物数/腹腔注射镇静催眠药的动物数。

# 3.3.2氯苯丙氨酸大鼠失眠模型

本模型试验应在向大鼠腹腔注射氯苯丙氨酸28小时～30小时且其出现失眠状态后，检测以下物质的含量和特性：

a) 脑组织中氨基酸类神经递质（如5-羟色胺、γ-氨基丁酸、多巴胺）的含量;  
b) 脑组织中一氧化氮含量和/或一氧化氮合酶活性;  
c) 血清细胞因子（如白介素1、白介素2、肿瘤坏死因子、干扰素）的含量。

# 3.3.3大鼠电刺激失眠模型

应将大鼠置于刺激器内，通过笼底部的电栅刺激足底，每30分钟刺激30秒，连续8小时，造成大鼠失眠，并记录其脑电图。

本模型试验的观测指标应包括：

a) 总睡眠时长;  
b) 觉醒时间;  
c) 慢波睡眠I期时长;  
d) 慢波睡眠U期时长;  
e) 异相睡眠时长;  
f) 快动眼睡眠时相时长。

# 3.4学习记忆

# 3.4.1 水迷宫法

应调置水迷宫仪的水温，保持水温恒定。对试验动物进行空间分辨学习记忆训练后，将其放入装有图像采集系统的水迷宫仪中，进行数据采集。

应通过观测以下指标观察试验动物的学习记忆能力：

a) 总活动度；b) 总平均速度;c) 逃避潜伏期;d) 上台前路程;e) 上台前平均速度。水迷宫试验类型包括Morris水迷宫法、通道式水迷宫法、隐蔽式站台水迷宫法和逃避式站台水迷宫法。

# 3.4.2 跳台法

应选小鼠为试验动物进行本试验。跳台仪底部铜栅电流强度应适当。当训练时，小鼠触电后跳上跳台以逃避电击，而获得记忆。记忆力与下台潜伏期呈正比。

本试验的观测指标及其计算方法应符合以下规定：

a) 试验动物遭电击的时间即为其在台下的时间；  
b) 试验动物逃避电击的时间即为其在台上的时间;  
c) 错误次数即为试验动物跳下台的次数。

# 3.4.3 避暗法

本方法又名穿梭箱法。利用鼠类喜好钻黑洞的习性，将明、暗两箱用隔板分开，底部留洞；并在暗箱底部铺设带电铜栅。将小鼠放入明箱，当其钻入暗箱即受到电击而返回明箱，从而获得记忆。在此过程中应对试验指标进行观测，并应在24小时后再次依法测定。

本试验的观测指标应包括：

a) 试验动物由明室进入暗室的潜伏期;  
b） 试验动物5分钟内进入暗室的次数。  
其中，其记忆力的长短与钻洞潜伏期呈正比，与进入暗室的次数成反比。

# 3.4.4 物体识别法

物体识别试验应分为以下3个阶段：

a）适应期：应先将试验动物依次放入实验装置中，使其适应环境;b）熟悉期：应在实验装置中放置两个相同的玩具，并记录试验动物在每个玩具上的探索时间;c） 测试期：用一个大小相同而形状不同的玩具替换其中的一个玩具，并分别记录试验动物探索新旧玩具的时间，计算分辨指数；其中，分辨指数的计算方法为：分辨指数=（探索新玩具时间-探索旧玩具时间）／（探索新玩具时间+探索旧玩具时间）。

# 4结果判定

4.1若试验结果能满足以下条件，则应判定该受试品具有改善睡眠功能：

a) 至少采用一种镇静类型试验和一种催眠试验；  
b) 受试品组与对照组相比较，试验结果间具有显著性差异（有统计学意义）。

4.2若试验结果能满足以下条件，则应判定该受试品具有醒脑通窍功能：

a) 至少采用2种学习记忆试验;  
b) 受试品组与对照组相比较，试验结果间具有显著性差异（有统计学意义）。

# 5注意事项

在本试验中应注意：

a）试验动物应以大、小鼠为主;b）试验环境应安静、光线适当;c）在学习记忆试验中，为增加受试品的敏感性，可用M-胆碱受体阻断剂制备记忆缺损模型或选用β-淀粉样肽前体蛋白转基因小鼠为试验动物。
