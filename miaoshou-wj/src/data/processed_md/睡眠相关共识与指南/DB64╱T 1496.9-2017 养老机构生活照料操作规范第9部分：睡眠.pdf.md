# 养老机构生活照料服务规范 第9部分：睡眠照料

# 1范围

本标准规定了养老机构生活照料穿衣、修饰、口腔清洁、饮食照料、排泄护理、皮肤清洁、压疮预防、整理床单位、睡眠照料操作规范，本标准适应于本中心和对外开展的所有养老生活照料服务。

# 3 术语和定义

3.1生活照料服务依据老人能力评估状况，为入住老人提供程度不同的持续性生活照顾，以确保老人享有舒适、清  
洁、安全的日常生活。

3.2睡眠照料促使老人睡眠所进行的服务。

# 4服务规范

4.1创造睡眠条件，促进老人睡眠。保持老人室内温度适宜，环境安静，光线适宜；尽量使老人睡眠 保持在6～7小时，觉醒后感觉精力充沛，情绪愉快。

4.2征询老人需求，调节室内适宜温湿度。检查老人睡眠时，用温暖的手触摸老人。老人睡眠质量不好时，帮助采取适当的措施。夜间有适当的照明设施，开夜灯或地灯，以免光线过暗造成老人看不清周围景物而发生跌倒坠床等安全问题。卫生间坐便器设有扶手，对于行走不便老人，可以在床旁放置坐便椅，减少老人夜间起夜视物不清而跌倒。护理员夜间巡视时要轻手轻脚，停留时间适宜，不影响老人睡眠。检查老人床铺有无渣屑，保持被褥平整，枕头高低适宜。夜间温度下降，老人觉醒时，为老人增盖薄被。

4.3根据看护老人数量，安排适当的护理员。老人使用的床上物品经济耐用，常用物品放置合适位置，以免坠落造成经济损失。

4.4标准：老人居室环境安静，温、湿度适宜，正常睡眠；发现老人睡眠异常，能客观分析判断，并能正确处置。  
4.5布置睡眠环境约5分钟，夜间2小时巡视一次。

# 5 服务提供规范

# 5.1 服务准备

5.1.1 环境准备

环境清洁、安静，温湿度适宜（夏季26～30℃，冬季18～22℃，相对湿度50%～60%为宜），居室整洁，关闭门窗。

5.1.2 物品准备

记录单、笔、根据季节、气候准备棉被、床褥、毛毯等。

5.1.3 人员准备

护士（护理员）服装整洁，洗净双手，戴口罩、帽子。查阅既往睡眠照料记录，了解老人近期睡眠 状况。老年人排便、洗漱完毕，平卧床上。

# 5.2 服务实施

# 5.2.1 过程与方法

方法：

a）沟通护理员轻敲房门后进入房间，告知老人准备熄灯休息。询问老人房间温湿度是否合适，有无需要帮助的地方。

b) 布置睡眠环境，协助入睡

1）护理员协助关闭窗户，闭合窗帘。  
2) 调节室内空调或暖气开关，调整温、湿度。  
3) 检查老人床铺有无渣屑，按压床铺硬度。展开被褥平整铺床，被褥松软适中。整理枕头至  
蓬松，高度随老人习惯适当调整。  
4) 协助老人上床就寝，盖好盖被。询问是否还有需求，及时满足。  
5） 调节光线，开启地灯，关闭大灯。

c) 观察睡眠

1）护理员夜间2小时查房一次。做到走路轻，关门轻。观察老人睡眠状况，夜间入睡时间、觉醒时间及次数、总睡眠时间、睡眠质量等。  
2）异常睡眠状况：入睡困难、不能维持睡眠、昼夜颠倒现象、睡眠呼吸暂停、夜间阵发性呼吸困难、嗜睡等。3）记录异常睡眠。包括：床号、姓名、睡眠一般情况（入睡时间、觉醒时间及次数、总睡眠时间、睡眠质量）、老人主诉、异常睡眠的表现、有无采取助眠措施等。  
d）沟通晨起巡视并询问老人睡眠情况。  
e）记录交班本记录入睡时间、觉醒时间及次数、总睡眠时间、睡眠质量，如有异常睡眠则需记录老人主诉、异常睡眠的表现、有无采取助眠措施。

# 5.2.2 服务的沟通与确认要求

护理员了解老人睡眠习惯，操作过程中及时询问老人需求，给老人创造适宜的休息环境。

# 5.2.3 服务组织与实施要求

护士长组织护理员学习，了解老人的生理睡眠特点，掌握观察并记录老年人睡眠异常的技能；护士（护理员）掌握营造老人睡眠环境并做好睡眠准备的方法；护理员能为老人布置睡眠环境，能观察并记录老年人睡眠异常。

# 5.2.4服务提供信息的跟踪、反馈、分析、处理要求

护士长现场检查工作质量每周至少1次，根据老人休息时居室的声音、光线、温湿度、有无噪音及老人的主诉等，了解老人的睡眠环境及睡眠状况。对不符合标准的环节查找问题根源，提出改进方法，继续组织学习培训。

# 5.3质量控制

# 5.3.1人员、设施设备要素控制

老人睡前，卧室适当通风换气，避免空气浑浊或异味影响睡眠；枕头不易太高或太低，软硬度适中。被褥薄厚随季节调整；保持居住环境安静，夜间提供适当的照明设施，可避免老人情绪兴奋或焦虑，同时，防止因光线过暗看不清周围景物而发生跌倒或坠床等安全问题。

# 5.3.2 服务提供过程检查方法

夜间护理员要2小时查房一次。查房时注意走路轻，关门轻，避免惊醒老人；及时填写巡视单，记录内容详细，字迹清楚。

# 5.3.3 服务结果的评价与改进

老人居室环境安静，温、湿度适宜，正常睡眠；发现老人睡眠异常，能客观分析判断，并能正确处置。

# 5.3.4 服务记录的管理

记录内容完整（见生活照料记录单）。
