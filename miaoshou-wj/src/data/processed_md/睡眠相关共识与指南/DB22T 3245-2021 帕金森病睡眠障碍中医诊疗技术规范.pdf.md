# 3 术语和定义

下列术语和定义适用于本标准。

# 3.1

帕金森病 parkinson's disease

又称震颤麻痹,是发生于中老年人群的进展性神经系统变性疾病。其主要病理改变为以黑质部位为主的多巴胺能神经元的进行性丢失以及残存神经元内路易氏包涵体的形成。主要临床特征为静止性震颤、肌强直、运动迟缓和姿势反射障碍。

# 3.2

帕金森病睡眠障碍parkinson's disease with sleep disorders

睡眠障碍是帕金森病最常见的非运动症状之一，其症状多样，包括日间嗜睡、入睡困难和快速眼动期睡眠行为障碍。

# 4诊断

# 4. 1 中医证候

# 4.1. 1 气血两虚证

神呆懒言，多寐，面色胱白，肢体震颤或头摇日久，震颤程度较重，项背部僵直或肢体拘挛，活动减少，行走不稳，气短乏力，头晕眼花，自汗，动则尤甚，皮脂外溢或口角流延，舌体胖、边有齿痕，舌质黯淡，舌苔薄白或白腻，脉细无力或沉细。

# 4.1.2 瘀血内阻证

失眠日久，躁扰不宁，胸不任物，胸任重物，夜多惊梦，夜不能睡，夜寐不安，面色青黄，手足震颤日久，屈伸不利，或重着刺痛，舌紫黯或有瘀斑，舌下青筋怒起，脉弦涩。

# 4.1.3 肝肾阴虚证

失眠多梦，腰酸膝软，肢体麻木，或呆傻健忘，肢体或头部震颤日久，其幅度及程度均较重，或肢体拘挛，活动迟缓，上肢协调不能，动作笨拙，常兼见头目眩晕，耳鸣，舌体偏瘦，舌质黯红，少苔，脉细数。

# 4. 1.4 肾阳虚衰证

夜难入寐，甚则彻夜不眠，手足震颤或不利已日久，兼见面色胱白，畏寒肢冷，腰酸膝软，头昏耳鸣，小便清长，大便溏薄，舌白滑，脉沉而迟。

# 4.2 西医诊断

帕金森病睡眠障碍应同时满足下列两项：

a) 帕金森病:

1) 运动减少。启动随意运动的速度缓慢，疾病进展后，重复性动作的运动速度及幅度均降低。  
2) 肌肉僵直或静止性4～6Hz震颤。  
3) 姿势不稳，非原发性视觉、前庭、小脑及本体感受功能障碍造成。

b) 睡眠障碍：

1）凡乎以失眠为唯一的症状,包括难以入睡、睡眠不深、多梦、早醒,或醒后不易再睡、醒后  
不适感、疲乏，或白天困倦等。  
2) 具有失眠和极度关注失眠结果的优势观念。  
3) 至少每周发生3次，并至少己存在1个月。  
4) 除外躯体疾病或精种障碍症状导致的继发性失眠。

# 6针灸

# 6.1 针刺

6.1.1主穴神门、内关、百会、四神聪。

# 6. 1.2 配穴

6.1. 2. 1 气血两虚证。心俞、脾俞、三阴交。  
6.1.2. 2 瘀血内阻。肝俞、膈俞、血海。  
6.1.2.3 肝肾阴虚证。肝俞、肾俞、三阴交。  
6. 1.2.4 肾阳虚衰证。太溪、心俞、肾俞。

# 6.1.3 手法

平补平泻法。

# 6.1.4 1.4疗程

每次留针30min，每天1次。针刺1w为1个疗程，连续治疗4个疗程。

# 6.1.5 注意事项

皮肤感染、溃疡或肿瘤部位，不宜针刺，有出血倾向者，慎行针刺。

# 6.2 浅针

6.2.1 取穴印堂、太渊（双侧）、太溪（双侧）、大陵（双侧）。

# 6.2.2 手法

6.2.2.1主穴平补平泻法。兼有外感或胃肠紊乱者，加合谷（双侧）、足三里（双侧），用泻法。

6.2.2. 2 兼喘咳，加期门（双侧）、足三里（双侧）、列缺（双侧），用补法。  
6.2. 2.3 兼虚烦，惊悸者，加气海、三阴交（双侧），用补法。  
6.2.2.4 兼胁痛、易怒，加章门（双侧）、气冲（双侧），用泻法。

# 6.2.3 疗程

每日1次，10 次为一疗程，疗程间隔1w。

# 6.2.4 注意事项

皮肤感染、溃疡或肿瘤部位，不宜针刺，有出血倾向者，慎行针刺。

# 6.3 热敏灸

6.3.1热敏穴位以头面部、腰背部及小腿内侧为高发区，多出现在百会、至阳、心俞、脾俞、胆俞、  
三阴交等区域，选取上述2组～3组穴位。  
6.3.2治疗以灸至感传消失为度。  
6.3.3每天1次～2次，10次为1个疗程，共2个～3个疗程，疗程间休息2d～5d。  
6.3.4皮肤感染、溃疡或肿瘤部位，有出血倾向者，禁用灸法。

# 7 耳穴压豆

7.1取穴。主穴选取神门、心、脾、肾、皮质下，配穴选取枕、交感、内分泌、神经衰弱点。  
7.2治疗前先用耳穴探测棒在耳穴上寻找阳性点，用75%酒精消毒耳廓后将粘有王不留行籽的胶布对  
准选定的耳穴贴紧并加压，使患者有酸麻胀痛或发热感。  
7.3嘱患者每天自行按压2 次～3次，每次每穴 30s。上述治疗隔日进行1次，5 次为1个疗程。  
7.4皮肤感染、溃疡禁用。

# 8 穴位贴敷

8.1取穴。选取太阳（双侧）、神门（双侧）、涌泉（双侧）。  
8.2选用夜交藤 15g，白芷12g，败酱草10g，粉碎，加入白醋调制成丸状，置于胶布固定至穴位。  
8.3夜晚睡前贴敷，晨起移除。  
8.4皮肤感染、溃疡禁用。

# 9推拿

9.1双手拇指桡侧缘交替推印堂至神庭 30 次。  
9.2 双手拇指螺纹面分推攒竹至太阳30 次。  
9.3拇指螺纹面点按百会、角孙、四神聪各 30 次～50 次。  
9.4拇指螺纹面按太阳前后各转 15 次。  
9.5 拿捏风池 10 次。  
9.6 由前向后用五指拿头顶，至后头部改为三指拿，顺势从上向下拿捏项肌3次～5次。  
9.7 双手大鱼际从前额正中线抹向两侧，在太阳穴处按揉3次～5 次，再推向耳后并顺势向下颈部，  
做3次。  
9.8皮肤感染、溃疡、骨质疏松及有出血倾向者禁用。
