# 中医技术操作规范 皮肤科 中药药浴

# 1范围

本文件规定了中药药浴概述、局部药浴、全身药浴、注意事项和禁忌等内容。  
本文件适用于中药药浴技术操作。

# 2规范性引用文件

本文件没有规范性引用文件。

# 3术语和定义

下列术语与定义适用于本文件。

3.1

中药药浴 medicated bathof Chinesemateria medica

根据中医理论辨证论治选择方药，将药物制成中药药液或将药物粉碎后用纱布包裹，然后根据病情选择合适的温度稀释洗浴全身或局部的一种治疗方法。

3.2

# 中药药液 liquid medicine of Chinese materia medica

将药材饮片或粗末加水煎煮，去渣取汁，供使用的液体剂型。

# 4概述

4.1 中药药浴包括：局部药浴和全身药浴。  
4.2 局部药浴用于四肢末端及臀部等部位出现皮疹的患者。  
4.3 全身药浴用于全身泛发多部位皮疹的患者。

# 5 局部药浴

5.1器械与用品主要包括：中药药液、浴具(盆或桶)、一次性浴具套。

5.2局部药浴操作步骤如下：

核对姓名、药物，询问患者进食、既往史等情况；  
清洁浴具(盆或桶)，套上一次性浴具套；  
根据医嘱调配中药药液，加入温水至浴具(盆或桶)的1/2～2/3;根据不同疾病和皮疹情况选择水温，水温为35℃～42℃；  
局部药浴1次用时15min～20 min，每日1次～2次；  
局部药浴结束后，无需清水冲洗，擦干皮肤即可；  
倒掉中药药液，取下一次性浴具套，清洁浴具(盆或桶)。

# 6全身药浴

6.1 器械与用品主要包括：中药药液、浴桶或浴缸、一次性浴具套。

6.2全身药浴操作步骤如下：

保持室温约25℃，相对湿度60%～70%；保持良好的通风环境；  
核对患者姓名，询问患者进食、既往史，向患者讲解药浴方法；  
清洁浴桶或浴缸，套上一次性浴具套；  
根据医嘱要求调配中药药液。患者于浴桶或浴缸取坐位，水位不应高于双乳连线，根据不同疾病和皮疹选择水温，水温为37℃～42℃；  
全身药浴过程中可使用柔软毛巾轻拭患处，以清除局部分泌物、鳞屑及痂皮;  
全身药浴1次15min～20 min，每日1次～2次；  
全身药浴结束后，请患者起浴，动作不应过快过急。无需清水冲洗，擦干皮肤即可；  
倒掉中药药液，取下一次性浴具套，清洁浴桶或浴缸。

# 7注意事项

7.1不同的病症宜选择不同的水温度：

急性炎症性皮肤病，如湿疹、皮炎、玫瑰糠疹、多形红斑等可选择水温35℃～37℃；一 慢性炎症性皮肤病，如手足皸裂、寻常型银屑病等或自身免疫性大疱病可选择水温37°℃～39℃；硬化萎缩性皮肤病，如硬皮病、硬化萎缩性苔藓等可选择水温40℃～42℃。

7.2中药药浴一般应在饭后约1h后进行。中药药浴之前，避免剧烈运动，尤其是老年患者,避免活动过后中药药浴时突发心脑血管意外。

7.3护理人员应注意观察患者的情况,询问患者是否有异常感觉,预防随时可能发生的意外。如患者感到不适应立即中止中药药浴。

# 8禁忌

8.1患有心血管疾病、心肺功能不全及精神疾病患者不应进行中药药浴。  
8.2 对药浴液过敏者不应进行中药药浴。  
8.3 患有急性传染病、妊娠或妇女月经期，骨折后伤口及手术切口未愈者不应进行全身药浴。  
8.4空腹及饱餐后不应进行全身药浴。
