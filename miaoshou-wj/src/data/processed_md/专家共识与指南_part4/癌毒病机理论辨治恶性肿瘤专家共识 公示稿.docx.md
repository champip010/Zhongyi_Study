# 引言

恶性肿瘤是严重危害人类健康的多发病、常见病，逐年攀升的发病率和死亡率给社会和家庭带来了沉重的经济负担。中医药在减轻放化疗的毒副反应、提高患者免疫力、预防转移与复发、改善患者生存质量等方面的作用已受到广泛认可。近年来，随着癌毒病机理论的推广应用，从“癌毒”病机辨治恶性肿瘤成为中医肿瘤专家的广泛共识。癌毒病机理论由国医大师周仲瑛教授“癌毒”学说传承创新而来，在指导恶性肿瘤临床诊疗方面收效颇丰。为此，在中华中医药学会的指导下，我们研制了《癌毒病机理论辨治恶性肿瘤专家共识》，目的在于将周仲瑛教授辨治肿瘤的经验加以总结并推广，帮助临床医生在恶性肿瘤的中医药诊疗中做出合理决策，切实提高临床疗效，服务大众健康。

本《共识》创新点在于以国医大师学术思想及临床经验为理论基础，科学运用德尔菲研究法，制定出符合临床实际的中医临床诊疗共识，最终达到提高临床疗效的目的。本《共识》创新性的基础在于癌毒理论经由国医大师周仲瑛教授率先提出，历经数十载临床实践验证及大批理论追随者的不断阐释、发展，逐步形成了完善的中医肿瘤辨治体系，内涵理论丰富，源流清晰，在业内具有广泛影响力。

本《共识》的第二个创新点是采用以“病机”为核心的编制模式，与周仲瑛教授倡导的以病机为核心的辨证论治体系相适应，更加符合中医临床特点。癌毒病机理论认为各类肿瘤虽发病部位各异，但存在病机上的共性，核心病机为“痰瘀郁毒、正气亏虚”，治疗原则为“抗癌祛毒、扶正祛邪”，治疗上以抗癌祛毒为核心，扶正补虚为根本，化痰祛瘀为重点，具体应用化痰散结、化瘀软坚、化湿泄浊、清热解毒、祛风搜毒等诸法复合，并通过特异抗癌药物或引经药物加减实现对各类肿瘤的针对性治疗。

本《共识》主要由周仲瑛教授的学术思想和临床经验总结而来，应用领域主要集中在江苏地区，纳入文献多数来自南京中医药大学，可能存在潜在的利益冲突。

癌毒 (Cancerous Toxin)

癌毒是在脏腑功能失调、气血郁滞的基础上，受内外多种因素诱导而生成，是导致癌病的一类特异性致病因子。癌毒属毒邪之一，既是致病因素，也是病理产物。

4癌毒的特性[1-2]

(1）隐匿性：起病之初，深伏脏腑经遂，潜藏骨髓血脉，隐而难察，一旦显露则已难遏制；（2)凶顽性：癌毒致病，病势凶猛，症情乖异，正邪混处，难拘一格；（3）多变性：癌毒致病，转移、复发，走注弥散，传变无常；（4）损正性：随着病情的进展，毒恋正虚，损伤脏腑，耗竭气血，因病成损；（5）难消性：痰瘀郁毒互结，成为有形的实质肿块，根深蒂固，胶着难解。

# 5癌毒的分类[1-5]

5.1痰毒 (PhlegmToxin)

癌毒与痰互结形成的复合病邪。癌毒内蕴，津液输布不畅，聚而生痰。癌毒与痰胶结而成痰毒，可沿经脉、络脉流注，于至虚处形成癌肿，亦侵袭扩散形成转移病灶。

# 5.2瘀毒 (Stasis Toxin)

癌毒与瘀互结形成的复合病邪。癌毒盘踞，阻滞气机，血行不畅，停而为瘀，或癌毒耗伤正气，致气虚无力行血，血行迟缓，停而为瘀，癌毒与瘀血胶结而成瘀毒。

5.3 湿 (浊）毒 (Damp Toxin)

癌毒与湿互结形成的复合病邪。癌毒阻滞中焦，致脾胃运化失健，湿浊内生，性质黏滞、重浊，易形成下注之像，致病隐匿，缠綿难复。

5.4热 (火）毒 (Heat Toxin)

癌毒与热互结形成的复合病邪。热毒内蕴，灼津成痰，煎熬精血，壅阻经络、滞于脏腑，具有动血生风、耗气伤津等特点。

# 5.5寒毒 (Cold Toxin)

癌毒与寒互结形成的复合病邪。人体阳气虚衰，阴寒内盛，与癌毒互结，酿生寒毒，寒毒形成后又可进一步耗损阳气，导致阳气日衰，阴寒日盛，形成恶性循环，变证百出。

# 5.6风毒 (Wind Toxin)

癌毒与风互结形成的复合病邪。素体肝肾阴虚，虚风内动，癌毒附邪，阻滞经络、血脉，上扰清空，结聚脑腑。风性善行而数变，易挟癌毒流窜经络、脏腑。

# 6癌毒致病机制[1-5]

癌毒形成后常依附于风、寒、热（火）、痰、瘀、湿（浊）等相关非特异性病理因素杂合为病，即毒必附邪。毒因邪而异性，邪因毒而鴟张，以痰瘀为依附而成形，耗精血自养而增生，随体质、病邪、病位而从化，表现证类多端，终至邪毒损正，因病致虚。癌毒与痰瘀互为搏结而凝聚，在至虚之处留着而滋生，与相关脏腑亲和而增长、复发、转移。

# 6.1癌毒留结为肿瘤发病之基

肿瘤病理过程虽复杂，但总由癌毒留著某处为先。癌毒一旦留结，阻碍气机运行，津液不能正常输布则留结为痰，血液不能正常运行则停留为瘀，癌毒与痰、瘀搏结形成肿块，在至虚之处留着而滋生。

# 6.2癌毒自养为肿瘤生长之源

癌毒一旦形成，阻滞体内，则病变乖戾，狂夺精微以自养，逐渐形成有形之肿块，致使瘤体迅速生长，机体急速衰弱，诸症叠起。同时癌毒损伤脏腑功能，妨碍气血津液的正常运行，气血津液等精微物质不断地被转化成痰瘀等病理产物，促使肿瘤不断生长发展。

# 6.3癌毒流注为肿瘤转移之因

转移是肿瘤的一大特点，导致肿瘤转移的根本原因是癌毒的流窜走注。当肿瘤生长到一定阶段，癌毒随血脉流窜走注，并在他处停积，继续阻碍气机，酿生痰瘀，癌毒与痰、瘀搏结形成新的肿块，与相关脏腑亲和而转移，故肿瘤转移一般有其特定的脏腑。

# 6.4癌毒残留为肿瘤复发之根

肿瘤经治疗后，可能症状缓解、肿块缩小，甚至达到临床治愈的效果，但肿瘤又常易复发，这是影响治疗效果的关键问题。肿瘤经有效治疗，癌毒之势可能大减，但很难彻底根除，此时仍有少量癌毒伏藏于体内，若不加巩固，癌毒逐渐萌生，又可致肿瘤复发。

# 6.5癌毒伤正为肿瘤恶化之本

肿瘤形成后，癌毒耗伤气血阴阳，脏腑失于濡养，正气亏虚，更无力制约癌毒，癌毒愈强，又愈耗伤正气，如此反复，则癌毒与日俱增，机体愈益虚弱，终致毒猖正损、难以恢复之恶境。

7诊断[6]

局部有形之结，长势迅猛；或软，或硬，或坚硬如岩，留于体内；或附着体表，触之有形，推之不移；呈翻花样或蕈样；根据病理学资料检测判定为恶性肿瘤，具体参见《临床诊疗指南·肿瘤分册》的诊断标准。

8.1.1临床表现

9 康复调摄[3、5、11] (IV证据，推荐级别：E)

9.1既病之后，调畅情志，保持良好的情绪；  
9.2既病之后，饮食易于消化，适当补充营养；  
9.3既病之后，起居有节，保证充足睡眠;  
9.4既病之后，根据体力水平，适度体育锻炼。
