# 中华中医药学会团体标准“卵巢癌中西医结合诊疗指南”推广应用方案

# 一、该团体标准推广措施

1、由国家中医药管理局统一组织行业内的推广和贯彻工作;

2、举办指南应用推广培训班、继续教育培训班；

3、利用中华中医药学会肿瘤分会这一平台，在其开展的学术活动中介绍;

4、以手册形式发行、在相关杂志、网站发表。

# 二、该团体标准推广的预期目标

1、本指南旨在规范卵巢癌的中西医结合的诊断、治疗，为临床医师提供中西医结合的标准化处理策略与方法,全面提高中西医结合治疗卵巢癌的临床疗效和科研水平，促进与国际学术发展接轨，同时减轻国家医疗保险的负担，提高社会经济效益。

2、宣贯场次及规模：线下线上宣讲4-6次/年，50人/场次。

# 三、该团体标准的推广机制

1、靶向本标准的使用人群，在学术会议、学术网站中发布，提高曝光率

2、通过国家中医药管理局统一组织行业内的推广和贯彻工作、举办培训班等举措提高社会对本标准的认知度与认可度。

# 四、该团体标准推广的保障措施

1、本指南在19家医院实施，囊括了中医医院、中西医结合医院、综合医院、专科医院、基层医院等多层次、多地域。

2、全国20余位执笔人均是卵巢癌领域的专家，具有良好的学术影响力和广泛的患者群体。

3、团队具有举办学术会议的经验，能够保障本指南的推广。

五、其它应予说明的事项无。
