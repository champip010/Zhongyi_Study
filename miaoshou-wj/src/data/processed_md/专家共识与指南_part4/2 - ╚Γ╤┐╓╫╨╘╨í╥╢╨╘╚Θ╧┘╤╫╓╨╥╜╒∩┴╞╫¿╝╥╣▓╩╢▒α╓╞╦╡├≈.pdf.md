# 肉芽肿性小叶性乳腺炎中医诊疗专家共识编制说明

# 一、工作简况

# （一）任务背景

肉芽肿性小叶性乳腺炎（Granulomatouos Lobular Mastitis,GLM)，又称乳腺肉芽肿性炎、特发性肉芽肿性乳腺炎等,是一种以乳腺小叶为中心，非干酪样坏死性肉芽肿为主要病理特征的慢性炎症性疾病。临床多表现为外周起病的疼痛性肿块，进展迅速，脓成后形成复杂性窦道，溃后经久不愈，部分患者可伴见肢体结节性红斑、发热等全身症状。国外临床流行病学调查显示，GLM约占良性乳腺疾病的1.8%，地中海国家、西班牙裔及亚裔女性更易患此病，土耳其的调查结果显示慢性肉芽肿性炎症占所有乳腺炎症的24%。国内虽无确切的流行病学统计资料，但近年来该病的报道日益增多，总体上看，GLM患者呈明显增多态势。中医药治疗肉芽肿性小叶性乳腺炎的方法多样，临床疗效确切。但目前国际上尚无中医药治疗肉芽肿性小叶性乳腺炎的临床实践共识。

《肉芽肿性小叶性乳腺炎中医诊疗专家共识》项目组在充分搜集和整理古今中医专家经验的基础上，整合和吸纳国内外中医药防治肉芽肿性小叶性乳腺炎的研究成果，征求行业内著名专家意见，借鉴临床流行病学的研究方法，按照标准编写规则，完成了起草阶段工作，形成了《肉芽肿性小叶性乳腺炎中医诊疗专家共识》的初稿，经专家共识会议，进一步修改初稿，形成《肉芽肿性小叶性乳腺炎中医诊疗专家共识·送审稿》，内容涉及肉芽肿性小叶性乳腺炎中医诊断、临床分期分型、病因病机、辨证论治方法、预防与调摄，并统一了本病的临床诊断参考标准及疗效评价标准。共识的制定对全面系统地认识肉芽肿性小叶性乳腺炎，提高中医药防治肉芽肿性小叶性乳腺炎的临床疗效具有重要的作用，同时也为中医临床研究开展提供了便利。

# （三）确定共识的题目和范围

通过循证查询，目前尚无关于肉芽肿性小叶性乳腺炎（GLM）中医诊疗的共识或指南。结合前期文献调研，发现中医药治疗肉芽肿性小叶性乳腺炎临床疗效肯定，但未形成临床共识。因此，亟需构建完善的中医药诊治GLM的标准体系，进一步规范中医药治疗肉芽肿性小叶性乳腺炎的诊治行为，并将该标准体系运用于中医院、中西医结合医院、综合医院，用于GLM的中医、中西医结合临床诊疗。

# 1.3检索结果

# 1.3.1古籍文献数据库检索：

1)一般数量统计：

“乳痈”共检索条文：2123条；“乳漏”共检索条文：20条。经仔细研读、严格执行纳入与排除标准，共得到“乳痈”106 条条文，“乳漏”2条条文。

2)文献分析结果：

治疗类型：治疗方法可分为单纯内治、单纯外治、内外合治、针刺、艾灸几种，以单纯内治、单纯外治及内外合治为主，条文分别为59条、16条、27条，其中有复方66个、单方/验方29 个。

证候类型：肉芽肿性乳腺炎多归属于“乳痈”范畴，临床上注重局部与全身辨证相结合，根据局部表现多分为初期、成脓期、溃后期三期。基于现有资料，可明确分期辨证的70条，分别为初期27条、成脓期32条、溃后期11条，部分方药稍做调整后可多期共用或贯穿疾病治疗始终，如消乳汤、内托千金散，神效瓜萎散等。归纳发现总以肝郁、热毒等证型多见。

内治方药：基于现有资料，复方与单方/验方共计内服方药有70个，其中有16个方药既可内服又可外用。70个内服方药共涉及107味中药，频率超过5次的中药有 29 味，分别为甘草（35）、当归（26）、金银花（19）、白芷（18）、穿山甲（17）、瓜萎（17）、青皮（16）、乳香（16）、白芷（15）、黄芪（15）、没药（14）、人参（13）、连翘（12）、白芍（10）、川芎（10）、贝母（10）、黄芩（9）、皂角刺（9）、蒲公英（9）、柴胡（7）、防风（7）、茯苓（7）、陈皮（橘皮）（6）、天花粉（6）、木香（6）、大黄（6）、橘叶（5）、桔梗（5）、升麻（5)，以清热解毒、通络散结、理气化痰、活血化瘀、益气合营等中药为主。

外治法：共有外用方药40个，其中有16个方药既可外用又可内服，外治方药中又以单方/验方为主，共有19个，制药及临床应用更便利。40个外用方药，根据疮疡“阴阳”辨证法，明确用于归属于阳证者有16个、阴证者有8个、半阴半阳者有4个。

临床选用：共收集符合要求的古籍文献108条，记载可用于肉芽肿性乳腺炎的治法有100 种。根据条文是否明确描述适应症、方药组成及剂量，及其在当前医疗环境下临床应用的安全性及可行性，共筛选出可选用的治法有65种，其中单纯内治法有36 种、单纯外治法16 种、内外合治12种、针刺法1种。

# 1.4访谈结论

2）根据病程进行临床分期对GLM诊疗具有意义，可其分为急性期、亚急性期、迁延期。

3）GLM的中医病名方面，过半专家认可肉芽肿性乳痈。

4）GLM的发病因素方面，专家认可应包含的主要因素。大部分专家认为本病多因回乳后乳汁残留，或异常乳汁分泌稽留体内，当机体过度劳累、或饮食不节、或情志不畅、或外伤碰撞等因素诱发。发病机制可能为：异物郁积，阻滞乳络，气血运行不畅，痰瘀交阻，则见局部肿痛；郁久化热，热盛肉腐则见乳房脓肿；日久耗气伤津，无力托毒生肌，则脓肿破溃，脓水淋漓，迁延难愈。

5）治疗方面，对GLM值得推荐的有效治疗方法有小切口引流、脓肿穿刺抽液、中医祛腐清创、中药辩证治疗、中药贴敷等。对于GLM，大部分专家认同“消、托、补”总的治疗原则。在消法中，可有清消、温消法；针对托、补法，可进一步选择代表方。不同的分期，则GLM的外治法侧重有所不同。

6）GLM评价方面，就GLM疗效评价应考虑的维度和指标进行讨论，专家认为应该多维度评价，GLM症状评分量表有助于疗效评价。

7）专家基本认可GLM的痊愈、临床治愈、复发等定义。

# 2.5统计分析方法

采用Delphi法对专家咨询问卷结果进行统计分析，即使用专家意见集中程度指标、专家意见协调程度指标对问卷中各项指标进行统计分析。

1）专家积极系数：即专家调查问卷的回收率，其大小说明专家对该项目研究的关心、了解程度。积极系数越高，提示专家对本项研究关心合作的程度越高，则研究结果的可信度越高。

2）专家基本情况：专家高职称比例越高、从事本专科专业时间越长、年收治病人数越多，提示所选专家有高的学科代表性及专业的权威性，从而反映研究结果的高可信度。

3）专家意见集中程度：专家对各指标相对重要性的意见集中程度。采用均数（X)、等级和(S)及不重要百分比(R)进行评价。X、S的分值越大，提示信息在相应部分中的地位越重要，专家的意见越集中；R值越大则提示该条目在相应部分中的不必要性越大，可作为条目删除的依据。

4）专家意见协调程度：专家意见的协调程度判断的是参与研究的专家对每项指标的评价是否存在较大分歧，或找出高度协调专家组和持异端意见的专家。采用变异系数(CV)进行评价。CV值越小，则提示专家对该条目重要性评价的一致性越高，专家协调程度越高。变异系数仅能说明m个专家对i指标意见的协调程度，但在研究中，往往还希望了解全部所有专家对全部指标意见的协调程度，用肯德尔和谐系数(Kendall'sW）表示。Kendall'sW值在0—1之间，如果所有专家对全部指标评价意见完全一致，则 Kendal1'sW为1，在意见极度相反的情况下，KcndalrsW为0。

5）专家意见权威程度：专家的权威程度一般由两个因素决定，一个是专家对方案作出判断的依据，一个是专家对问题的熟悉程度。专家的权威程度以自我评价为主。本研究专家权威程度的判断采用自我评价，因两轮专家基本相同，且专家做出判断的依据基本一致，故仅行一轮专家进行统计即可说明问题。

①判断依据：用Cα表示判断影响程度系数。专家一般以“临床经验”、“理论分析”、“对国内外同行的了解”及“直觉”等作为判断依据，判断系数Cα≤1。当Cα=1时，判断依据对专家的影响程度很大;当Cα=0.8时，对专家判断的影响程度中等；当Cα=0.6时，影响程度较小。

②专家对问题的熟悉程度：用Cs表示专家对问题熟悉程度系数。

③专家的权威程度：用Cr表示专家权威程度系数，其为判断系数和熟悉程度系数的算术平均值，即Cr=（Cα +Cs）/2。

# 2.6筛选原则

1）列入项目的原则：一轮答卷统计结果达到以下要求的指标列入下轮问卷：均数（X)≥1，且等级和(S)≥满分的50%（“满分”为该指标最高得分2分乘以答卷份数）。

2)删除项目的原则：一轮答卷统计结果为以下情况的指标原则上不列入下轮问卷：均数（X)≤0．6，或等级和(S)≤满分的30%，或不重要百分比(R)≥50%。3)需要研讨的项目：一轮答卷统计结果为以下情况的列为“需要研讨的项目”：均数介于0.6-1，等级和介于30%～50%。

指标筛选参考以上原则执行，同时结合文献研究报告、专家意见，经课题工作组评议后进行筛选。

# 2.7专家咨询统计结果

1)问卷回收情况：在全国范围内共发出两轮专家咨询问卷，回收地区包括广东、湖北、天津、河南、北京、安徽、上海、江苏、浙江、黑龙江、山东、广西、陕西13个省市。第一轮以邮件和微信形式共发出54份问卷，共收到回复44份，问卷回收率（专家积极系数）为81.5%；第二轮以微信形式对第一轮回复专家发出第二轮专家咨询，共发出44份问卷，回收43份，专家积极系数为97.7%。

2)专家基本情况：所有专家均是副高以上职称，所有专家从事乳腺专业工作均超过十年，最长者达52年；61.37%的专家年收治GLM患者超过50例。

# 四、与国内外同类指南/共识的对比和最新指南/共识采用情况

关于肉芽肿性小叶性乳腺炎目前尚无全国性的诊疗共识或指南，由中华预防医学会妇女保健分会乳腺保健与乳腺疾病防治学组牵头形成的《非哺乳期乳腺炎诊治专家共识》部分涉及了肉芽肿性小叶性乳腺炎内容，但内容过于简单且未涉及本病的中医病因病机及诊治。由湖南省相关专家制定的《肉芽肿性小叶性乳腺炎诊治湖南专家共识》以西医内容为主，少量涉及中医，且参与编写专家仅为湖南省部分专家，覆盖区域有限。

本共识与目前其他共识的区别：

1、覆盖范围广，代表性强：本共识覆盖全国各地17家医院，以三甲医院  
为主，纳入了全国中医/中西医治疗肉芽肿性小叶性乳腺炎具有代表性影响力  
的医院，能相对全面代表目前全国GLM的中医诊治情况。2、共识组专家影响力大：本共识由25位在中医/中西医乳腺病领域具有  
显著影响力的专家共同编撰而成，能够很好地反映目前对GLM的认识和中医诊  
疗情况。3、是目前唯一的中医诊疗专家共识：本共识创新性地制定了中医诊疗的  
共识，内容涉及中医及现代病因病机、诊断标准、辨证论治等，规范了本病的

分期分型、中医辨证论治及外治法等。

4、科学性、严谨性：本共识组在充分搜集和整理古今中医专家经验的基础上，整合和吸纳国内外中医药防治肉芽肿性小叶性乳腺炎的研究成果，征求行业内著名专家意见，借鉴临床流行病学的研究方法，按照标准编写规则来编撰本共识。

5、针对临床难点，创新性提出解决方法：GLM的疗效评价标准一直存在争议，临床评价方法多样，但各有不足之处，难以较全面体现该病的病情变化特点及疗效评价情况，本共识组在前期工作基础上，创新性提出肉芽肿性小叶性乳腺炎疾病活动指数（GMDAI）来评估疾病的活动性以及对治疗的反应，极大地利于临床诊疗的开展。

6、针对目前GLM诊断情况，补充性提出临床诊断要点：本共识提出若未见典型GLM病理特征，诊断存在困难，可以根据特征性临床表现，建立GLM的临床诊断参考标准，以指导临床诊治。这就为临床医生提供可以借鉴的临床参考标准。

7、内容全面又简洁，实操性强：本共识切合临床诊疗实际来制定，共识推荐清晰，方便推广及临床医生使用。

《共识》修订内容：

GLM的诊疗评价标准临床上尚有争议，无相对公认的标准，大部分仍处于个人经验和体会的水平。我们根据疾病特点，综合专家意见及文献结果，试图从肿块、疼痛、伤口创面、复发多个维度进行赋值评价以细化疗效评价标准。但观察3月来，临床结果并不满意。如肿块大小，因疾病进程不一，治疗过程中，可能此处渐愈彼处又起；疼痛也呈阶段性表现；创面修复与疾病恢复不呈正比。因此，根据目前的研究状况，给出一个更加细化、具体化、且公认的疗效标准尚有一定难度，有待下一步对GLM的疗效评价方法及客观化行更深入的研究。

此外，因GLM有一定自限性，较难用治愈、有效、无效的三分法进行评价。患者治疗后临床症状缓解，但可能遗留无症状的影像学下的稳定病灶，而这部分病灶可能会被机体自行吸收，并不需太多干预。因此，共识中借鉴结核病疗效评价中临床治愈、痊愈的概念，进行定义。

本次修改中，对疗效评价内容重新进行了组织编排，根据GLM临床治疗的目的主要在于缩短病程，尽量保证乳房的外形和功能，因此，在定义治愈与复发的基础上，将病程与乳房外形做为主要的评价指标。
