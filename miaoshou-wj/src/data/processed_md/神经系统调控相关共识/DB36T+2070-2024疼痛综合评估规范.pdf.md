# 疼痛综合评估规范

# 1范围

本文件规定了医疗机构不同疼痛类型（急性疼痛、慢性非癌性疼痛、癌性疼痛）的术语和定义、缩略语、评估基本要求、评估宣教、评估方法、评估相关记录单、评估流程、质量控制等内容。

本文件适用于各级、各类医疗机构开展疼痛综合评估工作。

# 2规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

T/CNAS01 成人癌性疼痛护理

# 3术语和定义

下列术语和定义适用于本文件。

3.1

疼痛pain一种与实际或潜在的组织损伤相关的不愉快的感觉和情绪情感体验，或与此相似的经历。

3.2疼痛评分量表pain scale用来帮助医护人员衡量和量化个体疼痛强度的工具。

# 3.3

疼痛评估 pain assessment

在疼痛诊疗全过程中，利用一定的方法测定患者的疼痛强度、类型、性质、部位等信息，为临床评判病情、制订治疗方案提供科学依据。

# 4缩略语

下列缩略语适用于本文件。  
VAS：视觉模拟评分量表（Visual Analogue Scale）。  
NRS：数字评分量表（Numeric Rating Scales）。  
VDS：文字描述评分量表（VerbalDescriptor Scale）。  
VRS：语言评分量表（VerbalRatingScale）。

Wong-Baker 面部表情疼痛量表（The Wong-Baker FACES Rating Scale）。N-PASS：中文版新生儿疼痛、躁动及镇静评估量表（Neonatal Pain，Agitation andSedation Scale)FLACC：行为学评估量表（Face，Legs，Activity，Cry，Consolability Observation Tool）。CPOT：危重症患者疼痛观察工具（Critical care Pain Observation Tool）。HADS：医院焦虑抑郁量表（Hospital Anxiety and Depression Scale）。

# 5疼痛综合评估基本要求

5.1疼痛综合评估的执行者应掌握相关专业知识及技能。  
5.2 疼痛综合评估的原则：相信患者的主诉，常规、全面、量化和动态的评估。  
5.3疼痛综合评估工具的选择：测量疼痛强度的疼痛评分量表及建立含疼痛综合评估的护理记录单等工具。

# 6疼痛综合评估宣教

6.1医护人员应指导患者及家属正确评估疼痛。

6.2目的及意义：疼痛综合评估是疼痛治疗的基础，准确、及时地评估为临床提供第一手资料，便于在疼痛诊疗过程中根据患者的疼痛状态调整治疗和护理方案，有助于提高镇痛效果。

# 7疼痛综合评估方法

# 7.1 疼痛筛查

医护人员应根据患者在某时间段的主诉进行疼痛筛查，判断有无疼痛。

# 7.2疼痛评分量表选择

7.2.1神志清楚能正常交流的3岁以上患儿或成年患者选择VAS或NRS 结合VDS与VRS(参见附录A.1、A.2、A3、A4）。  
7.2.2神志清楚有先天性认知障碍、语言表达困难的患者（如老年、老年痴呆、聋哑患者）或3岁以上不能交流的患儿采用Wong-Baker 面部表情疼痛量表（参见附录A.5）。  
7.2.3对3岁及以下患儿、意识障碍无法配合完成评估的患者可使用行为学评估量表FLACC（参见附录A.6）。  
7.2.4对新生儿疼痛评估可采用中文版新生儿疼痛、躁动及镇静评估量表N-PASS（参见附录A.7）。7.2.5对使用呼吸机、气管插管、气管切开等机械通气而无法有效沟通的患者可使用危重症患者疼痛观察工具CPOT（参见附录A.8）。

# 7.3评估内容

# 7.3.1. 能自我表述疼痛者

7.3.1.1疼痛综合评估内容：疼痛的部位、性质、发生与持续时间、有无伴随症状、诱发因素/缓解因素和心理、精神状态，运用疼痛评分量表测量疼痛强度。  
7.3.1.2疼痛部位：患者应主诉躯体上疼痛的确切部位，包括牵涉痛、放射痛部位，以及部位不确切的疼痛。

7.3.1.3疼痛性质：医护人员对疼痛性质进行评估，常见疼痛包括钝痛、隐痛、胀痛、抽搐痛、麻痛、电击样痛、痉挛性痛、绞痛、刀割样痛、针刺痛、烧灼痛、性质不确切的疼痛等。

7.3.1.4疼痛强度：评估患者某时间段内最痛的分值，根据世界卫生组织将疼痛分为四级三个程度。0级，无痛。—1级（1分-3分，轻度疼痛），有疼痛，可忍受，生活正常，睡眠不受影响。—2级（4分-6分，中度疼痛），疼痛明显，影响睡眠，难以入睡，半夜会痛醒。——3级（7分-10分，重度疼痛），疼痛剧烈，不能忍受，严重影响睡眠，完全无法入睡。—了解患者疼痛发生与持续时间、有无伴随症状、诱发因素/缓解因素。

7.3.1.5 心理评估：慢性疼痛患者宜采用 HADS 量表评估心理及精神状态（参见附录B.1）。

# 7.3.2无法自我表述疼痛者

运用疼痛评分量表评估疼痛强度，疼痛发生与持续时间、有无伴随症状、诱发因素/缓解因素。

# 7.4疼痛综合评估与记录频率

7.4.1无痛患者或疼痛强度1-3分患者：每日应评估记录1次，时间宜为2PM，评估患者前日 2PM一当日2PM期间的疼痛情况。  
7.4.2疼痛强度4-6分患者：每日应评估记录3次，时间宜为6AM、2PM、10PM，分别评估患者前日10PM-当日6AM，当日6AM-当日2PM、当日2PM-当日10PM期间的疼痛情况。  
7.4.3 疼痛强度≥4分（CPOT量表评分≥3分）、疼痛性质、疼痛部位有变化或有新伴随症状者应通知医生处理。  
7.4.4 爆发痛、疼痛强度≥7分（除产妇外）：遵医嘱静脉或鞘内给药后15 分钟、皮下注射或肌内注射后 30 分钟、口服给药/直肠给药或非药物干预后60分钟应再次疼痛综合评估并记录。

7.4.5疼痛评分≥4分患者，出院时进行疼痛评估与记录。

7.4.6门、急诊患者疼痛综合评估包括以下两种情况：一医护人员对疼痛患者进行疼痛综合评估并记录；—门急诊无痛人流、胃肠镜、纤维支气管镜等诊疗操作时，由麻醉医师在麻醉前、麻醉后进行疼痛筛查、评估并记录。

7.4.7手术患者术后疼痛综合评估（排斥医嘱术后）：

-麻醉苏醒室：术后患者在苏醒室清醒时、出苏醒室前与其它生命体征同步进行疼痛筛查、评估并记录；疼痛强度≥4分患者，遵医嘱药物干预，30分钟后再次评估记录；  
—术后返回病房时与血压、脉搏、呼吸同时评估一次；手术当日、术后第一天、术后第二天 6AM、2PM、10PM评估，连续评估3天；术后第三天后按术前疼痛综合评估的原则动态评估。

7.4.8产妇疼痛综合评估（可分为产前、产时、产后三阶段）：

——产前评估：产妇入院后至进产房前由病房护士按外科手术疼痛评估规范评估记录，如建立了分娩记录单则与专科评估项目同步进行疼痛评估记录;  
产时评估：产妇进入产房后与分娩记录单中专科评估项目同步进行疼痛评估记录;  
—产后评估：产后即时与分娩记录单中专科评估项目同步进行疼痛评估记录一次，返回病房由病房护士按外科手术后疼痛评估规范评估记录。

# 8疼痛综合评估相关记录单

8.1建立含疼痛综合评估的护理记录单及医疗记录单。  
8.2医护记录的疼痛部位、疼痛性质、疼痛评分量表应保持一致，护理记录的疼痛强度应在医疗疼痛

程度记录范围内。
