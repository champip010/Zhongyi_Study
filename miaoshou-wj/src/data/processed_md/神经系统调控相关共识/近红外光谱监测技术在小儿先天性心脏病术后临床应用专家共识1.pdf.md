【摘要】先天性心脏病(CHD)术后良好的脏器保护,尤其是脑、肾、胃肠等重要脏器的保护,不仅能提升生存率,而且还能极大的改善预后。近红外光监测技术(NIRS)可以实时无创多部位监测,定量提供重要脏器局部氧合的指标，有效地预警和指导CHD术后治疗和脏器保护，从而改善CHD患儿术后的转归。本专家共识对NIRS的工作原理,使用人群,使用方法学,在小儿CHD中的应用,包括脑部区域氧饱和度(rSO2)监测、腹部rSO监测、肾区rSO监测及体外膜肺氧合(ECMO)期间rSO监测等内容进行总结，旨在规范和指导NIRS的临床应用。

This expert consensus summarizes the working principle,population, methodology,and application of NIRS in pediatric CHD, including brain region oxygen saturation (rSO2） monitoring, abdominal rSO2monitoring，renal rSOmonitoring and also application in extracorporeal membrane oxygenation (ECMO),etc., with the aim to standardize and guide the clinical application of NIRS.

Jobsis于1977年首次将近红外光谱监测技术(near-infrared spectroscopy,NIRS)应用于脑氧饱和度的监测。在之后十年，随着光学仪器的技术进步和光在组织中传播技术的改进，NIRS开始在人体中应用[2-3],显示实时、连续的脑氧饱和度和躯体组织氧饱和度[4]。小儿先天性心脏病(简称先心病)术中NIRS监测是早期应用这项技术的领域之二[5-6]。神经系统并发症是先心病术后的严重并发症之一，尽管其病因受多因素影响，但是可以通过改善围手术期神经系统监测和优化脑氧供给进行预防[7。相比其他的监测手段而言，NIRS有如下优势：无创、便携、可连续测量。随着多通道NIRS监测在临床的应用，越来越多的心脏中心将多通道NIRS监测应用于围术期脏器的保护。本专家共识将结合国内外的多项临床研究和使用指南，制订NIRS在小儿先天性心脏病术后临床应用专家共识，以期指导NIRS监测在小儿先心病术后的规范应用。

# 一、近红外光谱监测的工作原理

NIRS是一种无创连续监测区域氧饱和度(regionaloxygensaturation,rSO)技术，可以在床旁监测组织的灌注和氧合状态。NIRS主要测量局部组织的rSO，其中静脉血占75%，动脉血占20%，毛细血管占5%[8]。与可见光或紫外光相比，近红外光（700\~1000nm波长)可以更好地穿透生物组织。氧合血红蛋白和脱氧血红蛋白之间的吸收光谱不同。与脱氧血红蛋白相比，氧合血红蛋白可以吸收更多的红外光,但吸收的红光更少。NIRS设备通过发光二极管发出两种不同波长（730和810nm）的光，光子可以通过组织，不同氧合状态的组织氧合和脱氧血红蛋白比例不同从而呈现不同的吸收光谱。未被吸收的部分光子由两个检测器接收。近端的探测器接收来自浅表组织的信号，而远端的探测器同时接收来自浅表组织和深部组织的信号。

远端值减去近端值的计算结果是组织1\~2cm深度处的氧饱和度，提示下层组织的rSO。rSO的计算公式如下：rSO=[氧合血红蛋白/氧合血红蛋白+脱氧血红蛋白）」。rSO下降提示血流量减少，导致运输到下层组织的氧气减少或下层组织的氧摄取增加。NIRS可以在不可逆的细胞或组织损伤发生之前发现组织灌注不足，因此，NIRS监测可以早期提示组织氧气的运输和消耗，即氧供需间的平衡，对危重症患儿及时干预并迅速了解干预措施的即时效果。

推荐意见1：NIRS监测通过对组织形成不同的吸收光谱，可以早期发现病情、及时干预，并从持续监测中观察病情发展趋势(推荐级别：I类，证据水平：C)

目前，有多种NIRS监测设备可以用于儿童的rSO监测[],包括INVOS5100（Somanetics,美国密歇根州特洛伊），ForeSight（CASMedical,美国康涅狄格州布兰福德），SenSmartX10O(NoninMedical，美国明尼苏达州普利茅斯)NIRO200NX(滨松光电公司，日本滨松)以及国内的EGOS-600B(苏州爱琴生物医疗电子有限公司）。目前，INVOS5100系统以其多通道监测和适用人群范围广优势日益突出。

# 二、NIRS监测技术使用人群

推荐意见1：NIRS监测技术适用于>2.5kg的患儿，<2.5kg的患儿可以观察变化趋势，无标准参考值(推荐级别：I类，证据水平：C)

推荐意见2：电极片黏贴位置的皮肤无缺损和炎症，新生儿电极片黏贴时去除胎脂使电极片贴合。

推荐意见3：腹部NIRS监测时注意做好肠道准备，定期通便避免肠胀气及粪块积在肠腔内。

推荐意见4：电极片使用：

推荐意见4.1配套的电极片如标注了一次性使用，应遵循一次性使用的原则，电极片的重复使用可能导致读数不准确、读数不稳定或无读数。重复使用可导致交叉感染风险的增加(推荐级别：Ⅲ类，证据水平：C)

推荐意见4.2配套的电极片如标注了可重复使用，应按照使用说明进行电极片的消毒。

推荐意见4.3根据电极片的说明，不同年龄段或不同体重的患儿应选用推荐型号的电极片。

早产儿、新生儿、小儿(含婴儿)的头部前额、肾部、腹部或四肢，在临床中用于监测氧饱和度并提供趋势监测。头部电极片黏贴位置为眉弓上1cm左右，以正中线为中心左右对称。腹部电极片一般黏贴在脐上或脐下，部分临床研究放在肝区。肾区的电极片放在脊柱左侧或右侧，肋弓下方的部位，肢体的电极片放在小腿的腹侧或足背。一般来说，电极片黏贴处无明显压疮和皮疹，极少见过敏性皮炎等副作用。

# 三、NIRS监测使用方法学

NIRS监测的步骤如下：首先打开NIRS机器或将嵌入式模块放入监护仪中，根据患儿体重选择合适的电极片，黏贴电极片，待数据稳定后设置基线，调整报警范围，根据需要保存患儿的一般信息，以便后续的储存和读取。NIRS读数仅代表传感器下的小部分组织，不能反映其他部位的氧合紊乱。保证所有连接器都充分连接并避免受潮。潮气侵入可能导致读数不准确，读数不稳定或无读数。放置或取出传感器时请谨慎小心。请勿将传感器放置在破损或未愈合的皮肤上。对于脑内放置场所的选择，请勿将传感器放置在痣，鼻窦腔，上矢状窦，硬膜下或硬膜外血肿处或其他异常部位如动静脉畸形处等，这可能导致读数不能反映脑组织状况或无读数。对于躯干位置选择，请勿将传感器放置在厚脂肪沉淀，毛发或骨突起部位。请勿放置在痣，血肿或破损皮肤处，这可能导致读数不能反映体细胞状态或无读数。进行X线或CT扫描时无需移除传感器。然而，传感器将出现在图像中。磁共振成像（magnetic resonance imaging,MRI)时需要移除传感器。确保报警限值适合于每一名患者。

四、NIRS监测技术在小儿先心病中的应用

# 1.脑部rSO监测

推荐意见1：脑损伤高危患儿可行左右脑NIRS监测，结合脑电图双频指数、动态脑电图和经颅多普勒了解脑损伤的情况(推荐级别：I类，证据水平：C)

推荐意见2：先心病术后低心输出量的患儿，建议脑部NIRS监测，观察其绝对值和趋势值，同时结合混合静脉血氧饱和度(推荐级别：I类，证据水平：C)

推荐意见3：脑部rSO的正常范围为60%\~80%(循证医学证据：I类，证据水平：C)

大脑的新陈代谢活动及对氧气的需求高于其他器官，脑rSO通常低于腹部或肾脏rSO。脑rSO的标准因不同的研究而异。据McCormick等人（INVOS2901)报道为57%\~77%,Lemmers等人(INVOS4100)[10]报道为62%\~78%，Petrova(INVOS5100)["1]报道为56%\~76%,McNeill等人(INVOS5100)[12]报道为66%\~83%。通常，新生儿脑rSO标准值是60%\~80%。国内一项多中心脑氧监测（TSAH-100型，安徽安恒光电)的观察性研究指出，正常足月新生儿rSO测定值为(62±2)%，低于58%提示脑组织缺氧，值得注意的是，3例重度缺氧缺血性脑损伤急性期，脑rSO有异常增高现象[13]。

脑rSO基线值受动静脉畸形，上矢状窦或硬膜外出血，额外的颅骨结构所引起的血流变化的影响[14-18]。因此,标注脑rSO的基线值很重要。在获得脑rSO基线值之后，我们需要关注其变化趋势。

NIRS最常用于新生儿心脏手术的围术期监测脑氧合以防止缺氧性损伤或休克。由于体外循环的心肺旁路会导致脑血流和氧合的突然变化，因此，我们常常在先心病的手术过程利用NIRS技术监测脑rSO是否下降，以预防缺氧性损伤。Hansen等人[19]的报告显示，心脏手术后大脑rSO降低可能与脑循环到躯干循环的血流重新分配有关。Phelps等人[20]的研究表明，在术后最初的48h内NIRS检测到脑rSO<56%与不良预后相关，例如死亡或需要体外膜肺氧合（extracorporealmembraneoxygentation,ECMO)支持或较长的重症监护滞留时间。因此，我们强烈建议监测脑rSO以迅速识别有不良预后风险的患者。同时，通过监测情况对高危患儿及时改善脑氧合情况可以预防不良事件发生。

先心病术后患儿的脑损伤发生率为2%\~25%[2，主要的危险因素包括低灌注、低心输出量和缺氧。为了实现婴儿复杂手术区域视野清晰，深低温停循环（deep hypothermia circulatoryarrest,DHCA)常被采用，其次包括选择性脑灌和逆行脑灌。然而，小儿DHCA超过40min会增加脑室周围白质软化和中风的风险。NIRS用于识别DHCA的安全持续时间和左脑灌注不足的风险。

围手术期脑氧饱和度下降与术后神经系统并发症相关。在小儿心脏手术中，NIRS常用于识别DHCA的安全持续时间。DHCA手术的儿童rSO基线值(INVOS5100B)总体下降60%\~70%，在20\~40 min内达到最低点[22],rS02下降到50%所需的时间与年龄呈负相关,新生儿为9min,婴儿为6min,儿童为1\~4min[23],这意味着，单位时间内脑氧消耗随着年龄的增长而增加。

在一项通过头臂动脉进行区域低流量脑灌注的19例新生儿研究中（INVOS5100），rSO两个半球的差异为6.3%，左脑显示的数值较低[24],然而，左右脑部rSO的差异不影响术后神经系统并发症的发生。DHCA手术中rSO的变化小反映了大脑降温不足，在一项对250名先天性心脏病儿童的回顾性研究（INVOS3100A)中，Austin等人[5分别采用经颅多普勒和NIRS监测发现，术中有效的脑流量控制是脑血流峰值流速比基线减少50%或rSO,下降20%，术中有效的干预可以减少术后神经病变发生率、住院时间明显缩短。需要注意的是，心脏外科围手术期脑保护中国专家共识(2019)提出[25],建议通过经颅多普勒和NIRS技术对脑灌注进行无创评估，结合颈静脉血氧饱和度(jugularvenousoxygensaturation，SjvO）监测进行有创评估。当术中经颅多普勒脑血流流速变小和/或脑rSO<50%或低于绝对值20%和/或Sjv02<50%时，应当根据具体情况采取相应措施以保证脑部血流和氧供[26-28]

一项针对单心室患儿脑rSO基线值(INVOS4100)的研究表明，I期手术前脑部rSO约60%，Ⅱ期手术前脑部rSO约40%，Ⅲ期手术前脑部rSO约70%[26]。Dent等人[29]使用INVOS5100监测22名新生儿左心发育不良术后脑部rSO，rSO<45%持续180min的患儿有73%术后MRI显示新的脑缺血病灶或原有缺血病灶加重。Phelps等人2%对50例左心发育不良的新生儿脑rSO监测，结果显示：rSO>60.8%的患儿预后良好，rSO<50.8%的患儿预后不良。

许多临床研究分析了脑部rSO与混合静脉血氧饱和度或颈静脉血氧饱和度的关系，多数具有统计学意义，斜率小于0.5且关联强度为0.3\~0.9[30-33]虽然脑部rSO不能替代混合静脉氧饱和度的临床意义，但监测rSO的趋势对临床有指导意义。

# 2.腹部rSO监测

推荐意见1：体肺分流术后、侧枝血管开放、动脉导管开放等肠系膜血供相对不足的患儿，建议行腹部rSO监测，脑-腹部部位的rSO数值缩小，甚至腹部小于脑部的rSO数值，应警惕体循环灌注不足。(推荐级别：I类，证据水平：C)

推荐意见2：坏死性小肠结肠炎(NEC)高危患 儿，建议行腹部rSO监测，腹部rSO2<56%，应警惕 NEC的发生。(推荐级别：I类，证据水平：C)

推荐意见3：腹部rSO监测可以有效指导先心病术后的肠内营养支持，当腹部rSO<35%时，应

# 停止喂养。(推荐级别：I类，证据水平：C)

坏死性小肠结肠炎（necrotizingenterocolitis,NEC)是早产儿最常见的胃肠道疾病之一，其发病机制是多因素的。然而，肠道缺血先于坏死性小肠结肠炎发生。Patel等人[34]使用InSpectra StO监测研究表明，与发生NEC的早产儿相比，无NEC的早产儿在产后第一周的腹部rSO更高(77.3%比70.7%，P=0.002)。腹部rSO≤56%的早产儿与腹部rSO>56%的早产儿相比，NEC风险增加11倍。Piyagarnt等人[35]的喂养指南提出，当腹部rSO2<35%时，应停止喂养。中国医师协会新生儿科医师分会循证专业委员会制订的新生儿坏死性小肠结肠炎临床诊疗指南指出[36],推荐使用NIRS监测肠道氧合(弱推荐，C级证据）。因此，对高危NEC婴儿应行腹部NIRS监测，有助于NEC的早诊断、早治疗，防止肠穿孔或死亡。

值得注意的是，NIRS数值受许多改变组织的需氧量和耗氧量因素影响，如组织的代谢活动、发烧或体温过低、灌注状态、血压和血红蛋白水平，腹部rSO（脐下)受传感器位置、腹膜透析、膀胱扩张或导尿管影响。因此，NIRS数据虽可为临床提供一定参考，但对疾病的诊断不能完全依赖NIRS。

# 3.肾区rSO监测

推荐意见1：先心病术后急性肾损伤高危患儿，除了脑、腹部监测，建议同时监测肾区rSO2(推荐级别：I类，证据水平：C)

推荐意见2：肾区rSO通常高于脑部rSO2，若肾区rSO低于脑部或持续下降，结合血尿量、血肌酐和乳酸等其他指标，可积极实施肾脏替代疗法(推荐级别：I类，证据水平：C)

先心病术后急性肾损伤(acute kidney injury，AKI)发生率大约为36%\~59%[37]，当血清肌酐升高时肾脏损伤已很严重[38],因此很多学者探究了可早期识别先心病术后AKI发生的生物标记物[39]。然而，这些检查都具有有创性和血液丢失的风险，而实时无创的肾区NIRS监测是AKI早期识别和监测的实用工具。Ruf等人4前瞻性研究了59例体外循环下行心脏手术的婴儿，NIRS(INVOS)用于监测术中及术后最少24h的肾区rSO,结果显示，术中及术后12h、24h发生AKI组患儿肾区氧饱和度明显低于非AKI组。同样，Owens等人4前瞻性研究了40例行双心室修补手术的婴儿，NIRS（INVOS5100B)监测患儿术后48h的肾区rSO，结果显示：肾区rSO<50%并持续大于2h的患儿发生AKI的风险升高。ColbyColasacco 等人[42]用NIRS监测术中及术后1天、术后2天的肾区rSO,,并将此值与同时间段测量的肌酐水平的变化进行相关分析，发现二者有强烈的关联(R=0.6,P<0.001)，肾rSO<80%对术后肾功能不全预测的灵敏度达100%，特异度

4.ECMO期间rSO监测

推荐意见1：ECMO患儿推荐使用多通道NIRS监测以定量评估组织氧合灌注情况(推荐级别：I类，证据水平：C)

推荐意见2：对于ECMO股动静脉置管的患儿，可使用NIRS监测肢体远端区域氧饱和度，评估肢体远端灌注情况。当肢体rSO<40%，或下降程度>25%，或左右两个肢体rSO的差异值>15%时，警惕肢体远端灌注不足或栓塞形成(推荐级别：I类,证据水平：C)

随着ECMO在临床的广泛开展，NIRS监测在ECMO中的临床意义被进一步挖掘。大多数的研究主要集中于脑部和肢体NIRS监测。一般认为rSO<50%或下降程度>20%，考虑脑灌注不足，提示预后不良,要进行积极的临床干预[43]。一项关于儿童颈动静脉插管ECMO的研究指出，结扎单侧的颈动脉，该侧的rSO（INVOS5100)可下降12%\~25%[44]。体外生命支持组织关于小儿ECMO指南提出[45],除了常规监测VA-ECMO有效灌注的指标，如血乳酸、混合静脉血氧饱和度，NIRS也可作为重要的监测手段。

对于外周ECMO肢体远端灌注情况的监测，

NIRS也有一定的临床意义，但这些研究多集中于成人。Patton-Rivera等人[46研究指出，当左右两个肢体rSO的差异值>15%时，警惕置管引起的缺血，该研究还显示，缺血肢体几乎都表现为rSO<50%大于4min。Wong 等人[47]通过NIRS（INVOS5100B)监测肢体远端灌注，制订了管理方案，当rSO<40%或下降程度>25%时，应采取积极干预方案，包括调整肢体远端灌注管或切开筋膜减压等。

综上所述，NIRS技术可以提供实时无创多部位监测，定量提供重要脏器局部氧合的指标，有效地预警和指导先心病术后治疗和脏器保护，从而改善先心病患儿术后的转归。但是，由于目前缺乏多中心/单中心的随机双盲对照研究,循证水平偏低，有待于开展更多的临床研究。
