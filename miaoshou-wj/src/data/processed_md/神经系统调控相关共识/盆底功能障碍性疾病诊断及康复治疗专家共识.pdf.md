# 盆底功能障碍性疾病诊断及康复治疗专家共识

江苏省中西医结合学会盆底疾病专业委员会上海市医师协会肛肠专业委员会北京整合医学学会盆底疾病专业委员会上海市中医药学会肛肠分会

盆底功能障碍性疾病（pelvic floor dysfunction，PFD）是指由盆底支持结构薄弱、缺损及功能障碍等原因引起一系列症状的非器质性疾病[]，与年龄、肥胖、妊娠分娩、慢性疾病、不良心理刺激等多种因素相关[2]。盆底康复治疗是一种基于多学科循证医学证据的治疗PFD的策略，旨在通过不同的物理治疗措施，激活盆底神经肌肉使盆底肌肉功能逐渐恢复，从而减轻患者盆底功能障碍相关症状，促进盆底康复，提高患者生活质量[3]。目前，PFD的范畴及康复治疗方案相对模糊[4-5]，为此，江苏省中西医结合学会盆底疾病专业委员会联合上海市医师协会肛肠专业委员会、北京整合医学学会盆底疾病专业委员会、上海市中医药学会肛肠分会，组织国内相关领域专家，以国内外研究成果为依据，结合我国临床实际情况制订本专家共识，以期为PFD规范诊断及康复治疗提供参考。 MEDI

# 二、诊断

# （一）病史采集及体格检查

推荐意见1：详细的病史采集和体格检查，有助于正确评估疾病，明确诊断。（证据质量：C级；推荐级别：强）

询问病史、了解患者症状十分重要。首先了解基本信息如姓名、性别、年龄、籍贯、民族、婚姻状况、就业情况、受教育程度等，其次主诉、现病史、既往史、婚育史、个人史、家族史、月经史、性生活史、近期服药情况、心理状态、医学人文关怀等。

应重视采集疾病相关的常见病因与高危因素［6-7]，既往慢性呼吸道疾病和咳嗽、便秘、糖尿病病史；个人史关注吸烟史及运动习惯；手术史关注子宫切除术、肛肠手术等盆腔手术史；家族史关注尿失禁、膀胱过度活动症或大便失禁等；月经婚育史关注分娩年龄、绝经年龄、既往分娩史、分娩方式及异常孕产史；性生活史主要了解患者性满意度、性功能障碍的症状、持续时间及性质、是否采用药物治疗及治疗效果等，还需要采集性健康相关的信息。盆腔脏器脱垂患者典型症状是看到、触到、感觉到阴道口有肿物脱出，可伴有运动、大笑、咳嗽、喷嚏等腹压增高时的漏尿，部分患者也可同时伴有尿频、尿急、急迫性尿失禁或排尿困难的症状[8]。

体格检查包括局部神经系统检查及专科检查。神经系统检查主要为腰骶神经控制区域的皮肤感觉，包括肛门部、会阴体、外阴部感觉，以及球海绵体肌反射、肛门括约肌反射等[3]。专科检查主要包括肛门直肠指诊、女性生殖系统检查（包括外生殖器及阴道）以及男性前列腺指诊。通过专科检查可发现生殖器有无畸形、盆底肌松弛或高张、盆底结构和功能异常（如子宫脱垂、阴道脱垂、尿失禁、直肠脱垂、肌筋膜综合征、肛门失禁）、泌尿生殖疾病（包括感染、性功能障碍、外伤、肿瘤）等。盆腔脏器脱垂患者体格检查前先排空膀胱，一般采取截石位，如病情需要可取蹲位或站立位，观察患者静息状态和深呼吸后屏气向下用力状态下，盆腔脏器脱垂程度，应用POP-Q评分表进行评估。肛管直肠指检时，体位可采用蹲位、侧卧位、胸膝位、截石位等，嘱患者做提肛和排粪动作，可初步评估肛门括约肌的舒缩力、肛管紧张度以及盆底肌肉运动协调性等[9-10]。 出妆

# （二）肛管直肠测压

推荐意见2：肛管直肠测压有助于评估肛门直肠感觉和（或）运动功能，测定肛门直肠功能的异常，为患者治疗方案的抉择提供帮助。（证据等级：B级；推荐等级：强）。

肛门直肠测压属于肛门运动功能测试，主要是模拟正常排便时直肠、肛管压力的变化[。临床常用于功能性排便障碍、大便失禁、先天性巨结肠及功能性肛门直肠痛等疾病评估[12]。具有易于操作、易被患者接受等优点，但不同的测压系统和检测导管间存在较大差别，此外，体位也可能影响测压结果，存在较大误差［13-14]。

# （三）盆底超声

推荐意见3：盆底超声是帮助诊断盆底功能障碍性疾病的一项重要检查，盆底超声可以静态和动态观察盆底结构及肌肉运动变化，对其功能进行多层面评估，尤其对肛提肌及肛门括约肌损伤的评估更为全面，为临床诊断及疗效评估提供客观指标。（证据等级：B级；推荐等级：强）

盆底超声可以全面评估盆腔器官，包括尿道、膀胱、阴道、宫颈、子宫、肛管、直肠以及其他盆底结构。盆底超声目前有以下几种检查模式：经会阴超声、经阴道超声、经直肠超声。

经会阴超声在矢状面成像时，可以观察到前盆腔、中盆腔、后盆腔的几乎所有解剖结构，临床上用于尿失禁、盆腔脏器脱垂、直肠前突、会阴下降、不协调性排便等盆底疾病诊断、评估[15]。经阴道超声可以显示盆底轴状位的四个标准评估平面[16],精确评估肛提肌完整性及肛提肌裂孔，辨别可能受伤的肛提肌部分（耻骨尾骨肌、耻骨直肠肌、髂骨尾骨肌等）[17]，临床用于盆底形态评估。经直肠三维超声可以清晰显示肛管三层面解剖[18-19]，精确测量各部分括约肌的长度、厚度，同时也提高了对括约肌缺损诊断的精确性，其对肛管括约肌缺损诊断的敏感度和特异度都超过90%[20]。经直肠超声临床上用于诊断的盆底疾病包括：大便失禁、功能性排便障碍、直肠前突、直肠黏膜内套叠、肛门直肠痛等[21]。

# （四）排粪造影

推荐意见4：X线排粪造影是出口梗阻型便秘、盆腔脏器脱垂等疾病的重要评估方法，但考虑其存在辐射、操作较为繁琐，具有一定局限性。（证据等级：A级；推荐等级：弱）

X线排粪造影是利用造影剂模拟排便过程，对患者直肠肛门进行静态、动态观察，能直观获取直肠及肛门的器质性及功能性改变[22]，是评估出口梗阻型便秘[23]、盆腔脏器脱垂等疾病的重要方法。其中，对于多盆腔脏器脱垂，采用盆腔双重及三重造影效果更佳[24]。但是，由于具有一定的辐射，且实际临床中部分老年患者耐受性较差、操作较为繁琐、图像分辨率低等问题，仍有一定局限性。

推荐意见5：超声排粪造影具有操作简便、耐受性好、避免射线暴露等优点，可应用于盆底功能障碍性疾病的评估，尤其是出口梗阻型便秘。（证据等级：B级；推荐等级：弱）

超声排粪造影是将内置超声探头和耦合剂置入直肠内，观察患者静息及力排时的超声图像，并实时记录患者的盆底运动情况，获得二维及三维图像，该检查方式可以清晰观察到肛管、直肠的解剖结构，并能三维成像观察盆底的动态改变，可应用于盆底失弛缓、直肠前突、盆底疝、直肠黏膜内套叠、会阴下降等方面的诊断[25-26]，尤其是出口梗阻型便秘[27-29]，具有操作简便、耐受性好、避免射线暴露、清晰显示盆底结构等优点。

推荐意见6：磁共振排粪造影可用于评估盆底功能障碍性疾病，尤其是后盆腔脏器脱垂，但受其费用高、检查耗时长等方面的影响，临床应用较为局限。（证据等级：B级；推荐等级：弱）

磁共振成像无辐射，软组织分辨力高，可清晰显示盆腔内各脏器和盆底肌肉的解剖，对于后盆腔的脏器脱垂的评估尤其重要，能清晰的检出小肠膨出、腹膜膨出和直肠套叠等疾病[30-31]。适用于盆腔脏器脱垂、直肠内套叠、大便失禁、盆底失弛缓、会阴下降、盆底松弛等盆底功能障碍疾病[32]。磁共振成像具有评估全面、显示解剖细节、成像更直观等优点[33]。

# （五）结肠传输试验

推荐意见7：结肠传输试验是区分不同类型便秘的关键。（证据等级：B级；推荐等级：弱）

结肠传输试验是评估结直肠运动功能的重要方法，在区分正常传输便秘、慢传输便秘及出口梗阻型便秘中起关键作用[34-35]。目前常用的检测方法包括不透X线标记物、核素显像、钡餐[36]、胶囊内镜等[37]，其中，不透X线标记物法在临床中应用较多，其具有简便、快捷等特点，但是常需多次拍摄腹部平片，存在多次辐射的缺点。

# （六）尿动力学检查

推荐意见8：尿动力学检查在压力性尿失禁的诊断中具有重要价值，但对于盆腔脏器脱垂的诊断仍需进一步高质量、大规模的研究验证。（证据等级：B级；推荐等级：弱）

尿动力学检查可以检测患者贮尿和排尿功能，且同时测定膀胱的功能状态，检查内容包括尿流率测定、膀胱压测定、压力-流率测定、膀胱压力测定、漏尿点压力测定及尿道压力测定等[38]。尿动力学检查在压力性尿失禁的诊疗中具有重要意义[39],尤其对于复杂性、混合型压力性尿失禁患者，尿动力学检查明确病因是十分有必要的[40-41]。

同时，尿动力学检查对盆腔脏器脱垂患者也具有辅助诊疗价值[42]，有利于了解患者的排尿及储尿情况，以此制定合适的临床处理方案[43-44]。Baines等[45]认为目前需要进一步的大型研究来评估其价值，并寻求革新以提高检查的准确性和患者的接受度。

# （七）盆底肌电评估

推荐意见9：Glazer盆底表面肌电评估技术是评价盆底肌功能的重要方法，尤其是产后盆底功能的评估。（证据等级：B级；推荐等级：强）

Glazer盆底表面肌电评估技术是评价盆底肌功能的重要方法[46]，文献报道［47]，Glazer盆底表面肌电评估是可靠的，并且与肛管测压密切相关。国内多项研究[48-50]表明Glazer盆底表面肌电评估用于评价产后早期盆底功能，能提高产妇盆底肌力、改善盆底功能，具有一定的临床应用价值。Zhang等[51]发现Glazer盆底表面肌电评估可作为产后压力性尿失禁盆底分析的定量评估工具。

# 二、康复治疗

# （一）生活方式干预

推荐意见10：生活方式干预治疗操作简单，随时可行，是轻度PFD患者的首选治疗方法，然而，一个人的生活方式一旦形成，就很难不受限制地改变。所以，该法具有很大的随机性，无法进行量化评估与监督，疗效因人而异。（证据等级：B；推荐强度：强）

目前指南中提出了可将改善行为习惯作为PFD的一线干预方法。主要的干预措施：（1）控制体重：建议BMI>30kg/m²的患者减重，以帮助缓解盆底功能障碍相关症状，并推荐患者控制BMI<24 kg/m²，但仍需长期膳食管理、保持运动习惯[53]。（2）改变饮食结构：对于便秘患者，推荐提高摄入膳食纤维（18\~30g/d）及饮水量（1.5\~2.0L/d）可以增加排便次数、改善便秘症状、降低患者对泻药的依赖[54]；对于尿频或尿失禁患者，应减少咖啡因及液体的摄入[55]。（3）戒烟：盆腔结缔组织重要组成成分为胶原蛋白，吸烟不仅影响胶原蛋白的合成，且吸烟导致的过度咳嗽或慢性呼吸道疾病会加剧解剖缺陷、损害盆底肌肉功能并使腹压增加而影响盆底肌力；对尿失禁患者而言，香烟中的一氧化碳和尼古丁对逼尿肌有刺激作用，可引起盆底神经损伤，加重尿失禁症状[56]。（4）控制血糖：糖尿病可导致盆底骨骼肌、结缔组织、周围神经组织及微血管损伤，造成盆底支持结构的完整性和功能异常[57]。并且研究发现[58]盆底功能性障碍疾病的病情严重程度与糖尿病的病程呈正相关性。（5）盆底肌锻炼：目前临床最常用的是Kegel锻炼

法，通过患者主动收缩和放松训练增加其盆底肌群的协调性，促进盆底肌力和肌张力的恢复。美国医师学会推荐将盆底肌锻炼作为女性尿失禁的一线治疗方案，尤其是压力性尿失禁和以压力性尿失禁为主的混合性尿失禁［59]。

# （二）生物反馈治疗

推荐意见11：生物反馈疗法是一种患者主动参与的治疗方法，用视觉信号反映肌肉活动信息，充分利用患者的主观能动性，不仅经济方便、痛苦小，而且患者接受度高。（证据质量：A；推荐级别：强）

生物反馈是运用仪器通过视觉或听觉信号，揭示人体内部正常或异常活动的方法，通过操控无意识的生理活动来控制机体内部变化[60]。Kegel运动和电刺激治疗分别是生物反馈疗法的重要组成部分。生物反馈电刺激疗法是在患者直肠或阴道内放置电极，首次治疗前评估患者盆底肌肉强弱，并根据疾病诊断设置相对应的治疗模式[61]。关于难治性慢性便秘的治疗，Jung等人[62]报道了电刺激疗法尤其有利于患者直肠感觉的恢复，有效率达59.2%。虽然各项研究有异质性、非随机化、各种治疗方案不同等差异，但Hite等人[63]研究报道了生物反馈治疗PFD的有效率为18%\~100%。Skardoon等[64]研究表明，70%\~80%的慢性便秘患者接受生物反馈治疗后便秘症状明显好转，且观察其远期疗效可达55%\~82%。

# （三）骶神经调节

推荐意见12：骶神经调节术对盆底疼痛综合征、神经源性下尿路功能障碍、间质性膀胱炎及慢性功能性便秘均有一定的疗效[65-68]。（证据质量：A;推荐级别：强）

骶神经调节术是借助介入技术对特定骶神经连续施加低频电脉冲，通过兴奋或抑制神经通路来调节异常的神经反射，进而影响并调节膀胱、尿道、肛门括约肌等骶神经支配器官、组织的功能，从而达到治疗目的的一种神经调节技术。骶神经调节术可应用于多种难治性下尿路功能障碍[69]。目前美国FDA批准的适应证为[70]：（1）顽固性尿频－尿急综合征、难治性急迫性尿失禁，也称难治性膀胱过度活动症；（2）特发性尿潴留或非梗阻性尿潴留；（3）排便功能障碍，如便失禁等。

# （四）经皮电刺激

推荐意见13：经皮电刺激价廉、安全、无创、 简便易学且可随意调节刺激电流频率与强度，便于

# 制定个性化治疗方案，易被患者与医生采纳。（证

# 据等级：A；推荐强度：强）

经皮电刺激可增强肌肉力量和肌肉敏感性，其通过体表将电流导入人体刺激盆底神经使盆底肌肉产生被动性收缩运动，促进局部血液循环，从而改善盆腔器官功能。司右等[7I]研究证明，经皮电刺激可通过收缩患者盆底肌达到锻炼目的，为预后提供保障。经皮电刺激短期内可增强盆底肌肉训练的效果，有效治疗盆底失禁类疾病，但Berghmans等人[72]研究发现，其6个月以上的远期疗效不佳，临床推荐与生物反馈治疗联合应用。

# （五）盆底磁刺激

推荐意见14：磁刺激疗法具有操作简便、无创、无侵入、穿透力强等特点，可有效改善盆底功能，临床推广应用意义较大。（证据等级：A；推荐强度：强）

磁刺激是电磁感应原理衍生出的一种治疗方法，组织中的电流受其精确诱导，促使神经轴去极化，反复激活肌肉组织，增强其肌力和耐力，使盆底功能受盆底肌群活动影响而改善。Lim等研究发现[73]压力性尿失禁患者对磁刺激治疗接受度高，耐受性好，治疗满意度高，不良事件发生率及脱落率低；且已有研究证实磁刺激治疗压力性尿失禁具有确切的远期疗效［74]。Bharucha等［75]研究发现，磁刺激治疗大便失禁患者的原理可能与增加肛门挤压的压力和直肠顺应性相关。

# （六）中药内治法

推荐意见15：中药汤剂口服治疗对盆底功能障碍性疾病的改善有一定的疗效，但是，由于个体差异大、重复性低，在临床研究和普及应用方面存在一定难度。（证据质量：B，推荐级别：强）

历代医家认为，PFD病位主要在脾、肾，涉及胞宫、心、肝。病机主要为肾气不固，脾胃虚弱，中气下陷，病性总属于虚，治以补肾固脱、补脾益气。代表方剂为补中益气汤，具有补中益气、升阳举陷的功效，出自金代名医李东垣的《脾胃论》。陈舟等［76]研究显示采用补中益气汤联合盆底康复治疗PFD患者，疗效明显优于单独盆底康复治疗组，并认为其作用机制可能与调节盆底组织的转化生长因子β1、锌离子蛋白水解酶、锌离子内切蛋白水解酶水平有关。王占利等[77]研究了女性PFD患者的内服中药用药规律，大多以补中益气汤为基础方加减，大剂量使用健脾益气、升阳举陷之药物，辅以补肾固冲，清热利湿兼平肝。另外，有研究报道金匮肾气丸、四物汤、举宫汤、龙胆泻肝汤、益气固脱方等均对PFD有显著疗效[78-82]。

# （七）中药外治法

推荐意见16：传统中药外治法包括穴位贴敷、中药灌肠、中药熏洗等，可作为一种盆底功能障碍性疾病患者的辅助疗法，值得临床推广应用。（证据质量：C，推荐级别：弱）

中药外治法包括中药熏洗、穴位贴敷、中药灌肠等治疗，杜永红等[83]研究发现，中药熏洗加穴位贴敷联合生物反馈治疗PFD患者的有效率（91.67%）明显优于对照组单纯生物反馈治疗（72.22%)。叶宇飞等[84]一项随机对照研究认为四黄祛毒汤保留灌肠联合熏洗治疗功能性肛门痛可明显缓解患者疼痛，显著改善患者生活质量，且简单易操作、稳定性好。一项多中心研究发现[85]，沉香通便膏穴位贴敷也可以有效改善功能性便秘患者的主要症状，且无明显不良反应。

# （八）针刺

推荐意见17：针刺疗法可作为一种主要治疗手段，适用于各种临床表现的PFD患者，亦可联合药物或生物反馈疗法，具有提高疗效、缩短疗程、减少不良反应等优势，有较高的临床推广应用价值。

# （证据质量：A，推荐级别：强）

针刺是中医疗法的重要组成部分，可以综合改善PFD的多种临床表现，提高患者生活质量。王晓峰［86]等基于文献分析发现，针刺治疗大便失禁的有效率为75.0%\~98.8%，可以降低失禁次数、改善肛门静息压和持续收缩压等。SUN等[87]进行的随机临床研究发现，电针可有效降低压力性尿失禁患者的漏尿次数及漏尿量，改善患者生活质量，且远期疗效可达3年。一项多中心、随机、对照临床研究［88]显示，针刺双侧的中髎穴和会阳穴治疗压力性尿失禁，治疗组漏尿明显减少。另一项多中心、随机、对照临床研究［89]显示，针刺双侧中膠、会阳、肾腧和三阴交穴治疗男性慢性前列腺炎/慢性盆腔疼痛综合征，结果显示治疗组中重度前列腺炎/慢性盆腔疼痛综合征的症状有明显的改善，而且治疗24周后效果仍然持续。针刺八髎穴可显著改善出口梗阻型便秘的排便困难程度、排便频率、排便不尽感和排便时间[90]。程思雨等[91]采用电针八髎穴治疗功能性肛门直肠痛，生物反馈治疗为对照，结果电针组在临床有效率、VAS疼痛评分及生活质量评分方面都优于对照组。

# （九）其他中医疗法

推荐意见18：掀针、穴位注射、艾灸、推拿等其他中医疗法，均可作为治疗PFD的一种辅助疗法，结合其他治疗，有助于进一步提高临床疗效。（证据质量：C，推荐级别：弱）

杨香燕等[92]发现撤针联合生物反馈治疗对于改善功能性便秘、慢传输型便秘患者的肠道平滑肌动力、缓解便秘症状和提高生活质量均具有显著效果。黄美霞等[93]观察了甲钴胺八穴注射治疗经阴道分娩PFD患者的临床疗效，6个月随访结果示穴位注射组患者盆底肌力等级的提高、盆底肌疲劳度的缓解、盆底障碍影响简易问卷评分均明显优于Kegel锻炼组和生物反馈组，且操作简单易行。王琳琳等[94]一项对60例盆腔脏器脱垂患者进行随机对照研究，通过比较电针联合艾灸与生物反馈电刺激治疗产后盆腔器官脱垂的临床疗效观察，艾灸联合电针可明显增强盆腔脏器脱垂患者盆底肌力及耐力，提高患者的生活质量。杨秋晔等[95]艾灸中极穴、关元穴、神阙穴、肾俞等，发现艾灸可以调节膀胱气机，增强膀胱对尿液的约束能力，缓解尿失禁。徐丹丹等[96]将70例产后压力性尿失禁患者随机分为康复训练组和推拿组，结果显示产后压力性尿失禁经太极推拿结合康复训练治疗后，盆底肌肌力和腹直肌分离明显改善，漏尿明显减少。

# （十）心理干预疗法

推荐意见19：在实际心理干预治疗中，应围绕患者需求，与患者建立双向沟通，实施特定的心理干预计划，从而帮助患者尽快恢复健康。（证据

# 质量：B级，推荐级别：强）

盆底功能障碍性疾病常常会带来各种负面情绪，如焦虑、抑郁和自卑等，心理干预是治疗盆底功能障碍性疾病和提高患者生活质量的重要手段之一。常见的心理干预方法有认知行为治疗、正念练习和解离情感治疗等。这些干预方法的重点在于帮助患者从负面思维和情绪中解放出来，并激发他们主动的、积极的理念以及自我调节的行为能力，并学会用更好的方法去处理疼痛和不适，从而减轻心理压力，恢复生活质量[97-98]。
