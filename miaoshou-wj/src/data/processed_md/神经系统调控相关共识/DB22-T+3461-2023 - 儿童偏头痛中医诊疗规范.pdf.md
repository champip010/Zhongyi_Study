# 儿童偏头痛中医诊疗规范

# 1范围

本文件规定了儿童偏头痛中医诊疗的禁忌症、诊断、辨证论治和预防调护。  
本文件适用于18周岁以下人群儿童偏头痛的中医诊断和防治。

# 2规范性引用文件

本文件没有规范性引用文件。

# 3术语和定义

下列术语和定义适用于本文件。

3.1

儿童偏头痛 migraine in children

一种常见的神经血管性疾病，以反复发作的一侧或双侧搏动性头痛为特点，发作时多有恶心呕吐、面色苍白、心率及呼吸增快、胃肠道功能紊乱等。男性儿童发病的平均年龄在7岁左右，女性儿童发病的平均年龄在11岁左右。属中医“偏头风”、“偏正头风”、“偏头痛”、“偏正头痛”、“脑风”等范畴。

# 4禁忌症

4.1 有进展性神经系统变性疾病或代谢性疾病者。  
4.2 合并严重的全身性疾病、器质性病变、精神障碍、认知功能障碍者。  
4.3 需要外科手术治疗者。

# 5诊断

5.1头痛部位多在头部一侧额颞、前额、巅顶，或左或右辗转发作，或呈全头痛.疼痛性质多样但以搏动性最具特点，头痛每次发作可持续数分钟、数小时、数天，也有持续数周者。  
5.2隐匿起病，逐渐加重，反复发作，突然停止。  
5.3应查血常规，测血压，必要时做腰穿、骨穿，脑电图；有条件时做经颅多普勒、CT、MRI等检查，以明确头痛的病因，排除器质性疾病。

# 6辨证论治

# 6.1 外感头痛

# 6.1.1 风寒头痛

# 6.1. 1. 1 证候

头痛连及项背，常有拘急收紧感，或伴恶风畏寒，口不渴，苔薄白，脉浮紧。

# 6.1.1. 2 治法

疏散风寒止痛。

# 6.1. 1. 3 主方

川芎茶调散：川芎、白芷、藁本、羌活、细辛、荆芥、防风、炙甘草。

# 6.1. 1. 4 加减治疗

6.1.1.4.1头痛、恶寒明显者，加麻黄、桂枝。  
6.1.1.4.2寒邪侵于厥阴经脉，症见巅顶头痛，干呕，吐涎沫，加吴茱萸、法半夏。

# 6.1.2 风热头痛

# 6.1. 2. 1 证候

头痛而胀，发热或恶风，面红目赤，口渴喜饮，大便秘结，舌尖红，苔薄黄，脉浮数。

# 6.1.2.2 治法

疏风清热止痛。

# 6.1.2.3 主方

芎芷石膏汤：菊花、桑叶、薄荷、川芎、白芷、羌活、生石膏。

# 6.1.2.4 加减治疗

6.1.2.4.1 烦热口渴，舌红少津者，可重用石膏，加知母、天花粉。  
6. 1.2.4. 2 大便秘结，腑气不通，加黄芩、栀子。

# 6.2内伤头痛

6.2.1 肝阳上亢

# 6.2.1. 1 证候

头昏胀痛，两侧为重，心烦易怒，夜寐不宁，口苦面红，或兼胁痛，舌红苔黄，脉弦数。

# 6.2.1. 2 治法

平肝潜阳止痛。

# 6.2.1.3 主方

天麻钩藤饮：天麻、钩藤、石决明、山栀、黄芩、丹皮、桑寄生、杜仲、牛膝、益母草、白芍、夜交藤。

# 6.2.1.4 加减治疗

6.2.1.4.1 头痛剧烈，便秘溲黄者，加夏枯草、龙胆草、大黄。

2

6.2.1.4.2头晕目涩，视物不明，加枸杞、白芍。

# 6.2.2 痰蒙清窍

# 6.2.2.1 证候

头痛昏蒙，胸部满闷，纳呆呕恶，舌苔白腻，脉滑。

# 6.2.2.2 治法

燥湿通窍止痛。

# 6.2.2.3 主方

半夏白术天麻汤：半夏、橘红、茯苓、白术、天麻、刺蒺藜、炙甘草。

# 6.2.2.4 加减治疗

6.2.2. 4. 1 痰湿化热，口苦便秘，加黄苓、竹茹、枳实。  
6.2.2.4.2 胸闷、呕恶明显，加厚朴、枳壳、生姜。

# 6.2.3 瘀血头痛

# 6.2.3.1 证候

头痛经久不愈，头痛如刺，舌紫暗，或有瘀斑、瘀点，苔薄白，脉细涩。

# 6.2.3.2 治法

化瘀通窍止痛。

# 6.2.3.3 主方

通窍活血汤：川芎、赤芍、桃仁、当归、白芷、细辛、生姜。

# 6.2.3.4 加减治疗

头痛较剧，久痛不已，加全蝎、蜈蚣、地鳖虫。

# 6.3 针刺疗法

6.3.1 取穴

6.3.1. 1 发作期

6.3.1. 1.1 主穴阿是穴、丝竹空、率谷、太阳、风池、合谷、太冲、足临泣。

6.3.1. 1.2 方法刺法，用泻法。

# 6.3.1. 2 缓解期

6.3.1. 2. 1 主穴

# DB22/T3461—2023

阿是穴、四神聪、头维、印堂、神门、安眠、角孙、完骨。

# 6.3.1. 2.2 方法

刺法，用补法。

# 6.3.2 注意事项

严重的冻伤、湿疹、溃疡等皮肤病变者禁用。

# 7 预防调护

7. 1 顺应四时变化，寒温适宜，起居定时，增强体质。  
7.2 适当休息，保持环境安静。  
7.3 避免精神刺激，消除患儿恐惧心理，减少精神压力。  
7.4 禁食肥甘厚腻、辛辣之品。
