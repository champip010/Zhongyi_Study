# 一、背景介绍

# （一） 任务背景

慢性阻塞性肺疾病（Chronic Obstructive Pulmonary Disease，COPD）是一种常见的、可以预防和治疗的疾病，以持续呼吸症状和气流受限为特征，通常是由于明显暴露于有毒颗粒或气体引起的气道和/或肺泡异常所导致。COPD最常见的症状为咳嗽、咳痰、呼吸困难，在其病程中常出现急性加重，急性加重是促进疾病持续进展、导致额外治疗的主要因素。COPD主要累及肺脏，但大多数患者存在重大的共患慢性疾病，增加COPD 的致残率和死亡率。目前COPD 居全球死亡原因的第4位，预计2020年将成为全球致死原因第3位和经济负担的第5位。全球COPD的总体患病率约为11.7%，每年约有350万人死于COPD，世界卫生组织预计到2030 年全球每年约有超过450万人死于COPD 和其相关疾病。2012～2014年我国40 岁以上人群的COPD 患病率约为13.7%，COPD 排在我国疾病死亡原因的第3位，我国COPD 的死亡率及疾病负担均高于全球平均水平。COPD多属于中医学的“咳嗽”、“喘病”、“肺胀”等范畴。

为进一步完善诊疗规范，促进中医诊治COPD水平的提高，2011年国家中医临床研究基地重点病种（慢阻肺）建设项目组组织专家，按照国际相关指南研制规范，制定了《慢性阻塞性肺疾病中医诊疗指南（2011年版)》。2011年以来，国际学术界对COPD 的认识发生了新的变化，并对相关指导性文件进行了重要修订。同时，我国中医药防治COPD的临床研究也取得了一些新的成果。为了及时反映国内外研究进展，更好地指导我国COPD的中医临床诊疗工作，中华中医药学会肺系病分会联合中国民族医药学会肺病分会组织成立了由中医或中西医结合临床和基础研究、临床流行病学、循证医学、卫生经济学、药剂学等多学科人员组成的《慢性阻塞性肺疾病中医诊疗指南（2018修订版)》修订小组，参照 2011版《慢性阻塞性肺疾病中医诊疗指南》，依据临床实践指南更新规范（Checklist for the Reporting of Updated Guidelines,CheckUp）、指南报告规范-RIGHT标准的要求，以循证医学思想为指导，注重中医特色，对现有的相关证据进行了充分收集和评价，编写成本指南，以期能够为从事COPD防治的中医、中西医结合临床医师提供指导性意见。

# （一）启动部署阶段

2017年6月20日，李健生教授在河南中医药大学第一附属医院召开“2018 慢性阻塞性肺疾病中医诊疗指南”修订项目讨论会，会议确定《慢性阻塞性肺疾病中医诊疗指南（2011版)》进行修订，以期为中医药诊疗COPD临床应用给出切实可行的推荐或建议。工作组基于现有最佳证据、结合专家经验、结合中医药自身特点及我国的实际情况，制订《慢性阻塞性肺疾病中医诊疗指南（2018修订版)》。为此，李建生教授组建了8人小组，李建生教授作为课题的整体顶层设计领导者，成立个工作小组。为了保证研究进度，工作组设立了例会制，定期汇报工作进度，完善方案，加强质量控制，同时制订了COPD工作组指南编制时间节点，见图1。2018年5月下旬在国际指南注册平台进行了双语注册，获得注册号：IPGRP-2018CN029；6月底获得中华中医药学会团体标准立项。

# 2．组织管理

本指南设置指导委员会、秘书处、共识专家小组、系统评价小组、起草工作组。指导委员会负责确定指南主题和范围；组建共识专家组和秘书组，并管理其利益声明；监督指南修订流程；批准推荐意见和指南全文；监测并评估指南的更新需求。秘书处负责全程协调整个指南编撰的组织管理和业务管理工作，详细记录指南修订整个过程；协调指南修订相关事项；协助完成指南外审工作。共识专家小组对于指南修订过程中，有关临床PICO的讨论、推荐意见的形成、专家共识的形成都需要共识专家发挥积极作用。系统评价小组负责完成证据的检索和评价，以及GRADE的评价。起草工作组负责协助调研临床PICO问题；协助完成系统评价、证据分级和形成决策表。

本指南所有参与成员均签署利益冲突声明，申明无任何与本部指南主题相关的任何商业的、专业的或其他方面的利益冲突。

# 4．确定修订的主题和目的，制定修订计划

本指南是《慢性阻塞性肺疾病中医诊疗指南（2011版)》的修订版。项目组依据指南更新报告规范的要求，对《慢性阻塞性肺疾病中医诊疗指南（2011版)》内容进行了全面梳理，确定需修订的范围，并在相关法律法规和技术文件指导的框架下，基于现有最佳证据、结合专家经验、结合中医药自身特点及我国的实际情况，制订《慢性阻塞性肺疾病中医诊疗指南（2018修订版)》，以期为临床应用给出切实可行的推荐或建议，为从事慢阻肺防治的中医、中西医结合临床医师提供指导性意见，提升中医药防治慢阻肺水平。

# 6．针对各结局对证据体进行质量评价

Meta分析的评价：采用AMSTAR量表进行文献质量评价。每个条目评价结果分为“是”、“否”、“不清楚"或"未提及”三种，并给予加分如“是"为1分，“否”、“不清楚"或"未提及"为0分，共11分。AMSTAR 量表得分0\~4分为低质量，5\~8分为中等质量，9\~11分为高质量。纳入与本指南 PICO一致的中、高质量的 Meta分析。对发表年份于2015年以前的系统评价，若有新的RCT文献则进行更新后纳入。

RCT文献评价方法：根据Cochrane评价手册Handbook5.1.0中的"偏倚风险评估”工具对进行评价：①随机分配方法；②分配方案隐藏；③对研究对象，治疗方案实施者采用盲法；④对研究结果测量者采用盲法；③结果数据的完整性；③选择性报告研究结果；?其他偏移来源。最终对文献做出“偏倚风险低”、“偏倚风险高”、“偏倚风险不确定"的判断。

证据的可信度及推荐强度采用国际统一的2004年制订的推荐分级的评估、制定和评价(The Grading of Recommendations Assessment Development andEvaluation,GRADE)标准。

# 7．名义组法实现共识过程

证据综合报告形成：系统检索、筛选中医药治疗COPD的文献，按GRADE标准对符合PICO的Meta分析及RCT文献进行评价，得出证据级别；并检索相关经典古籍、现代呼吸专著、医保目录、中华人民共和国药典等标准化文件，作为证据补充，形成证据综合报告。

临床专家调研报告：基于证据综合报告基础上，进行2轮专家问卷调研（副主任医师以上职称，涉及17家医疗单位，覆盖全国16个省份和直辖市)，第一轮基于证据收集后的干预措施进行评价；第二轮对需要更新版指南编写体例及推荐的中成药进行评价。结合专家调研结果，拟定出推荐方案。最终进行了38人次的调研：第一轮20名，第二轮18名。

药品不良反应及经济学评价信息：充分收集推荐方案中药物说明书或文献中提及药物不良反应信息；将推荐方案与同类药物进行经济学评价对比，并结合患者价值观及偏好，得出推荐方案的经济学信息。

制定推荐意见表：运用GRADE网格将推荐意见所基于的证据质量、临床专家调研结果、药品说明书信息、患者偏好与价值观以及相关经济学分析数据信息整合，形成推荐意见表及投票单。

召开名义组会议：①由13名共识组专家于2018年2月3日在河南中医药大学第一附属医院召开名义组会议。共识组成员包括：从事呼吸系统疾病诊疗的临床一线中医、西医专家；方法学专家：中医药循证医学方面的专家；临床药师1名。②由会议秘书阐述指南更新推荐方案，引导共识组专家阅读推荐意见表的相关信息，共识组专家充分权衡利弊，做出"强推荐”、“弱推荐”、“无明确推荐”、“弱不推荐”、“强不推荐"的判断，并独立填写推荐意见投票单。③统计推荐结果，并达成共识。统计方法如下：若除了“0"以外的任何1格票数超过50%，则视为达成共识，可直接确定推荐意见方向及强度;若"0"某一侧两格总票数超过70%，亦视为达成共识，可确定推荐方向，推荐强度则直接定为"弱”；其余情况视为未达成共识。未达成共识的推荐意见，将由共识专家阐述意见，会议秘书记录、汇总、写成备注后，进入下一轮投票，经过≥2轮的投票，最终达成共识。

# 四、标准编制的原则

本指南的修订程序、方法、结构，是在借鉴国际上通用的临床实践指南制定方法的基础上，注重中医特色，既要保证指南制订的科学性，又要符合中医疾病临床实际。由于本病的复杂性，很多临床中医药治疗多为专家经验，缺乏严格的临床证据，所以本指南的制订，广泛听取专家意见，对指南的整体拟解决的临床问题进行了前期调研和证据检索，做到全有过程和环节有据可循，可以溯源。

另外，本指南的体例结构将严格遵循中华中医药学会团体标准所要求的格式，进行统一规划撰写，力争能够符合要求并顺利通过审核、发布。
