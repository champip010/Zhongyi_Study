# 变更日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [1.0.0] - 2024-08-21

### 新增
- 🎉 初始版本发布
- 🔍 基于BGE-large-zh的向量检索系统
- 🗄️ Qdrant向量数据库集成
- 🧠 Self-RAG和HyDE高级检索技术
- 🏥 医学领域优化的分块策略
- 📚 多跳检索和知识连接
- 🧪 完整的测试套件

### 特性
- 支持中文医学文档的智能分块
- 基于语义的文档检索
- 多维度重排序算法
- 医学知识图谱构建
- 离线模型加载支持
- GPU加速计算

### 技术架构
- 模块化设计，易于扩展
- 支持多种embedding模型
- 灵活的配置系统
- 完整的错误处理
- 详细的日志记录
